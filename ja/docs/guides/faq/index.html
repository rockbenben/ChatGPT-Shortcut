よくある質問

なぜ英語のプロンプトを使うのですか？​

AI モデルは一般的に英語の理解力が高く、英語のプロンプトを使用することで、より安定した出力結果を得ることができます。

中国語のプロンプトを使用してもそれなりの結果は得られますが、同じ中国語のプロンプトを繰り返し入力すると、結果に大きなばらつきが生じる可能性があります。そのため、重要なシナリオでは英語のプロンプトを使用することをお勧めします。

中国語での返信を指定：プロンプトの最後に respond in Chinese を追加すると、中国語で返信が得られます。

毎回プロンプトを入力する必要がありますか？​

API の使用：プロンプトを「system prompt」として設定でき、その後の会話に自動的に適用されます。

Web 版：トピックを切り替えていない場合、ChatGPT は現在の会話のプロンプト設定を記憶します。返信が期待から外れ始めたら、AI がプロンプトを「忘れた」ことを意味するため、再入力する必要があります。

ヒント：よく使う会話をブックマークしておくと、後ですばやくアクセスできて便利です。

AI が偽情報を出力した場合はどうすればよいですか？​

AI は時々「幻覚」を起こし、もっともらしく見えるが実際には誤っている情報を出力することがあります。使用上の提案：

  1. 重要な情報の確認：特にデータ、引用、コードなど
  2. 多段階の最適化：AI に回答を再度チェックさせ、最適化させる
  3. 相互検証：異なるプロンプトやモデルを使用して、重要な結論を検証する

適切なプロンプトは、AI の幻覚の発生を減らすのに役立ちます。

プロンプトをバックアップするには？​

  1. ユーザーセンターに入ります
  2. 「プロンプトをエクスポート」ボタンをクリックします
  3. システムが JSON ファイルを生成してダウンロードします

データの損失を防ぐために、定期的にバックアップすることをお勧めします。

プロンプトの効果が理想的でない？​

  1. 最適化を試みる：AI に回答を再度最適化させてください
  2. 視点を変える：同じ要件を表現するために異なるプロンプトを使用してみてください
  3. フィードバックと共有：問題を見つけたり、良いプロンプトがある場合は、フィードバックでお知らせいただくか、コミュニティで共有してください
