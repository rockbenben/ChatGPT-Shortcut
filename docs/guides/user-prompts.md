# 自定义提示词

登录后，您可以创建自己的提示词，方便日后快速调用。自定义提示词会保存到您的账户中，并显示在我的收藏视图。

## 创建提示词

1. 点击页面右上角的「添加提示词」按钮
2. 填写提示词信息：
   - **标题**：简短描述提示词用途
   - **描述**：提示词的完整内容
   - **备注**：使用说明或注意事项（可选）
3. 选择是否分享到社区
4. 点击提交

## 编辑提示词

在我的收藏视图中，点击您创建的提示词卡片，可以：

- 修改标题、描述和备注
- 切换分享状态（公开/私密）
- 保存更改

## 删除提示词

在编辑界面中点击「删除」按钮可删除提示词。删除后无法恢复，请谨慎操作。

## 分享到社区

创建或编辑提示词时，勾选「分享到社区」选项：

- **公开**：提示词会显示在社区页面，其他用户可以看到并收藏
- **私密**：仅自己可见，不会出现在社区页面

您可以随时修改分享状态。

## 导出备份

导出您创建的所有提示词：

1. 进入个人中心
2. 点击「导出提示词」按钮
3. 系统生成 JSON 文件并自动下载

导出内容包括：

- 提示词标题、描述、备注
- 创建和更新时间
- 分享状态

建议定期导出备份，防止数据丢失。
