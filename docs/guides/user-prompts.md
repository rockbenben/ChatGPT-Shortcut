# 自定义提示词

登录后，你可以创建自己的提示词，保存到账户里随时调用。自定义提示词会出现在「我的收藏」视图中。

## 创建提示词

1. 登录后，点击页面顶部的「**创建提示词**」按钮
2. 在弹出的「创建提示词」窗口填写表单：
   - **提示词名称**（必填）：给提示词起个名字
   - **提示词内容**（必填）：提示词正文，方括号 `[...]` 在展示时会作为占位符高亮
   - **作用/用途**（选填）：简要描述提示词能做什么
   - **备注说明**（选填）：来源、其他语言版本或补充说明
3. 底部「是否愿意将该提示词分享到公开页面？」开关默认开启——关闭则仅自己可见
4. 点「创建提示词」提交

![创建提示词弹窗](/img/docs/zh/user-prompts-create.png)

> 💡 **填写示例**——一条"周报写作助手"提示词：
> - **提示词名称**：周报写作助手
> - **提示词内容**：你是一位职场写作助手，请基于以下内容生成本周工作周报，要求条理清晰、突出重点：[本周完成的事项]
> - **作用/用途**：把流水账整理成结构化周报
> - **备注说明**：每周一上午用一次

## 编辑提示词

在我的收藏视图，点击你创建的提示词卡片上的编辑（铅笔）按钮，打开「编辑提示词」窗口，可以：

- 修改名称、内容、作用、备注
- 切换分享状态（公开/私密）
- 点「保存」更新

![编辑提示词界面](/img/docs/zh/user-prompts-edit.png)

## 删除提示词

在编辑窗口点「删除」即可。删除后无法恢复，请谨慎操作。

## 分享到社区

创建或编辑时，底部「是否愿意将该提示词分享到公开页面？」开关控制可见性：

- **开启（默认）**：提示词显示在 [社区提示词](./community) 页面，其他用户可看到并收藏
- **关闭**：私密提示词，仅自己可见

分享状态可随时切换。

## 导出备份

1. 进入「我的账户」→ 找到「数据管理 → 导出提示词」一栏
2. 点「导出数据」按钮
3. 系统生成 JSON 文件并自动下载

导出内容包括：

- 提示词的名称、内容、作用/用途、备注说明
- 分享状态
- 你的收藏、收藏排序与自定义标签

建议定期导出备份，防止数据丢失。

## 导入提示词

从 JSON 文件导入提示词和收藏：

1. 进入「我的账户」→ 找到「数据管理 → 导入提示词」一栏
2. 点「导入数据」按钮
3. 选择 JSON 文件
4. 系统自动合并数据（按标题去重；他人 ID 的提示词自动设为私密）

### 团队协作

团队内分享提示词的推荐流程：

1. **筛选与分享**：导出后手动删掉收藏列表或筛选出要分享的提示词，发文件给团队成员导入
2. **隐私保护**：导入他人分享的提示词（ID 不属于当前账户）会自动设为**私密**，各成员数据互不干扰
