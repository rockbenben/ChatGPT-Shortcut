# 账户管理

AiShort 不登录也能用——但**登录后**你可以收藏喜欢的提示词、创建自己的提示词、分享到社区，并跨设备同步。下面介绍登录、注册、密码与缓存的相关操作。

## 登录方式

登录弹窗提供「**密码登录**」和「**免密登录**」两种方式，外加 Google 一键登录。

### 密码登录

用用户名或注册邮箱 + 密码登录。

### 免密登录

1. 切到「免密登录」，输入注册邮箱
2. 点「**获取登录链接**」——登录链接将发送至你的邮箱
3. 打开邮件中的链接即可登录

不用记密码，适合不常登录的场景。

### Google 登录

点登录弹窗的 Google 按钮，一键登录。首次会自动创建账户，不用单独注册。

![](/img/docs/zh/account-login.png)

## 注册账户

1. 在登录弹窗点「**立即注册**」切到注册
2. 填邮箱、用户名和密码，并勾选同意「隐私政策」和「服务条款」
3. 提交后自动登录

![](/img/docs/zh/account-register.png)

## 我的账户

登录后进入「**我的账户**」页面，可以管理资料、密码、等级和数据。

![我的账户页面](/img/docs/zh/account-profile.png)

### 修改用户名

点用户名旁的编辑图标，改完保存即可。

### 修改密码

1. 输入当前密码
2. 输入新密码并确认
3. 点「修改密码」提交

忘记密码时，在登录弹窗点「忘记密码」，输入注册邮箱点「发送重置邮件」，按邮件链接重置即可。

### 我的等级

账户页展示你的**等级（L0–L9）**和升级进度——等级由你**分享到社区的提示词数量**决定，分享越多档位越高。

## 数据管理

「我的账户 → 数据管理」一栏提供三项操作：

- **清除缓存** — 刷新本地缓存数据；数据不同步或显示异常时使用，收藏和自定义提示词不会丢，只是强制重新同步
- **导出提示词** — 导出为 JSON 文件，方便备份（详见[自定义提示词](./user-prompts#导出备份)）
- **导入提示词** — 从 JSON 文件导入提示词和收藏
