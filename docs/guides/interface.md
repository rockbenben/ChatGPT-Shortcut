# 界面说明

本节介绍 AiShort 首页的三大区域——**标签区**（顶部场景标签）、**搜索区**（关键词输入框）、**提示词展示区**（下方卡片列表）——以及怎么用它们快速找到想要的提示词。

![](/img/docs/interface-home.png)

## 🏷︎ 标签筛选

按提示词的领域和功能筛选。右上方的 **OR / AND** 切换器决定多标签的筛选规则：

- **OR**（默认）：命中任一标签即显示
- **AND**：必须同时命中所有标签

![](/img/docs/interface-filter.png)

## 🔍 关键词搜索

支持搜索提示词的标题、简介、内容。

- 首页搜索范围是**精选提示词**（不含社区）。已选标签时，搜索范围限定在这些标签内
- 登录后，搜索范围扩展到你的**收藏**和**自定义提示词**
- 用短关键词在首页搜不到时，去 [社区提示词](./community) 页面再搜一次

![](/img/docs/interface-search.png)

## 🔬 提示词卡片

### 查看详情

点击卡片打开详情弹窗——查看完整内容、备注，一键复制。

![](/img/docs/prompt-card.gif)

### 复制提示词

点卡片下方「复制」按钮，粘贴到任意 AI 工具即可使用（对话页、编程工具、API 调用等场景都适用）。

### 收藏提示词

登录后，卡片上会显示收藏按钮，点击加入我的收藏。

## 🔥 热门排序

按提示词的受欢迎程度排序，帮你发现热门内容。标签排序也会按热度调整，反映社区最新偏好。
