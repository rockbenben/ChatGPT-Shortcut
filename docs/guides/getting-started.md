# 开始上手

在与 AI 对话时，好的提示词能帮助你更快获得高质量的回答。AiShort 为你整理了大量实用提示词，复制粘贴即可使用。

## 基本使用

1. 在首页搜索或选择你需要的提示词
2. 点击卡片查看详情，或直接点击「复制」按钮
3. 打开任意 AI 模型，将提示词粘贴进去
4. 按照提示词的指引，补充你的具体问题
