# 开始上手

使用 AiShort，只需将选中的提示词复制到任意 AI 对话模型中，即可获取指定方向的输出。以下是具体步骤：

1. 搜索或选择你想要的提示词，然后点击提示词上的复制按钮。
2. 将提示词复制到 ChatGPT、Gemini 或任意 AI 对话模型中。
3. 根据对话提示，输入你的要求。
