# 开始上手

好的提示词决定 AI 输出的质量。AiShort 收录大量精选提示词，复制粘贴即用。

> 📋 **开始前你需要**：一个浏览器 + 一个能用的 AI 工具（ChatGPT、Claude 等对话页 / Cursor 等编程工具 / 自己的 API 调用——任选其一）。AiShort 网页版无需注册。

## 基本使用

1. 在首页搜索或浏览感兴趣的提示词
2. 找到合适的提示词，点「复制」按钮（或先点开卡片看完整内容）
3. 粘贴到任意 AI 工具——ChatGPT 等对话页、Cursor 等编程工具、API 调用等
4. 按提示词中的**占位符**（如 `[内容]`、`[主题]`）补充你的具体需求，发送给 AI

![](/img/docs/zh/how-to-use-aishort.webp)

> 💡 **小贴士**：大部分提示词用方括号 `[...]` 标记需要你补充的内容——把方括号里的占位文本换成你的具体内容即可。
