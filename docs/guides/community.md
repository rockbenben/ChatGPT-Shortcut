# 社区提示词

社区提示词版块展示由用户分享的自定义提示词。您可以在这里发现灵感，也可以分享自己的创意。

## 排序和筛选

社区页面提供多种筛选方式：

- **发布时间**：查看最新分享的提示词
- **投票排序**：查看获得最多赞成票的提示词
- **关键词搜索**：搜索特定主题的提示词

## 投票机制

通过赞/踩投票决定提示词的曝光度：

- **赞成**：帮助优质内容获得更多曝光
- **反对**：踩数达到一定数量后，提示词会被隐藏

**收藏联动**：收藏社区提示词时，系统会自动投一票赞成。

> 投票仅对登录用户开放，每人对同一提示词的投票以最近一次为准。

## 私密提示词

不想公开的创意可以设为私密：

- 创建或编辑提示词时，取消「分享到社区」选项
- 私密提示词仅自己可见
- 可随时切换公开/私密状态
