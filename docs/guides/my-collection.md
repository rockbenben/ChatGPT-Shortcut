# 我的收藏

登录后，**我的收藏**视图让你像管理书签一样组织喜欢的提示词——拖拽排序、自定义标签、按标签筛选，方便快速调用。

:::tip 在线版与离线版
在线版收藏数据存在服务器（需登录）；[离线部署版](./offline) 收藏数据存在浏览器本地（localStorage，无需登录），功能一致。
:::

## 视图切换

首页顶部的视图切换按钮提供两种模式：

- **我的收藏**：展示你的收藏 + 自定义提示词
- **提示词库**：浏览公共提示词库

浏览器会记住你的选择，下次访问自动切回上次的视图。

![我的收藏视图](/img/docs/my-collection-view.gif)

## 添加收藏

在卡片上点「收藏」按钮即可加入我的收藏。

收藏社区提示词会自动投一票赞成，帮优质内容获得更多曝光。

## 移除收藏

在我的收藏视图，点击卡片上的爱心图标即可取消收藏。

## 自定义标签

给收藏的提示词加标签，方便分类：

### 添加标签

1. 点击控制栏的标签图标
2. 选已有标签或点「+」新建
3. 新标签可自定义名称和颜色

### 管理标签

点「管理标签」进入管理界面：

- 改标签名称
- 换标签颜色（11 种预设色）
- 删除标签

![标签管理弹窗](/img/docs/my-collection-tag.png)

### 按标签筛选

我的收藏视图的筛选栏可按自定义标签筛选。

## 拖拽排序

在我的收藏视图支持拖拽调整卡片顺序：

1. 拖动卡片左侧的拖拽手柄
2. 移到目标位置后释放
3. 新顺序自动保存（在线版存服务器，离线版存浏览器本地）

排序仅在我的收藏视图可用。

![拖拽排序操作](/img/docs/my-collection-drag.gif)
