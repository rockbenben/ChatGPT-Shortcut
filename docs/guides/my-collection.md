# 我的收藏

登录后，您可以收藏喜欢的提示词并进行个性化管理。我的收藏视图提供了丰富的管理功能，帮助您高效组织和使用提示词。

## 视图切换

首页顶部提供视图切换按钮，支持两种模式：

- **我的收藏**：展示您收藏的提示词和自定义提示词
- **提示词库**：浏览公共提示词库，发现新内容

浏览器会自动记忆您的视图选择，下次访问时自动切换到上次的视图。

## 添加收藏

在提示词卡片上点击「收藏」按钮，即可将其添加到我的收藏。

收藏社区提示词时，系统会自动为其投一票赞成票，帮助优质内容获得更多曝光。

## 移除收藏

在我的收藏视图中，点击卡片上的爱心图标可取消收藏。

## 自定义标签

为收藏的提示词添加个人标签，方便分类管理：

### 添加标签

1. 点击卡片右上角的标签图标
2. 选择已有标签或点击「+」创建新标签
3. 新标签可自定义名称和颜色

### 管理标签

点击「管理标签」进入标签管理界面：

- 修改标签名称
- 更换标签颜色（11 种预设色可选）
- 删除不需要的标签

### 按标签筛选

在我的收藏视图的筛选栏中，可按自定义标签筛选内容。

## 拖拽排序

在我的收藏视图中支持拖拽调整卡片顺序：

1. 拖动卡片左侧的拖拽手柄
2. 移动到目标位置后释放
3. 新顺序自动保存到服务器

排序功能仅在我的收藏视图中可用。
