收藏与提交提示词

登录后，你可以自由添加或删除收藏的提示词，或添加个人提示词。

收藏提示词​

添加收藏​

登录后，系统默认的收藏标签将被移除，提示词旁边将显示「收藏」按钮。之后，你收藏或取消收藏时，都将实时显示收藏状态的提示。

移除收藏​

点击收藏提示词右侧的爱心图标，即可将其移除出收藏。

添加自定义提示词​

提交提示词后，提示词将显示在「我的提示词」标签中。
