# 如何使用浏览器扩展？

ChatGPT Shortcut 支持 18 种主流语言，扩展语言将根据你的浏览器环境自动设置。你也可以在设置页面手动更改语言。

![](/img/docs/extension-settings.png)

## 显示模式

扩展支持三种显示模式，按你的使用习惯选一种：

- **侧边栏模式**：在浏览器侧边栏打开——与网页并排显示，适合一边看 AI 对话页一边浏览提示词
- **弹窗模式**：点扩展图标弹出小窗——快速查看后即关，不占屏幕
- **独立窗口模式**：在独立的浮动窗口打开——可以拖到副屏或独立摆放

## 保持侧边栏打开

开启此选项后，侧边栏将在浏览器标签页切换时保持开启状态，方便持续使用。

## 深色模式

扩展支持深色模式，可在设置页面中开启「深色模式」选项，启用后界面将切换为深色外观。

![](/img/docs/extension-sidebar.png)

## 热键激活

使用 `Alt+Shift+S` 快捷键可以直接激活扩展。
