# 如何使用浏览器扩展？

ChatGPT Shortcut 支持 13 种主流语言，扩展语言将根据你的浏览器环境自动设置。你也可以在设置页面手动更改语言。

![](/img/docs/extension-settings.png)

## AiShort 侧边栏

AiShort 扩展目前采用侧边栏（Side Panel）形式运行，提供流畅的交互体验。

若在设置中开启"保持侧边栏打开"选项，侧边栏将在浏览器标签页切换时保持开启状态，方便持续使用。

![](/img/docs/extension-sidebar.png)

## 深色模式

扩展支持深色模式，可在设置页面中开启"深色模式"选项，启用后侧边栏和设置页面都将切换为深色外观。

## 热键激活

使用 `Alt+Shift+S` 快捷键可以直接激活 AiShort 侧边栏。
