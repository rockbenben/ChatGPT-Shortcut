# Chrome ZIP 插件本地安装指南

> 本教程假设你已从[安装入口](./README.md)下载并解压了 ZIP 安装包。
>
> ⚠️ **两个常见踩坑**：
> 1. 解压出的文件夹要**长期保留**（插件运行时会读取这个目录）——请先放到一个永久位置再继续
> 2. 拖入的是**整个解压文件夹**（不是 zip 文件本身、也不是文件夹里的单个文件）

## 开启开发者模式

打开 Chrome 的「管理扩展程序」页面，并开启「开发者模式」。

复制以下地址粘贴到浏览器地址栏，按回车打开。在页面右上角开启「开发者模式」。

```txt
chrome://extensions
```

![](https://img.newzone.top/2024-08-12-22-05-52.png?imageMogr2/format/webp)

## 安装插件

将解压出的文件夹拖入到扩展程序页面，即可完成安装。

![](https://img.newzone.top/2024-08-12-22-27-47.png?imageMogr2/format/webp)

安装成功后，可查看[插件使用教程](./usage)。
