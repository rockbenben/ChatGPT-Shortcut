Руководство по Локальной Установке Расширения ZIP для Chrome

Скачать Расширение​

Скачайте установочный пакет ZIP расширения ChatGPT Shortcut (ChatGPT_Shortcut-zip-3.x.x.zip) и распакуйте его. (Распакованную папку нужно сохранить на компьютере, поэтому не забудьте распаковать ее в постоянное место).

  • GitHub: GitHub Release
  • Скачивание в Китае: Alist Drive; Lanzou Cloud, пароль: 1qow

Включить Режим Разработчика​

Откройте страницу "Управление расширениями" в Chrome и включите "Режим разработчика".

Скопируйте следующий адрес и вставьте его в адресную строку браузера, затем нажмите Enter, чтобы открыть. Включите "Режим разработчика" в правом верхнем углу страницы.

chrome://extensions

Установить Расширение​

Перетащите распакованную папку на страницу расширений, чтобы завершить установку.

После успешной установки вы можете ознакомиться с Уроком по Использованию Плагина.
