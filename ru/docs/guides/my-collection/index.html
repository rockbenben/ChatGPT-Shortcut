Моя коллекция

После входа в систему «Моя коллекция» даёт удобное пространство в стиле закладок для любимых промптов — сортировка перетаскиванием, теги для контекста и фильтрация по тегу, когда нужно быстро найти нужное. Здесь же появляются созданные вами промпты.

Онлайн- и офлайн-версии

В онлайн-версии коллекции хранятся на сервере (требуется вход); в офлайн-версии — в локальном хранилище браузера (localStorage, вход не нужен). Функциональность одинаковая.

Переключение представления​

Переключатель в верхней части главной страницы предлагает два режима:

  • Моя коллекция: ваши собранные промпты и свои промпты
  • Обзор: публичная отборная библиотека

Браузер запомнит последний использованный режим — при следующем посещении вы продолжите с того же места.

Обзор коллекции​

Сводная карточка вверху представления «Моя коллекция» даёт быстрый обзор:

  • Четыре показателя: Всего / Мои промпты / Избранное / Теги
  • Значок уровня (L0–L9), который растёт по мере того, как вы делитесь промптами с сообществом

Добавление в коллекцию​

Нажмите кнопку-сердце «Добавить» на карточке, чтобы добавить промпт в «Мою коллекцию».

Добавление промпта из сообщества автоматически засчитывается как голос «за» — это привлекает внимание к качественному контенту.

Удаление из коллекции​

В представлении «Моя коллекция» нажмите иконку сердца на карточке, чтобы убрать промпт из коллекции.

Фильтрация и поиск​

Панель инструментов в верхней части представления коллекции помогает быстро найти нужное:

  • Сегментированный переключатель Все / Промпты / Избранное для фильтрации по типу
  • Метки пользовательских тегов — нажмите для фильтрации по тегу
  • Строка поиска (Ctrl + K для перехода) для поиска по ключевому слову

Пользовательские теги​

Добавляйте теги к собранным промптам для удобной классификации:

Добавление тега​

  1. Нажмите «Управление тегами» на панели инструментов
  2. Введите название тега и выберите один из 11 предустановленных цветов
  3. Сохраните — тег теперь доступен в панели фильтров

Управление тегами​

В диалоге «Управление тегами» можно:

  • Переименовать теги
  • Изменить цвет тега (11 предустановленных цветов)
  • Удалить теги

Сортировка перетаскиванием​

В представлении «Моя коллекция» доступна сортировка перетаскиванием:

  1. Возьмитесь за ручку перетаскивания на левом крае карточки
  2. Перетащите её в нужное положение
  3. Новый порядок сохраняется автоматически (на сервере — в онлайн-версии, в браузере — в офлайн-версии)
