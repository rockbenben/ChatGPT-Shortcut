계정 관리

AiShort는 언제든지 개인 데이터에 액세스할 수 있도록 다양한 로그인 방식을 제공합니다.

로그인 방식​

계정 비밀번호 로그인​

사용자 이름 또는 등록된 이메일과 비밀번호를 사용하여 로그인합니다.

비밀번호를 잊어버린 경우 로그인 페이지의 "비밀번호 찾기"를 클릭하고 등록된 이메일을 입력하면 시스템이 재설정 링크를 발송합니다.

Google 로그인​

로그인 페이지의 Google 아이콘을 클릭하여 Google 계정으로 원클릭 로그인합니다.

처음으로 Google 로그인을 사용하는 경우 별도의 가입 없이 자동으로 계정이 생성됩니다.

비밀번호 없는 로그인​

  1. 로그인 페이지에서 등록된 이메일 입력
  2. "로그인 링크 보내기" 클릭
  3. 이메일을 확인하고 링크를 클릭하여 로그인

이 방식은 비밀번호를 기억할 필요가 없으며 자주 로그인하지 않는 사용자에게 적합합니다.

계정 등록​

  1. 로그인 페이지의 "등록" 옵션 클릭
  2. 사용자 이름, 이메일 및 비밀번호 입력
  3. 등록 완료 후 자동 로그인

비밀번호 변경​

로그인 후 사용자 센터에서 비밀번호를 변경할 수 있습니다.

  1. 현재 비밀번호 입력
  2. 새 비밀번호 입력 및 확인
  3. 변경 사항 저장

캐시 지우기​

데이터 동기화 문제나 표시 이상이 발생한 경우:

  1. 사용자 센터 진입
  2. "캐시 지우기" 버튼 클릭
  3. 페이지가 자동 새로 고침되고 서버에서 데이터를 다시 가져옵니다

캐시를 지워도 컬렉션과 커스텀 프롬프트는 삭제되지 않으며 서버 데이터를 강제로 다시 동기화할 뿐입니다.
