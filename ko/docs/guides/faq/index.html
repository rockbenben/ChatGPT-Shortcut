자주 묻는 질문

왜 영어 프롬프트를 사용하나요?​

AI 모델은 일반적으로 영어 이해력이 더 뛰어나며 영어 프롬프트를 사용하면 더 안정적인 출력 결과를 얻을 수 있습니다.

한국어 프롬프트를 사용해도 괜찮은 결과를 얻을 수 있지만 동일한 한국어 프롬프트를 반복 입력할 때 결과에 큰 편차가 발생할 수 있습니다. 따라서 중요한 시나리오에서는 영어 프롬프트를 사용하는 것이 좋습니다.

한국어 답변 지정: 프롬프트 끝에 respond in Korean을 추가하면 한국어 답변을 받을 수 있습니다.

매번 프롬프트를 입력해야 하나요?​

API 사용: 프롬프트를 "system prompt"로 설정하여 후속 대화에 자동으로 적용할 수 있습니다.

웹 버전: 주제를 전환하지 않은 경우 ChatGPT는 현재 대화의 프롬프트 설정을 기억합니다. 답변이 예상에서 벗어나기 시작하면 AI가 프롬프트를 "잊어버린" 것이므로 다시 입력해야 합니다.

팁: 자주 사용하는 대화를 북마크로 저장하여 나중에 빠르게 액세스하세요.

AI가 거짓 정보를 출력하면 어떻게 해야 할까요?​

AI는 때때로 그럴듯해 보이지만 실제로는 잘못된 정보를 출력하는 "환각"을 일으킵니다. 사용 제안:

  1. 핵심 정보 확인: 특히 데이터, 인용, 코드 등
  2. 다중 최적화: AI가 답변을 다시 확인하고 최적화하도록 합니다
  3. 교차 검증: 다른 프롬프트나 모델을 사용하여 중요한 결론을 검증합니다

적절한 프롬프트는 AI 환각의 발생을 줄이는 데 도움이 될 수 있습니다.

프롬프트를 백업하려면 어떻게 해야 하나요?​

  1. 사용자 센터 진입
  2. "프롬프트 내보내기" 버튼 클릭
  3. 시스템이 자동으로 JSON 파일을 생성하여 다운로드

데이터 손실을 방지하기 위해 정기적으로 백업하는 것이 좋습니다.

프롬프트 효과가 이상적이지 않나요?​

  1. 최적화 시도: AI가 답변을 다시 최적화하도록 하세요
  2. 관점 변경: 다른 프롬프트를 사용하여 동일한 요구 사항을 표현해 보세요
  3. 피드백 및 공유: 문제를 발견하거나 좋은 프롬프트가 있는 경우 피드백을 통해 알리거나 커뮤니티에서 공유해 주세요
