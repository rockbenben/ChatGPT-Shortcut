자주 묻는 질문

AiShort은 AI 프롬프트 생성기인가요?​

아닙니다. AiShort은 시나리오별로 엄선하고 정리한 수작업 prompt 템플릿 라이브러리이며 LLM으로 prompt를 자동 생성하는 도구가 아닙니다. 모든 prompt는 선별을 거쳐 태그별로 분류되어 있고 한 번의 클릭으로 복사하여 ChatGPT / Claude / Gemini / DeepSeek 등 어떤 AI 대화 도구에도 붙여넣을 수 있습니다. 「요구사항으로부터 prompt를 자동 생성」하는 도구를 찾는다면 PromptPerfect, ChatGPT prompt generator 같은 별도 카테고리의 제품을 살펴보십시오.

AiShort은 무료인가요? API Key가 필요한가요?​

완전히 무료이며 API Key도 회원가입도 필요하지 않습니다. 프롬프트 탐색, 검색, 복사에는 어떠한 계정도 요구되지 않습니다.

회원가입 후에는 즐겨찾기 및 드래그 정렬, 커스텀 태그, 개인 프롬프트 생성·관리, 커뮤니티 공유와 투표, JSON 데이터 내보내기, 기기 간 동기화 등 확장 기능을 사용할 수 있습니다. 이 또한 모두 무료입니다. 코드는 전부 GitHub에 오픈소스로 공개되어 있습니다.

참고: AiShort은 프롬프트만 제공하므로 AI 모델 자체의 호출 요금은 사용하는 AI 서비스가 결정합니다(예: ChatGPT 무료판/유료판, API 과금 등).

프롬프트는 몇 개? 어떤 시나리오를 다루나요?​

AiShort에는 총 5000+ 개의 프롬프트가 등록되어 있으며, 두 개의 라이브러리로 나뉘어 있습니다:

  • 큐레이션 라이브러리 —— 수작업 선별, 25+ 시나리오 태그로 분류, 18개 언어 번역 완비. 태그: 작문 보조, IT/프로그래밍, 글/리포트, SEO/마케팅, 기업 직무, 학술/교사, 교육/학생, 언어/번역, 심리/사회, 사고 훈련, 생산성 도구, 터미널/인터프리터, 토론/연설, 리뷰/평가, 흥미 과학, 생활 백과, 의료 건강, 금융 어드바이저, 음악 예술, 철학/종교, 텍스트/단어, 재미있는 게임, 전문 어드바이저 등.
  • 커뮤니티 라이브러리 —— 대다수를 차지하며, 사용자가 지속적으로 투고, 인기순 또는 최신순 정렬 및 검색 필터 지원; 시나리오의 세분화와 범위가 큐레이션보다 더 넓습니다.

큐레이션에서 찾을 수 없으면 커뮤니티에서 다시 검색해 보세요.

어떤 AI 모델을 지원하나요? 모델 간 호환되나요?​

AiShort의 프롬프트는 범용적이며 프롬프트를 입력할 수 있는 모든 AI 환경에 적용됩니다. 대화 페이지뿐 아니라 Cursor 같은 프로그래밍 도구, API 호출, AI 에이전트 등에서도 그대로 사용할 수 있습니다. 아래는 대표적인 대화형 모델입니다.

  • 글로벌: ChatGPT, Gemini, Claude, Grok
  • 국내: HyperCLOVA X, Solar, Gauss, A.X
  • 중국 본토: DeepSeek, Tongyi Qianwen, Ernie Bot, Doubao, Kimi, ChatGLM, iFLYTEK Spark, Tencent Yuanbao
  • API 플랫폼: OpenRouter, SiliconFlow, Groq

더 많은 모델 목록은 시작하기 → 일반적인 AI 모델을 참조하십시오.

모델 간 호환성: 대부분의 텍스트 prompt는 모델에 관계없이 호환됩니다. 글쓰기, 번역, 프로그래밍, 질의응답 같은 범용 작업은 동일한 prompt를 주요 대규모 언어 모델에서 실행해도 유효한 결과를 얻을 수 있으며 AiShort의 모든 prompt는 기본적으로 특정 모델에 종속되지 않습니다. 다만 결과에는 차이가 있습니다. 글쓰기 계열에서는 Claude가 디테일이 더 섬세하고 ChatGPT-5는 지시를 더 충실히 따르는 편이며 프로그래밍에서는 GPT-5, Gemini Pro, DeepSeek이 각각 강점을 보입니다. 추론 집약 과제에서는 Claude Opus / Gemini Deep Thinking / o-series가 더 안정적입니다. 그림 생성 prompt(Midjourney / Stable Diffusion / DALL·E)는 호환되지 않으며 각각의 문법에 맞춰 조정해야 합니다.

프롬프트가 영어로 작성된 이유는?​

대다수 글로벌 AI 모델은 학습 데이터의 비중상 영어 이해력과 추론 안정성이 더 높아 출력이 더 안정적입니다. 또한 국내 IT·스타트업 현장에서는 이미 영문 기술 용어(deploy, sprint, async 등)가 표준으로 쓰이고 있어 영어 프롬프트와 자연스럽게 어울립니다.

한국어 프롬프트도 물론 사용 가능하지만 동일한 한국어 문장을 반복 실행하면 결과 편차가 큰 편입니다. 권장 패턴은 「영어로 지시 + 한국어로 답변」 조합으로 안정성과 가독성을 모두 잡는 방식입니다. 한국어 출력의 자연스러움이 중요한 경우(공식 문서·고객 응대 메일 등)에는 한국어 특화 모델(HyperCLOVA X, Solar 등)도 고려해 보십시오.

💡 한국어 답변이 필요하다면 프롬프트 끝에 Respond in Korean. 또는 한국어로 답변해 주십시오.를 추가하십시오.

매번 프롬프트를 입력해야 하나요?​

API 사용: 프롬프트를 system prompt로 설정하면 후속 대화에 자동으로 적용됩니다.

웹 버전: 대화를 전환하지 않는 한 ChatGPT는 현재 프롬프트를 기억합니다. 답변이 예상에서 벗어나기 시작하면 프롬프트를 「잊은」 것이므로 다시 한 번 붙여넣으면 됩니다.

팁: 자주 사용하는 대화를 브라우저 북마크로 저장하면 다음에 바로 열 수 있습니다.

어떻게 적합한 프롬프트를 고르나요?​

원하는 출력에서 역으로 찾으십시오. 글쓰기에는 write 또는 article 태그를, 코드 작성에는 code 또는 interpreter를, 번역에는 language를, 롤플레이에는 games를 사용하십시오. 태그에 익숙하지 않다면 키워드로 직접 검색하십시오. 각 prompt 카드에는 인기도(복사 횟수)가 표시되며 복사가 많은 prompt일수록 일반적으로 품질이 더 안정적입니다.

관련 프롬프트를 찾을 수 없는 이유는?​

홈페이지의 검색 범위는 엄선된 프롬프트 라이브러리(로그인 후에는 개인 프롬프트 포함)이며 커뮤니티에 공유된 프롬프트는 포함하지 않습니다.

짧은 키워드로 홈페이지에서 찾을 수 없는 경우 커뮤니티 프롬프트 페이지에서 다시 검색하십시오. 그곳에는 더 많은 사용자 공유 콘텐츠가 있습니다.

AI가 잘못된 정보를 출력하면 어떻게 하나요?​

AI는 때때로 「환각」을 일으켜 그럴듯해 보이지만 실제로는 잘못된 정보를 출력합니다. 대처 방법:

  1. 핵심 정보 확인: 특히 데이터, 인용, 코드 등
  2. 다중 최적화: AI가 답변을 다시 확인하고 최적화하도록 요청합니다
  3. 교차 검증: 다른 프롬프트나 모델을 사용하여 중요한 결론을 검증합니다

적절한 프롬프트는 AI 환각의 발생을 줄이는 데 도움이 됩니다.

프롬프트 효과가 좋지 않으면 어떻게 조정하나요?​

프롬프트를 바로 바꾸지 말고 먼저 다음 방향으로 조정해 보십시오.

  1. 대괄호 [플레이스홀더]를 더 구체적으로 작성: 스타일, 분량, 분야, 독자 페르소나 등의 세부 사항을 추가하십시오
  2. AI에게 최적화 요청: 만족스럽지 않은 답변에 대해 「더 X하게 다시 작성해 주십시오」 또는 「Y 스타일로 다시 써 주십시오」라고 추가 지시하면 보통 한두 번의 반복으로 목표에 근접합니다. AI에게 자신의 답변을 평가하고 개선하도록 시키는 방법도 효과적입니다
  3. 모델 교체 비교: 동일한 prompt라도 Claude Sonnet / ChatGPT / Gemini / DeepSeek에서 결과 차이가 클 수 있습니다. 글쓰기 계열은 보통 Claude가 더 섬세하고 프로그래밍 계열은 GPT-5 / DeepSeek이 더 안정적입니다
  4. 커뮤니티 활용: 커뮤니티 프롬프트에 더 적합한 버전이 있을 수 있습니다. 좋은 프롬프트가 있다면 그곳에 공유해 주십시오
  5. 피드백 제안: 문제를 발견하거나 개선 제안이 있다면 피드백을 환영합니다

프롬프트를 백업하는 방법은?​

  1. 「내 계정」 → 「데이터 관리 → 프롬프트 내보내기」 항목으로 이동
  2. 「데이터 내보내기」 버튼 클릭
  3. 시스템이 자동으로 JSON 파일을 생성하여 다운로드

데이터 손실을 방지하기 위해 정기적으로 백업하는 것이 좋습니다.

AiShort 프롬프트를 상업 프로젝트에 사용할 수 있나요?​

가능합니다. 다만 라이선스에 따라 경우가 나뉩니다.

  1. Awesome ChatGPT Prompts 출처의 콘텐츠는 CC0 퍼블릭 도메인을 따르며 어떠한 제한도 없습니다
  2. AiShort 자체 제작 및 커뮤니티 투고 콘텐츠는 기본적으로 CC BY-SA 4.0 오픈 라이선스를 적용합니다. 상업 이용, 재배포, 개작이 모두 허용되며 AiShort / 원저자의 표시(attribution)를 유지하고 다운스트림에도 동일한 개방을 허용해야 합니다
  3. prompt로 생성한 AI 콘텐츠의 저작권은 사용한 AI 서비스 제공사(OpenAI / Anthropic / Google 등)의 이용 약관에 따라 귀속되며 대부분의 경우 사용자에게 귀속됩니다
  4. prompt 카드 설명에 「개인용 한정」이라고 명시된 극소수 항목은 제외됩니다

프롬프트를 복사하면 내 데이터가 유출되나요?​

유출되지 않습니다. AiShort의 프롬프트는 정적 JSON 형태로 사이트 자원에 패키징되어 있으며 복사 동작 자체는 사용자 브라우저의 로컬 클립보드에서 완결됩니다.

사용자가 대괄호 플레이스홀더에 입력하는 구체적인 내용과 AI가 보내는 응답은 모두 사용자와 선택한 AI 플랫폼(ChatGPT / Claude / Gemini / DeepSeek 등) 사이에서 직접 전송되며 AiShort은 그 내용을 알 수 없습니다.

다만 한 가지 설명이 필요합니다. 복사 시 백엔드로 익명 카운트 이벤트(POST /cards/<id>/copy)가 전송됩니다. 이는 각 프롬프트의 인기도(카드에 표시되는 「복사 횟수」)를 집계하는 용도로만 사용됩니다. 카드 ID만 전송되며 사용자가 입력한 내용도, 개인 정보도, 사용자 신원과의 연결도 포함되지 않습니다.

로그인 후의 즐겨찾기 목록, 커스텀 프롬프트, 커뮤니티 투고는 기기 간 동기화를 위해 백엔드와 동기화되며 언제든 JSON으로 내보내거나 모든 데이터를 한 번에 삭제할 수 있습니다.
