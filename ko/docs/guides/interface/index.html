인터페이스 가이드

이 페이지에서는 AiShort 홈 화면의 핵심 세 영역, 즉 태그 영역(위쪽 시나리오 태그), 검색 영역(키워드 입력란), 프롬프트 영역(아래쪽 카드 목록)을 살펴보고, 이를 잘 엮어 원하는 프롬프트를 빠르게 찾는 방법을 알려드립니다.

🏷︎ 태그 필터링​

프롬프트를 분야와 기능에 따라 골라 볼 수 있습니다. 오른쪽 위의 OR / AND 전환 버튼이 여러 태그를 다루는 방식을 결정합니다.

  • OR (기본값): 고른 태그 중 하나라도 맞으면 표시합니다
  • AND: 고른 태그 전부와 맞아야 표시합니다

🔍 키워드 검색​

프롬프트의 제목, 설명, 본문을 모두 훑어 줍니다.

  • 홈 화면 검색은 엄선된 프롬프트만 다룹니다(커뮤니티 제외). 태그를 이미 골랐다면 그 안에서만 찾습니다
  • 로그인하면 즐겨찾기와 사용자 정의 프롬프트까지 검색 범위가 넓어집니다
  • 짧은 단어로 홈에서 못 찾았다면 커뮤니티 프롬프트 페이지에서 한 번 더 검색해 보세요

🔬 프롬프트 카드​

자세히 보기​

카드를 누르면 상세 창이 열려 전체 본문과 메모를 확인하고, 그대로 복사까지 한 번에 처리할 수 있습니다.

프롬프트 복사​

카드 아래쪽의 "복사" 버튼을 누른 뒤, 사용하시는 어떤 AI 도구에 붙여넣어도 바로 사용하실 수 있습니다 — 채팅 페이지, 코딩 도구, API 호출 등 어디서나 적용됩니다.

프롬프트 즐겨찾기​

로그인하면 카드에 즐겨찾기 버튼이 보입니다. 한 번 눌러 두면 바로 마이 컬렉션에 담깁니다.

🔥 인기순 정렬​

프롬프트를 인기 순서로 보여 줘서 지금 잘 쓰이는 글을 한눈에 찾을 수 있습니다. 태그 순서 역시 사용자들의 최신 선호도에 맞춰 함께 조정됩니다.
