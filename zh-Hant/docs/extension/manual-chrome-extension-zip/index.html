Chrome ZIP 插件本地安裝指南

本教程假設你已從安裝入口下載並解壓了 ZIP 安裝包。

⚠️ 兩個常見踩坑：

  1. 解壓出的文件夾要長期保留（插件運行時會讀取這個目錄）——請先放到一個永久位置再繼續
  2. 拖入的是整個解壓文件夾（不是 zip 文件本身、也不是文件夾裏的單個文件）

開啓開發者模式​

打開 Chrome 的「管理擴展程序」頁面，並開啓「開發者模式」。

複製以下地址粘貼到瀏覽器地址欄，按回車打開。在頁面右上角開啓「開發者模式」。

chrome://extensions

安裝插件​

將解壓出的文件夾拖入到擴展程序頁面，即可完成安裝。

安裝成功後，可查看插件使用教程。
