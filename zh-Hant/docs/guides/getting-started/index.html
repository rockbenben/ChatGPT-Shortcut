開始上手

在與 AI 對話時，好的提示詞能幫助你更快獲得高質量的回答。AiShort 爲你整理了大量實用提示詞，複製粘貼即可使用。

基本使用​

  1. 在首頁搜索或選擇你需要的提示詞
  2. 點擊卡片查看詳情，或直接點擊「複製」按鈕
  3. 打開任意 AI 模型，將提示詞粘貼進去
  4. 按照提示詞的指引，補充你的具體問題

快速入口​

我要...去哪裏
複製使用提示詞直接在首頁搜索或瀏覽
收藏和管理提示詞我的收藏
創建自己的提示詞自定義提示詞
註冊或登錄賬戶管理
發現社區分享社區提示詞
