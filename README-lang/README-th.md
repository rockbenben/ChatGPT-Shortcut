## 💎 ทำไมต้อง AiShort?

**ความต่างระหว่าง "ใช้ AI ได้" กับ "ใช้ AI เป็น" คือพรอมต์ที่ดีหนึ่งอัน**

คำถามเดียวกัน พรอมต์ต่างกัน — และผลลัพธ์จาก AI จะแตกต่างกันราวฟ้ากับดิน การเขียนพรอมต์ระดับผู้เชี่ยวชาญด้วยตัวเองต้องใช้เวลาหลายปีในการลองผิดลองถูก AiShort มอบห้องสมุดพรอมต์ที่ผ่านการพิสูจน์แล้วจากชุมชน ครอบคลุมการเขียน การเขียนโค้ด งานออฟฟิศ การเรียนรู้ ดีไซน์ การตลาด และอื่นๆ อีกมากมาย คัดลอก → วาง → ผลลัพธ์ระดับผู้เชี่ยวชาญ ทันที

ไม่ต้องสมัคร ไม่ต้องจ่ายเงิน ไม่ต้องติดตั้ง — เปิดแล้วใช้เลย

### ฟีเจอร์หลัก

🚀 **พรอมต์ที่คัดสรรแล้ว** — ครอบคลุมการเขียน การเขียนโค้ด งานออฟฟิศ และอื่นๆ — พร้อมคัดลอกและใช้งาน

🔍 **ค้นหาด้วยแท็ก + คำหลัก** — กรองด้วยแท็กตามสถานการณ์ หรือค้นหาด้วยคำหลัก

🌍 **18 ภาษา** — มีการแปล UI และพรอมต์ครบถ้วน พร้อมตอบกลับในภาษาของคุณ

📦 **พร้อมใช้งาน** — ไม่ต้องลงทะเบียน เปิดแล้วใช้เลย

### ฟีเจอร์ขั้นสูง (ต้องเข้าสู่ระบบ)

📂 **My Collection** — การเรียงลำดับแบบลากและวาง, การจัดหมวดหมู่ด้วยแท็กที่กำหนดเอง

✏️ **Custom Prompts** — สร้าง แก้ไข และจัดการพรอมต์ของคุณเอง

🗳️ **ชุมชน** — แบ่งปันพรอมต์ให้ชุมชน โหวต และพูดคุยในคอมเมนต์

📤 **ส่งออกข้อมูล** — ส่งออกพรอมต์ทั้งหมดเป็น JSON ในคลิกเดียว

🔐 **ตัวเลือกการเข้าสู่ระบบที่หลากหลาย** — รหัสผ่าน, Google, หรือลิงก์อีเมลแบบไม่ต้องใช้รหัสผ่าน

🏆 **ระดับ** — สะสมระดับ (L0–L9) เมื่อคุณแบ่งปันพรอมต์กับชุมชน

## 🧩 ส่วนขยายเบราว์เซอร์

เข้าถึงพรอมต์ของ AiShort ได้ทุกที่ด้วยส่วนขยายเบราว์เซอร์ รองรับ Chrome, Edge, และ Firefox — เปิดแถบด้านข้างด้วย `Alt + Shift + S`

- **Chrome**: [Chrome Web Store](https://chrome.google.com/webstore/detail/chatgpt-shortcut/blcgeoojgdpodnmnhfpohphdhfncblnj)
- **Edge**: [Microsoft Edge Addons](https://microsoftedge.microsoft.com/addons/detail/chatgpt-shortcut/hnggpalhfjmdhhmgfjpmhlfilnbmjoin)
- **Firefox**: [Firefox Add-ons](https://addons.mozilla.org/addon/chatgpt-shortcut/)
- **GitHub**: [Releases](https://github.com/rockbenben/ChatGPT-Shortcut/releases/latest)

หรือใช้สคริปต์ Tampermonkey [ChatGPT Shortcut Anywhere](https://greasyfork.org/scripts/482907-chatgpt-shortcut-anywhere) เพื่อเรียกแถบด้านข้าง AiShort บนเว็บไซต์ใดก็ได้

## 🚀 ติดตั้ง

ติดตั้งอินสแตนซ์ของคุณเองผ่าน Vercel, Cloudflare Pages, Docker หรือแบบ local ดู [คู่มือการติดตั้ง](https://www.aishort.top/th/docs/deploy) สำหรับรายละเอียดทั้งหมด

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https%3A%2F%2Fgithub.com%2Frockbenben%2FChatGPT-Shortcut%2Ftree%2Fmain)

> **ใช้งานออฟไลน์ / อินทราเน็ต?** มี [Offline Edition](https://www.aishort.top/th/docs/deploy/offline) ที่สร้างขึ้นสำหรับเครือข่ายองค์กรหรือหน่วยงานรัฐที่ปิดจากอินเทอร์เน็ต — ไม่ต้องใช้ backend หรือบัญชีผู้ใช้ ข้อมูลจัดเก็บในเบราว์เซอร์ พร้อมฟีเจอร์เรียกดู/ค้นหา/บันทึก/พรอมต์กำหนดเองเหมือนเดิมทุกประการ

> **เคล็ดลับ**: การคลิกเดียวเพื่อติดตั้งของ Vercel จะสร้างโปรเจกต์ใหม่ (ไม่ใช่การ fork) ดังนั้นการตรวจสอบอัปเดตจาก upstream จึงไม่ทำงาน หากต้องการซิงค์อัตโนมัติ ให้ fork repo ก่อน แล้วจึงนำเข้า fork นั้นใน Vercel — ดูวิธีการแบบเต็มได้ในคู่มือ [การติดตั้ง](https://www.aishort.top/th/docs/deploy/sync-updates)
