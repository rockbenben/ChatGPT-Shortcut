## ⚡ 30 秒快速開始

1. 打開 [aishort.top](https://www.aishort.top/zh-Hant/)
2. 搜索或瀏覽你需要的提示詞
3. 點擊「複製」，粘貼到任意 AI 工具——ChatGPT 等對話頁、Cursor 等編程工具、API 調用等場景都適用

![AiShort Interface](../static/img/docs/interface-home.png)

就這麼簡單！更多功能請查看[使用教程](https://www.aishort.top/zh-Hant/docs/guides/getting-started)。

## 💎 為什麼選擇 AiShort？

**會用 AI，和會用好 AI，差的是一段好提示詞。**

同樣的問題，專業的提示詞能讓 AI 輸出從泛泛而談變得精準專業——但自己寫出這種提示詞，往往要長期實踐與迭代。AiShort 收錄上千條已被社群驗證的精選提示詞，覆蓋寫作、編程、辦公、學習、設計、營銷等場景。**複製 → 粘貼 → 立刻獲得專業級輸出。**

無需註冊、無需付費、無需安裝——打開就用。

### 核心功能

🚀 **精選提示詞** — 覆蓋寫作、編程、辦公等場景，複製即用。

🔍 **標籤 + 關鍵詞搜索** — 按場景標籤篩選，或直接搜關鍵詞。

🌍 **18 種語言** — 界面與提示詞翻譯完整，可指定母語回覆。

📦 **開箱即用** — 無需註冊帳號，打開即可使用。

### 高級功能（登錄後）

📂 **我的收藏** — 拖拽排序，自定義標籤分類。

✏️ **自定義提示詞** — 創建、編輯、管理自己的提示詞。

🗳️ **社群互動** — 分享提示詞到社群，參與投票。

📤 **數據導出** — 所有提示詞一鍵導出為 JSON。

🔐 **多種登錄方式** — 帳號密碼、Google、無密碼郵件鏈接。

## 🧩 瀏覽器插件

AiShort 插件讓你隨時調用提示詞庫。支持 Chrome、Edge、Firefox，使用 `Alt + Shift + S` 快速喚出側邊欄。

- **Chrome**: [Chrome 應用商店](https://chrome.google.com/webstore/detail/chatgpt-shortcut/blcgeoojgdpodnmnhfpohphdhfncblnj)
- **Edge**: [Edge 插件商店](https://microsoftedge.microsoft.com/addons/detail/chatgpt-shortcut/hnggpalhfjmdhhmgfjpmhlfilnbmjoin)
- **Firefox**: [Firefox 附加組件](https://addons.mozilla.org/addon/chatgpt-shortcut/)
- **GitHub**: [下載地址](https://github.com/rockbenben/ChatGPT-Shortcut/releases/latest)

也可使用[油猴腳本](https://greasyfork.org/scripts/482907-chatgpt-shortcut-anywhere) 在任意網站調出 AiShort 側邊欄。

## 🚀 部署

支持通過 Vercel、Cloudflare Pages、Docker 或本地環境部署。詳見[部署指南](https://www.aishort.top/zh-Hant/docs/deploy)。

[![Vercel 部署](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https%3A%2F%2Fgithub.com%2Frockbenben%2FChatGPT-Shortcut%2Ftree%2Fmain)

> **提示**：Vercel 一鍵部署會創建新項目（而非 fork），導致上游更新檢測失效。要獲得自動同步，請先 fork 倉庫，再在 Vercel 導入 fork。詳細步驟見[部署指南](https://www.aishort.top/zh-Hant/docs/deploy#開啟同步更新)。

## 🤝 貢獻指南

歡迎各種形式的貢獻：

- **提交提示詞** 或 **報告 bug** → 提一個 [GitHub Issue](https://github.com/rockbenben/ChatGPT-Shortcut/issues/new)
- **提交 PR** → fork 倉庫、創建分支、發起 pull request
- **新增翻譯** 或 **改進文檔** → 見 `i18n/` 和 `docs/` 文件夾
- **Star ⭐ 並分享** 幫助更多人發現有用的提示詞

本地開發環境搭建，參考[部署指南](https://www.aishort.top/zh-Hant/docs/deploy)。
