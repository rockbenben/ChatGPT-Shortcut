常见问题
AI 评分和真考官一致吗？
评分维度(Task Response/Coherence/Lexical/Grammar)框架准确,具体分数有时偏松或偏紧 0.5 分。作为自我练习参考没问题,真正的官方评分需要做模考或请外教给意见。
和 ID 249 雅思写作②有什么区别？
两个 prompt 是同一个作者的两个版本,细节配置略有不同。建议两个都试一次,看哪个输出的评分逻辑更清晰、例文更贴近考官口味,挑一个固定用,两个反复切换反而学不专。
如何使用这个提示词？
