সাধারণ প্রশ্ন

AiShort কি একটি AI প্রম্পট জেনারেটর?​

না। AiShort হলো পরিস্থিতি অনুসারে নির্বাচিত ও সংগঠিত মানব-নির্মিত prompt টেমপ্লেট লাইব্রেরি, এটি LLM দিয়ে স্বয়ংক্রিয়ভাবে prompt তৈরি করার টুল নয়। প্রতিটি prompt যাচাই করে ট্যাগ অনুযায়ী শ্রেণীবদ্ধ করা হয়েছে — এক ক্লিকে কপি করে ChatGPT / Claude / Gemini / DeepSeek সহ যেকোনো AI চ্যাট টুলে পেস্ট করলেই হলো। "প্রয়োজন থেকে স্বয়ংক্রিয়ভাবে prompt তৈরি" এমন টুল চাইলে PromptPerfect, ChatGPT prompt generator আলাদা ধরনের প্রোডাক্ট।

AiShort কি বিনামূল্যে? API Key দরকার কি?​

সম্পূর্ণ বিনামূল্যে, API Key লাগে না, রেজিস্ট্রেশনও দরকার নেই। প্রম্পট ব্রাউজ, অনুসন্ধান ও কপি করতে কোনো অ্যাকাউন্ট প্রয়োজন হয় না।

রেজিস্ট্রেশন করলে সংগ্রহ ও ড্র্যাগ সর্টিং, কাস্টম ট্যাগ, ব্যক্তিগত প্রম্পট তৈরি ও পরিচালনা, কমিউনিটি শেয়ারিং ও ভোটিং, JSON ডেটা এক্সপোর্ট, একাধিক ডিভাইস সিঙ্ক — এসব বাড়তি ফিচার আনলক হবে, এগুলোও বিনামূল্যে। সম্পূর্ণ কোড GitHub-এ ওপেন সোর্স।

দ্রষ্টব্য: AiShort শুধু প্রম্পট সরবরাহ করে, AI মডেল কল করতে খরচ হবে কিনা তা আপনার ব্যবহৃত AI পরিষেবার উপর নির্ভর করে (যেমন ChatGPT-এর ফ্রি/পেইড সংস্করণ, API বিলিং ইত্যাদি)।

কতগুলি প্রম্পট আছে? কী কী পরিস্থিতিতে কভার করে?​

AiShort-এ মোট 5000+ প্রম্পট রয়েছে, দুটি লাইব্রেরিতে সংগঠিত:

  • কিউরেটেড লাইব্রেরি —— হাত দিয়ে বাছাই করা, 25+ পরিস্থিতি ট্যাগ দিয়ে শ্রেণীবদ্ধ, 18টি ভাষায় সম্পূর্ণ অনুদিত। ট্যাগ: লেখার সহায়তা, IT/প্রোগ্রামিং, প্রবন্ধ/প্রতিবেদন, SEO/বিপণন, ব্যবসায়িক কাজ, একাডেমিক/শিক্ষক, শিক্ষা/ছাত্র, ভাষা/অনুবাদ, মনোবিজ্ঞান/সামাজিক, চিন্তা প্রশিক্ষণ, উৎপাদনশীলতা সরঞ্জাম, টার্মিনাল/ইন্টারপ্রেটার, বিতর্ক/বক্তৃতা, পর্যালোচনা/সমালোচনা, মজার বিজ্ঞান, জীবন বিশ্বকোষ, চিকিৎসা স্বাস্থ্য, আর্থিক পরামর্শদাতা, সঙ্গীত/শিল্প, দর্শন/ধর্ম, পাঠ্য/শব্দ, মজার গেম, পেশাদার পরামর্শদাতা ইত্যাদি।
  • সম্প্রদায় লাইব্রেরি —— অধিকাংশই, ব্যবহারকারীদের দ্বারা ক্রমাগত যোগ করা হয়, জনপ্রিয়তা বা নতুনত্ব অনুসারে সাজানো যায় এবং অনুসন্ধান সমর্থিত; কিউরেটেড-এর চেয়ে আরও সূক্ষ্ম এবং বিস্তৃত পরিস্থিতি কভার করে।

কিউরেটেড-এ না পেলে সম্প্রদায়ে অনুসন্ধান করুন।

কোন AI মডেলগুলি সমর্থিত? প্রম্পট কি মডেল জুড়ে কাজ করে?​

AiShort-এর প্রম্পটগুলি সর্বজনীন, যেকোনো পরিস্থিতিতে যেখানে prompt ইনপুট করা যায়, সেখানেই কাজ করে — শুধু চ্যাট পেজ নয়, Cursor-এর মতো কোডিং টুল, API কল, AI এজেন্ট ইত্যাদিও অন্তর্ভুক্ত। সাধারণ চ্যাট-ধরনের মডেল:

  • আন্তর্জাতিক: ChatGPT, Gemini, Claude, Grok
  • চীনা: DeepSeek, Tongyi Qianwen, ERNIE Bot, Doubao, Kimi, ChatGLM, Spark Desk, Tencent Yuanbao
  • API প্ল্যাটফর্ম: OpenRouter, SiliconFlow, Groq

আরও মডেলের তালিকার জন্য শুরু করুন → সাধারণ AI মডেল দেখুন।

মডেল ক্রস-সামঞ্জস্যতা: প্রায় সব টেক্সট-ধরনের prompt মডেল জুড়ে কাজ করে — লেখা, অনুবাদ, প্রোগ্রামিং, প্রশ্নোত্তরের মতো সাধারণ কাজে একই prompt প্রধান বড় ভাষা মডেলগুলিতে কার্যকর ফলাফল দেয়, AiShort-এর প্রতিটি prompt ডিফল্টভাবে কোনো নির্দিষ্ট মডেলের সাথে বাঁধা নেই। তবে ফলাফলে পার্থক্য থাকে: লেখার ক্ষেত্রে Claude সাধারণত আরও সূক্ষ্ম, ChatGPT-5 নির্দেশ আরও কড়াকড়ি মানে; প্রোগ্রামিং-এ GPT-5, Gemini Pro, DeepSeek প্রত্যেকের নিজস্ব শক্তি আছে; যুক্তিনির্ভর কঠিন প্রশ্নে Claude Opus / Gemini Deep Thinking / o-series সাধারণত আরও স্থিতিশীল। চিত্রাঙ্কনের prompt (Midjourney / Stable Diffusion / DALL·E) সর্বজনীন নয়, প্রত্যেকের নিজস্ব সিনট্যাক্স অনুযায়ী সমন্বয় করতে হয়।

প্রম্পট কেন ইংরেজিতে লেখা?​

AI মডেলগুলি ইংরেজি আরও সঠিকভাবে বোঝে, আউটপুটও আরও স্থিতিশীল হয়। বাংলা প্রম্পটও কাজ করে, কিন্তু একই বাংলা প্রম্পট একাধিকবার চালালে ফলাফলে বড় পার্থক্য দেখা দিতে পারে। গুরুত্বপূর্ণ ক্ষেত্রে ইংরেজি ব্যবহারের পরামর্শ দেওয়া হয়।

💡 বাংলায় উত্তর চান? প্রম্পটের শেষে respond in Bengali যোগ করুন।

কি প্রতিবার প্রম্পট দিতে হবে?​

API কল: প্রম্পটটিকে system prompt হিসাবে সেট করুন, পরবর্তী কথোপকথনে স্বয়ংক্রিয়ভাবে কার্যকর হবে।

ওয়েব সংস্করণ: কথোপকথন পরিবর্তন না করলে ChatGPT বর্তমান প্রম্পট মনে রাখবে। উত্তর যদি বিচ্যুত হতে শুরু করে, বুঝতে হবে এটি "ভুলে গেছে" — আবার পেস্ট করে দিন।

ছোট্ট টিপ: প্রায়ই ব্যবহৃত কথোপকথনগুলিকে ব্রাউজার বুকমার্ক হিসাবে সংরক্ষণ করুন, পরের বার সরাসরি খুলুন।

সঠিক প্রম্পট কীভাবে বেছে নেব?​

যে আউটপুট চান সেটার ভিত্তিতে উল্টোভাবে খুঁজুন: প্রবন্ধ লিখতে চাইলে write বা article ট্যাগ; কোড লিখতে চাইলে code বা interpreter; অনুবাদ করতে চাইলে language; রোলপ্লে করতে চাইলে games; ট্যাগগুলোর সাথে পরিচিত না হলে সরাসরি কীওয়ার্ড দিয়ে অনুসন্ধান করুন। প্রতিটি prompt কার্ডে জনপ্রিয়তা (কপি সংখ্যা) দেখা যায়, বেশি কপি হওয়াগুলোর মান সাধারণত বেশি স্থিতিশীল।

সম্পর্কিত প্রম্পট কেন খুঁজে পাচ্ছি না?​

হোমপেজ অনুসন্ধানের পরিধি হলো নির্বাচিত প্রম্পট লাইব্রেরি (লগইনের পরে আপনার ব্যক্তিগত প্রম্পটও যুক্ত হয়), কমিউনিটি দ্বারা শেয়ার করা প্রম্পট এতে অন্তর্ভুক্ত নয়।

ছোট কীওয়ার্ড দিয়ে হোমপেজে কিছু না পেলে কমিউনিটি প্রম্পট পৃষ্ঠায় আবার অনুসন্ধান করুন — সেখানে ব্যবহারকারীদের শেয়ার করা আরও বিষয়বস্তু আছে।

AI ভুল তথ্য দিলে কী করব?​

AI কখনও কখনও "হ্যালুসিনেশন" তৈরি করে, যা যুক্তিসঙ্গত মনে হলেও আসলে ভুল। সমাধানের উপায়:

  1. মূল তথ্য যাচাই করুন: বিশেষ করে ডেটা, উদ্ধৃতি, কোড ইত্যাদি
  2. মাল্টি-রাউন্ড অপ্টিমাইজেশন: AI-কে তার উত্তর পুনরায় চেক ও অপ্টিমাইজ করতে বলুন
  3. ক্রস-ভ্যালিডেশন: গুরুত্বপূর্ণ সিদ্ধান্ত যাচাই করতে বিভিন্ন প্রম্পট বা মডেল ব্যবহার করুন

উপযুক্ত প্রম্পট AI হ্যালুসিনেশন কমাতে সাহায্য করে।

প্রম্পট ভালো কাজ করছে না — কীভাবে সমন্বয় করব?​

সরাসরি প্রম্পট পরিবর্তন না করে, প্রথমে এই দিকগুলো থেকে সমন্বয় করুন:

  1. স্কোয়ার ব্র্যাকেট [প্লেসহোল্ডার] আরও সুনির্দিষ্ট করুন — শৈলী, দৈর্ঘ্য, ক্ষেত্র, পাঠকের প্রোফাইল ইত্যাদির বিবরণ যোগ করুন
  2. AI-কে অপ্টিমাইজেশন জিজ্ঞাসা করুন: অসন্তোষজনক উত্তরের ক্ষেত্রে "আরও X করুন" বা "Y শৈলীতে আবার লিখুন" বলুন, AI সাধারণত এক-দুই রাউন্ডের মধ্যেই লক্ষ্যের কাছাকাছি পৌঁছায়; AI-কে নিজের উত্তর স্কোর করে উন্নত করতেও বলতে পারেন
  3. মডেল সুইচ করে তুলনা করুন: একই prompt Claude Sonnet / ChatGPT / Gemini / DeepSeek-এ ফলাফলের পার্থক্য কখনও কখনও অনেক বেশি — লেখায় Claude সাধারণত সূক্ষ্মতর, প্রোগ্রামিং-এ GPT-5 / DeepSeek স্থিতিশীল
  4. কমিউনিটি চেষ্টা করুন: কমিউনিটি প্রম্পট-এ আরও উপযুক্ত সংস্করণ থাকতে পারে — আপনার ভালো প্রম্পটগুলিও সেখানে শেয়ার করতে স্বাগতম
  5. মতামত পাঠান: সমস্যা বা উন্নতির পরামর্শ আছে? প্রতিক্রিয়া-এ জানান

আমার প্রম্পট কীভাবে ব্যাকআপ করব?​

  3. সিস্টেম স্বয়ংক্রিয়ভাবে ডাউনলোডের জন্য একটি JSON ফাইল তৈরি করবে
