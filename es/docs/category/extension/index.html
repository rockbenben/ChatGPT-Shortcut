Extensión del Navegador

📄️Instalar Extensión del Navegador

Usa los prompts de AiShort directamente desde la barra lateral en páginas de chat de IA como ChatGPT, Gemini, Claude y Doubao, sin cambiar de pestaña para copiar y pegar. Instalación con un clic en Chrome, Edge y Firefox.

📄️Cómo usar la extensión

Domina la extensión AI Short! Activa la barra lateral con Alt+Shift+S.

📄️Guía de Instalación CRX para Chrome

Instala fácilmente el archivo CRX en modo desarrollador. Instrucciones de arrastrar y soltar y soluciones a problemas comunes.

📄️Guía de Instalación ZIP para Chrome

Alternativa al CRX. Guía paso a paso para instalar la extensión de Chrome manualmente con el archivo ZIP descomprimido.

📄️Configuración de Extensión de Firefox

Configura Firefox correctamente. Aprende a anclar la extensión y autorizar la barra lateral en ChatGPT y Gemini.
