Prompt ที่กำหนดเอง

หลังเข้าสู่ระบบ คุณสามารถสร้าง prompt ของตัวเองสำหรับการเข้าถึงอย่างรวดเร็วในภายหลัง Prompt ที่กำหนดเองจะถูกบันทึกไปยังบัญชีของคุณและแสดงในมุมมองคอลเลกชันของฉัน

สร้าง Prompt​

  1. คลิกปุ่ม "เพิ่ม Prompt" ที่มุมขวาบนของหน้า
  2. กรอกข้อมูล prompt:
    • หัวข้อ: คำอธิบายสั้นๆ เกี่ยวกับวัตถุประสงค์ของ prompt
    • คำอธิบาย: เนื้อหาทั้งหมดของ prompt
    • หมายเหตุ: คำแนะนำการใช้งานหรือบันทึก (ไม่บังคับ)
  3. เลือกว่าจะแบ่งปันไปยังชุมชนหรือไม่
  4. คลิกส่ง

แก้ไข Prompt​

ในมุมมองคอลเลกชันของฉัน คลิกการ์ด prompt ที่คุณสร้างเพื่อ:

  • แก้ไขหัวข้อ คำอธิบาย และหมายเหตุ
  • สลับสถานะการแบ่งปัน (สาธารณะ/ส่วนตัว)
  • บันทึกการเปลี่ยนแปลง

ลบ Prompt​

คลิกปุ่ม "ลบ" ในอินเทอร์เฟซแก้ไขเพื่อลบ prompt การลบไม่สามารถยกเลิกได้ โปรดดำเนินการด้วยความระมัดระวัง

แบ่งปันไปยังชุมชน​

เมื่อสร้างหรือแก้ไข prompt ให้เลือกตัวเลือก "แบ่งปันไปยังชุมชน":

  • สาธารณะ: Prompt จะแสดงบนหน้าชุมชน ผู้ใช้คนอื่นสามารถเห็นและรวบรวมได้
  • ส่วนตัว: มองเห็นได้เฉพาะตัวคุณเอง จะไม่ปรากฏบนหน้าชุมชน

คุณสามารถแก้ไขสถานะการแบ่งปันได้ตลอดเวลา

ส่งออกการสำรองข้อมูล​

ส่งออก prompt ทั้งหมดที่คุณสร้าง:

  1. เข้าศูนย์ผู้ใช้
  2. คลิกปุ่ม "ส่งออก Prompt"
  3. ระบบจะสร้างไฟล์ JSON และดาวน์โหลดโดยอัตโนมัติ

นำเข้า Prompt​

นำเข้า prompt และรายการโปรดจากไฟล์ JSON:

  1. เข้าศูนย์ผู้ใช้
  2. คลิกปุ่ม "นำเข้า Prompt"
  3. เลือกไฟล์ JSON
  4. ระบบจะรวมข้อมูลโดยอัตโนมัติ

เอกสารที่เกี่ยวข้อง​

  • คอลเลกชันของฉัน - การจัดการคอลเลกชันและแท็ก
  • Prompt ชุมชน - แบ่งปันและโหวต
  • จัดการบัญชี - เข้าสู่ระบบและการตั้งค่า
