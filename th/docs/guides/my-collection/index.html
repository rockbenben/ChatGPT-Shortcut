คอลเลกชันของฉัน

เมื่อเข้าสู่ระบบแล้ว คอลเลกชันของฉัน จะมอบพื้นที่คล้ายบุ๊กมาร์กส่วนตัวสำหรับ prompt ที่คุณชอบ — ลากวางเพื่อจัดลำดับ ติดแท็กเพื่อจัดหมวดหมู่ และกรองตามแท็กเมื่อต้องการหยิบใช้งานได้อย่างรวดเร็ว Prompt ที่คุณสร้างเองก็จะปรากฏที่นี่เช่นกัน

เวอร์ชันออนไลน์และออฟไลน์

เวอร์ชันออนไลน์เก็บข้อมูลคอลเลกชันไว้บนเซิร์ฟเวอร์ (ต้องเข้าสู่ระบบ) ส่วน เวอร์ชันออฟไลน์ เก็บข้อมูลคอลเลกชันไว้ในเบราว์เซอร์ (localStorage ไม่ต้องเข้าสู่ระบบ) ฟังก์ชันการใช้งานเหมือนกัน

การสลับมุมมอง​

ปุ่มสลับมุมมองที่ด้านบนของหน้าแรกมีสองโหมด:

  • คอลเลกชันของฉัน: แสดง prompt ที่รวบรวมและ prompt ที่กำหนดเองของคุณ
  • สำรวจ: เรียกดูคลัง prompt สาธารณะที่คัดสรรไว้

เบราว์เซอร์จะจำมุมมองที่คุณใช้ล่าสุด เพื่อให้คุณกลับมาต่อได้ทันที

ภาพรวมคอลเลกชัน​

การ์ดสรุปที่ด้านบนของมุมมองคอลเลกชันของฉันจะแสดงข้อมูลเด่นในทันที:

  • สถิติสี่รายการ — รวม / พรอมต์ ของฉัน / รายการโปรด / แท็กที่กำหนดเอง
  • ป้ายระดับ (L0–L9) ที่เลื่อนขึ้นตามจำนวน prompt ที่คุณแบ่งปันสู่ชุมชน

เพิ่มในคอลเลกชัน​

คลิกปุ่มหัวใจ "รวบรวม" บนการ์ดเพื่อเพิ่มใน คอลเลกชันของฉัน

การรวบรวม prompt ชุมชนจะนับเป็นการโหวตขึ้นให้โดยอัตโนมัติ ช่วยให้เนื้อหาคุณภาพได้รับการมองเห็นมากขึ้น

ลบจากคอลเลกชัน​

ในมุมมองคอลเลกชันของฉัน คลิกไอคอนหัวใจบนการ์ดเพื่อยกเลิกการรวบรวม

กรองและค้นหา​

แถบเครื่องมือที่ด้านบนของมุมมองคอลเลกชันช่วยให้คุณค้นหาสิ่งที่ต้องการได้รวดเร็ว:

  • ปุ่มสลับ ทั้งหมด / พรอมต์ / รายการโปรด เพื่อกรองตามประเภท
  • ชิปแท็กที่กำหนดเอง — คลิกเพื่อกรองตามแท็ก
  • ช่องค้นหา (กด Ctrl + K เพื่อโฟกัส) สำหรับค้นหาคำสำคัญ

แท็กที่กำหนดเอง​

เพิ่มแท็กให้ prompt ที่รวบรวม เพื่อสะดวกในการจัดหมวดหมู่:

เพิ่มแท็ก​

  1. คลิก "จัดการแท็ก" บนแถบเครื่องมือ
  2. ป้อนชื่อแท็กและเลือกหนึ่งใน 11 สีที่ตั้งไว้
  3. บันทึก — แท็กจะปรากฏในแถบกรองทันที

จัดการแท็ก​

ในหน้าต่าง "จัดการแท็ก" คุณสามารถ:

  • เปลี่ยนชื่อแท็ก
  • เปลี่ยนสีแท็ก (11 สีที่ตั้งไว้)
  • ลบแท็ก

การลากวางจัดเรียง​

มุมมองคอลเลกชันของฉันรองรับการลากวางเพื่อปรับลำดับ:

  1. จับที่จับลากที่ขอบซ้ายของการ์ด
  2. วางลงที่ตำแหน่งเป้าหมาย
  3. ลำดับใหม่จะบันทึกอัตโนมัติ (ที่เซิร์ฟเวอร์สำหรับเวอร์ชันออนไลน์ ที่เบราว์เซอร์สำหรับเวอร์ชันออฟไลน์)

เอกสารที่เกี่ยวข้อง​

  • Prompt ที่กำหนดเอง - สร้างและจัดการ prompt
  • คู่มืออินเทอร์เฟซ - ค้นหาและกรอง
  • จัดการบัญชี - เข้าสู่ระบบและการตั้งค่า
