คู่มือการใช้งาน

📄️คู่มืออินเทอร์เฟซ

ใช้งานหน้าหลัก AI Short ให้คล่อง ค้นหา prompt ด้วยการกรองแท็กและค้นหาคำสำคัญ ดูรายละเอียดการ์ดและคัดลอก prompt ที่ชอบเข้าคอลเลกชันได้ในไม่กี่คลิก

📄️บัญชี

เข้าสู่ระบบด้วย Google คลิกเดียว หรือใช้ลิงก์เข้าสู่ระบบทางอีเมลแบบไม่ต้องจำรหัสผ่าน เข้าถึงคอลเลกชันส่วนตัวข้ามอุปกรณ์ ลืมรหัสผ่านก็รีเซ็ตได้ทันที พร้อมจัดการข้อมูลบัญชีในที่เดียว

📄️คอลเลกชันของฉัน

รวบรวม prompt AI ที่ชอบ จัดกลุ่มด้วยแท็กที่กำหนดเอง ลากวางเพื่อจัดลำดับใหม่ ซิงค์ข้ามอุปกรณ์ สร้างคลัง prompt ส่วนตัวที่หยิบใช้ได้ทันที

📄️Prompt ที่กำหนดเอง

สร้าง prompt AI ของคุณเอง บันทึกในบัญชีส่วนตัวเพื่อเรียกใช้ได้ทุกเมื่อ รองรับการแบ่งปันในชุมชนหรือเก็บเป็นส่วนตัว ส่งออกข้อมูลสำรองได้ด้วยคลิกเดียว

📄️Prompt ชุมชน

สำรวจ prompt AI ที่ชุมชนช่วยกันคัดสรรผ่านระบบโหวต ดัน prompt คุณภาพให้คนเห็น แบ่งปันไอเดียของคุณเองให้คนอื่นได้ลองใช้ หรือเก็บเป็นส่วนตัวให้เห็นเฉพาะคุณก็ได้
