การติดตั้งโปรเจกต์

เหมาะสำหรับ: นักพัฒนาที่ต้องการโฮสต์หรือปรับแต่ง AiShort ด้วยตนเอง ผู้ใช้ทั่วไปสามารถใช้งาน aishort.top ได้เลย โดยไม่จำเป็นต้องอ่านส่วนนี้

เลือกรูปแบบการติดตั้ง​

เลือกรูปแบบที่เหมาะกับความต้องการของคุณ:

รูปแบบBackendหมายเหตุ
มาตรฐาน (ค่าเริ่มต้น)ใช้ backend ส่วนกลางของทางการร่วมกันหลังจาก fork แล้วสามารถปรับชื่อไซต์ คำอธิบาย prompt ฯลฯ ได้ (ดู การกำหนดค่า) ฟีเจอร์ล็อกอิน รายการโปรด ชุมชน และการซิงค์พร้อมใช้งานทันที
เวอร์ชันออฟไลน์ไม่มี backend ข้อมูลเก็บในเบราว์เซอร์ท้องถิ่นสำหรับเครือข่ายองค์กรหรือภาครัฐที่ไม่ได้เชื่อมต่ออินเทอร์เน็ต ไม่จำเป็นต้องมีบัญชี
self-hosted backend สมบูรณ์Backend อิสระของตัวเองเมื่อต้องการระบบบัญชีอิสระ ความเป็นเจ้าของข้อมูลเต็มรูปแบบ และชุมชนส่วนตัว

สองรูปแบบแรกอธิบายไว้ในคู่มือนี้ สำหรับรูปแบบที่สาม เนื่องจาก backend service ยังไม่ได้เปิดเป็น open-source กรุณาส่งอีเมลหาผู้พัฒนา พร้อมคำอธิบายสั้นๆ เกี่ยวกับกรณีการใช้งานและขนาดโปรเจกต์ของคุณ เพื่อรับแผนการติดตั้งและการสนับสนุน

เอกสารการติดตั้ง​

แบ่งตามขั้นตอนการติดตั้งออกเป็นหัวข้อต่อไปนี้ เลือกอ่านได้ตามต้องการ:

  • การติดตั้งแบบมาตรฐาน —— ใช้ backend ส่วนกลางของทางการร่วมกัน รองรับการ build ในเครื่อง, Vercel, Cloudflare Pages และ Docker
  • เวอร์ชันออฟไลน์ (อินทราเน็ตองค์กร) —— โซลูชันออฟไลน์สำหรับเครือข่ายองค์กรหรือภาครัฐที่ไม่ได้เชื่อมต่ออินเทอร์เน็ต ไม่ต้องมี backend หรือบัญชี
  • การกำหนดค่าและการปรับแต่ง —— แก้ไขชื่อไซต์ คำอธิบาย prompt และการเชื่อมต่อ backend ที่กำหนดเอง
  • การซิงค์ Fork ให้ทันสมัย —— ให้ fork ของคุณซิงค์อัปเดตจาก upstream โดยอัตโนมัติ ไม่ให้ฟีเจอร์ล้าหลัง
