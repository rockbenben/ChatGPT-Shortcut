คู่มือติดตั้งส่วนขยาย Chrome CRX ในเครื่อง

ดาวน์โหลดแพ็คเกจ CRX​

ดาวน์โหลดแพ็คเกจติดตั้ง crx ของ ChatGPT Shortcut (ChatGPT_Shortcut-crx-3.x.x.zip) และแตกไฟล์ (ไฟล์ crx อยู่ในโฟลเดอร์ที่แตกแล้ว)

  • GitHub: GitHub Release

เปิดโหมดนักพัฒนา​

เปิดหน้า "จัดการส่วนขยาย" ของ Chrome และเปิด "โหมดนักพัฒนา"

คัดลอกที่อยู่ต่อไปนี้และวางในแถบที่อยู่ของเบราว์เซอร์ จากนั้นกด Enter เพื่อเปิด เปิด "โหมดนักพัฒนา" ที่มุมขวาบนของหน้า

chrome://extensions

ติดตั้งส่วนขยาย​

ติดตั้งส่วนขยาย ChatGPT Shortcut (หมายเหตุ ⚠️: คุณต้องลากไฟล์ .crx เข้ามา อย่าคลิก "โหลดส่วนขยายที่แตกแล้ว")

หลังจากติดตั้งสำเร็จ ตรวจสอบ บทเรียนการใช้งานปลั๊กอิน

มีปัญหาในการติดตั้ง?​

  1. ผู้ใช้ Windows: ตรวจสอบว่าคุณได้แตกไฟล์แพ็คเกจติดตั้งที่ดาวน์โหลดแล้วหรือยัง (แทนที่จะดับเบิลคลิกเพื่อเปิด)?

  2. เปิด "โหมดนักพัฒนา" แล้วหรือยัง? ถ้ายังไม่ได้ ดูขั้นตอนที่ 2

  3. คุณลากไฟล์ crx เข้าไปในหน้า "ส่วนขยาย" แล้วหรือยัง? หมายเหตุ ⚠️: อย่าคลิก "โหลดส่วนขยายที่แตกแล้ว" คุณต้องลากไฟล์ crx

  4. เบราว์เซอร์ไม่อนุญาตให้ติดตั้งไฟล์ crx? ลองติดตั้งไฟล์ zip! คลิกที่นี่เพื่อดูบทเรียนการติดตั้ง zip
