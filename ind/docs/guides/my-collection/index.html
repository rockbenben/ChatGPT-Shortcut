Koleksi Saya

Setelah login, Koleksi Saya memberikan ruang seperti bookmark untuk prompt favorit Anda — seret untuk mengubah urutan, beri tag sebagai konteks, dan filter berdasarkan tag saat Anda perlu menemukan sesuatu dengan cepat. Prompt yang Anda buat juga akan muncul di sini.

Online vs. Offline

Pada edisi online, data koleksi tersimpan di server (butuh login). Pada edisi offline, data koleksi tersimpan secara lokal di browser (localStorage, tanpa login). Fungsinya sama persis.

Beralih Tampilan​

Tombol beralih tampilan di bagian atas halaman utama menyediakan dua mode:

  • Koleksi Saya: Menampilkan koleksi dan prompt kustom Anda
  • Jelajahi: Menelusuri perpustakaan prompt pilihan publik

Browser akan mengingat tampilan terakhir yang Anda gunakan, sehingga Anda dapat langsung melanjutkan dari titik terakhir.

Ringkasan Koleksi​

Kartu ringkasan di bagian atas tampilan Koleksi Saya menampilkan gambaran sekilas:

  • Empat statistik — Total / Prompt Saya / Favorit / Tag Kustom
  • Lencana level (L0–L9) yang naik seiring semakin banyak prompt yang Anda bagikan ke komunitas

Tambah ke Koleksi​

Klik tombol hati "Kumpulkan" di kartu untuk menambahkannya ke Koleksi Saya.

Menyimpan prompt komunitas secara otomatis memberikan satu upvote, membantu konten berkualitas mendapatkan lebih banyak visibilitas.

Hapus dari Koleksi​

Di tampilan Koleksi Saya, klik ikon hati pada kartu untuk menghapusnya.

Filter dan Pencarian​

Bilah alat di bagian atas tampilan koleksi membantu Anda menemukan sesuatu dengan cepat:

  • Tombol segmen Semua / Prompt / Favorit untuk filter berdasarkan jenis
  • Chip tag kustom — klik untuk filter berdasarkan tag
  • Kotak pencarian (Ctrl + K untuk fokus) untuk pencarian kata kunci

Tag Kustom​

Tambahkan tag pada prompt yang dikumpulkan untuk memudahkan klasifikasi:

Tambah Tag​

  1. Klik "Kelola Tag" di bilah alat
  2. Masukkan nama tag dan pilih salah satu dari 11 warna preset
  3. Simpan — tag kini tersedia di bilah filter

Kelola Tag​

Di dialog "Kelola Tag" Anda dapat:

  • Mengubah nama tag
  • Mengubah warna tag (11 warna preset)
  • Menghapus tag

Pengurutan Seret-Lepas​

Pengurutan seret-lepas didukung di tampilan Koleksi Saya:

  1. Pegang gagang seret di tepi kiri kartu
  2. Lepaskan di posisi target
  3. Urutan baru tersimpan otomatis (ke server pada edisi online, ke browser pada edisi offline)
