Koleksi Saya

Setelah login, Anda dapat mengumpulkan prompt favorit dan mengelolanya secara personal. Tampilan Koleksi Saya menyediakan fungsi manajemen yang kaya untuk membantu Anda mengatur dan menggunakan prompt secara efisien.

Beralih Tampilan​

Bagian atas halaman utama menyediakan tombol beralih tampilan, mendukung dua mode:

  • Koleksi Saya: Menampilkan prompt yang dikumpulkan dan prompt kustom Anda
  • Perpustakaan Prompt: Jelajahi perpustakaan prompt publik untuk menemukan konten baru

Browser akan secara otomatis mengingat pilihan tampilan Anda dan beralih ke tampilan terakhir pada kunjungan berikutnya.

Tambah ke Koleksi​

Klik tombol "Kumpulkan" di kartu prompt untuk menambahkannya ke Koleksi Saya.

Saat mengumpulkan prompt komunitas, sistem akan secara otomatis memberikan upvote, membantu konten berkualitas mendapatkan lebih banyak eksposur.

Hapus dari Koleksi​

Di tampilan Koleksi Saya, klik ikon hati di kartu untuk batal mengumpulkan.

Tag Kustom​

Tambahkan tag personal ke prompt yang dikumpulkan untuk manajemen klasifikasi yang nyaman:

Tambah Tag​

  1. Klik ikon tag di bilah kontrol
  2. Pilih tag yang ada atau klik "+" untuk membuat tag baru
  3. Tag baru dapat memiliki nama dan warna kustom

Kelola Tag​

Klik "Kelola Tag" untuk masuk ke antarmuka manajemen tag:

  • Ubah nama tag
  • Ubah warna tag (11 warna preset tersedia)
  • Hapus tag yang tidak diinginkan

Filter berdasarkan Tag​

Di bilah filter tampilan Koleksi Saya, Anda dapat memfilter konten berdasarkan tag kustom.

Pengurutan Seret-Lepas​

Penyesuaian urutan kartu dengan seret-lepas didukung di tampilan Koleksi Saya:

  1. Seret pegangan di sisi kiri kartu
  2. Pindahkan ke posisi target dan lepaskan
  3. Urutan baru secara otomatis disimpan ke server

Fungsi pengurutan hanya tersedia di tampilan Koleksi Saya.

Dokumentasi Terkait​

  • Prompt Kustom - Buat dan kelola prompt
  • Panduan Antarmuka - Pencarian dan filter
  • Manajemen Akun - Login dan pengaturan
