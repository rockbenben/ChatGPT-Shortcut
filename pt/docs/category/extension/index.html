Extensão do Navegador

📄️Instalar Extensão do Navegador

Use os prompts do AiShort diretamente na barra lateral das páginas de chat de IA como ChatGPT, Gemini, Claude e Doubao, sem precisar trocar de aba para copiar e colar. Instalação com um clique no Chrome, Edge e Firefox.

📄️Como usar a extensão

Domine a extensão de navegador AI Short! Ative a barra lateral e use o atalho Alt+Shift+S para abrir a biblioteca de prompts em um clique.

📄️Guia de Instalação CRX para Chrome

Instale manualmente a extensão AI Short via arquivo CRX, com instalação por arrastar após ativar o modo desenvolvedor. Inclui solução de problemas comuns.

📄️Guia de Instalação ZIP para Chrome

Instale a extensão AI Short no Chrome via arquivo ZIP, basta descompactar e arrastar para a página de extensões. Útil quando não é possível instalar o CRX.

📄️Configuração da Extensão Firefox

Guia de configuração da extensão Firefox: fixe a extensão na barra de ferramentas e autorize a execução em sites como ChatGPT, Gemini para garantir a exibição correta da barra lateral.
