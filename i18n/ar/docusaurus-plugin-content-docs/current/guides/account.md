# إدارة الحساب

يعمل AiShort بدون حساب — لكن **تسجيل الدخول** يُتيح لك تجميع مطالباتك المفضلة، وإنشاء مطالباتك الخاصة، ومشاركتها في المجتمع، ومزامنتها عبر الأجهزة. تتناول هذه الصفحة تسجيل الدخول والتسجيل وكلمات المرور وإدارة البيانات.

## طرق تسجيل الدخول

يوفر مربع حوار تسجيل الدخول تبويبَي "**تسجيل الدخول بكلمة المرور**" و"**تسجيل الدخول بدون كلمة مرور**"، إضافةً إلى تسجيل الدخول بنقرة واحدة عبر Google.

### تسجيل الدخول بكلمة المرور

سجّل الدخول باستخدام اسم المستخدم أو بريدك الإلكتروني المسجّل مع كلمة المرور.

### تسجيل الدخول بدون كلمة مرور

1. انتقل إلى تبويب "تسجيل الدخول بدون كلمة مرور" وأدخل بريدك الإلكتروني المسجّل
2. انقر "**الحصول على رابط تسجيل الدخول**" — يُرسَل رابط الدخول إلى صندوق الوارد
3. افتح الرابط لتسجيل الدخول

لا شيء تحفظه — خيار عملي إذا كنت تسجّل الدخول من حين لآخر فقط.

### تسجيل الدخول بـ Google

انقر زر Google في مربع حوار تسجيل الدخول للدخول بنقرة واحدة. في المرة الأولى، يُنشأ لك حساب تلقائيًا — دون خطوة تسجيل منفصلة.

![](/img/docs/account-login.png)

## تسجيل حساب

1. في مربع حوار تسجيل الدخول، انقر "**سجّل الآن**" للانتقال إلى التسجيل
2. أدخل البريد الإلكتروني واسم المستخدم وكلمة المرور، ووافق على سياسة الخصوصية وشروط الخدمة
3. يتم تسجيل دخولك تلقائيًا بعد الإرسال

![](/img/docs/account-register.png)

## حسابي

بعد تسجيل الدخول، افتح صفحة "**حسابي**" لإدارة ملفك الشخصي وكلمة المرور ومستواك وبياناتك.

![صفحة حسابي](/img/docs/account-profile.png)

### تغيير اسم المستخدم

انقر أيقونة التحرير بجوار اسم المستخدم، ثم احفظ.

### تغيير كلمة المرور

1. أدخل كلمة المرور الحالية
2. أدخل كلمة المرور الجديدة وأكِّدها
3. انقر "تغيير كلمة المرور" للإرسال

نسيت كلمة المرور؟ في مربع حوار تسجيل الدخول، انقر "نسيت كلمة المرور؟"، أدخل بريدك الإلكتروني المسجّل، انقر "إرسال بريد إعادة التعيين"، واتبع الرابط.

### مستواي

تعرض صفحة الحساب **مستواك (L0–L9)** والتقدم نحو المستوى التالي — يتحدد مستواك بعدد **الموجهات التي شاركتها في المجتمع**.

## إدارة البيانات

يوفر قسم "حسابي ← إدارة البيانات" ثلاثة إجراءات:

- **مسح ذاكرة التخزين المؤقت** — يُحدِّث البيانات المخزنة محليًا؛ استخدمه حين لا تتزامن البيانات أو يبدو العرض غير طبيعي. لن تفقد مجموعاتك ومطالباتك المخصصة، بل يُجبر النظام فقط على إعادة المزامنة
- **تصدير التوجيهات** — يُصدِّر ملف JSON للنسخ الاحتياطي (انظر [مطالبات مخصصة](./user-prompts#تصدير-التوجيهات))
- **استيراد التوجيهات** — يستورد المطالبات والمجموعات من ملف JSON
