# مطالبات مخصصة

بعد تسجيل الدخول، يمكنك إنشاء مطالباتك الخاصة وحفظها في حسابك للوصول إليها بسرعة لاحقًا. تظهر المطالبات المخصصة في عرض "مجموعتي".

## إنشاء مطالبة

1. بعد تسجيل الدخول، انقر زر "**إنشاء موجه**" في أعلى الصفحة
2. املأ حقول مربع حوار "إنشاء موجه":
   - **اسم التوجيه** (مطلوب): اسم للمطالبة
   - **محتوى التوجيه** (مطلوب): نص المطالبة الأساسي؛ يُبرَز النص داخل الأقواس المعقوفة `[...]` كعنصر نائب عند العرض
   - **الاستخدام** (اختياري): وصف موجز لما تفعله المطالبة
   - **ملاحظات** (اختياري): المصدر، أو نسخ بلغات أخرى، أو ملاحظات تكميلية
3. مفتاح "**هل ترغب في مشاركة هذا التوجيه على الصفحة العامة؟**" في الأسفل مفعَّل بشكل افتراضي — أوقفه لإبقاء المطالبة خاصة
4. انقر "إنشاء موجه" للإرسال

![نافذة إنشاء المطالبة](/img/docs/user-prompts-create.png)

> 💡 **مثال** — مطالبة "مولِّد تحديث الاجتماع اليومي":
> - **اسم التوجيه**: مولِّد تحديث الاجتماع اليومي
> - **محتوى التوجيه**: أنت مساعد تواصل موجز. حوِّل الملاحظات التالية إلى تحديث اجتماع يومي بثلاثة أقسام — الأمس، اليوم، العوائق. اجعل كل نقطة مختصرة ومركّزة على النتائج: [ملاحظاتي الأولية]
> - **الاستخدام**: يحوّل الملاحظات المبعثرة إلى تحديث اجتماع يومي منظّم
> - **ملاحظات**: شغِّله قبل اجتماع الصباح اليومي

## تحرير مطالبة

في عرض مجموعتي، انقر زر التعديل (القلم) على بطاقة مطالبة أنشأتها لفتح مربع حوار "تعديل التوجيه". يمكنك:

- تعديل الاسم والمحتوى والاستخدام والملاحظات
- تبديل حالة المشاركة (عام / خاص)
- النقر "حفظ" للتحديث

![واجهة تحرير المطالبة](/img/docs/user-prompts-edit.png)

## حذف مطالبة

انقر "حذف" في مربع حوار التحرير. الحذف نهائي ولا يمكن التراجع عنه — تأنَّ قبل المتابعة.

## المشاركة مع المجتمع

عند الإنشاء أو التحرير، يتحكم مفتاح "**هل ترغب في مشاركة هذا التوجيه على الصفحة العامة؟**" في الأسفل بمدى الظهور:

- **مفعَّل (افتراضي)**: تظهر المطالبة في صفحة [مطالبات المجتمع](./community)، حيث يمكن للمستخدمين الآخرين رؤيتها وتجميعها
- **متوقف**: خاصة — مرئية لك وحدك

يمكن تبديل حالة المشاركة في أي وقت.

## تصدير التوجيهات

1. انتقل إلى "حسابي" وابحث عن قسم "إدارة البيانات ← **تصدير التوجيهات**"
2. انقر زر "تصدير البيانات"
3. يُنشئ النظام ملف JSON ويُنزِّله تلقائيًا

يتضمن المحتوى المُصدَّر:

- اسم المطالبة ومحتواها واستخدامها وملاحظاتها
- حالة المشاركة
- مجموعاتك وترتيب المفضلة والعلامات المخصصة

يُوصى بالتصدير الدوري لتفادي فقدان البيانات.

## استيراد التوجيهات

استورد المطالبات والمجموعات من ملف JSON:

1. انتقل إلى "حسابي" وابحث عن قسم "إدارة البيانات ← **استيراد التوجيهات**"
2. انقر زر "استيراد البيانات"
3. اختر ملف JSON
4. يدمج النظام البيانات تلقائيًا (إزالة التكرار حسب العنوان؛ المطالبات التي تنتمي معرّفاتها لحساب آخر تُضبط تلقائيًا كخاصة)

### التعاون الجماعي

سير العمل الموصى به لمشاركة المطالبات داخل الفريق:

1. **التصفية والمشاركة**: بعد التصدير، احذف يدويًا قائمة المجموعات أو صفِّ المطالبات المراد مشاركتها، ثم أرسل الملف لأعضاء الفريق للاستيراد
2. **حماية الخصوصية**: المطالبات المستوردة المملوكة لآخرين (معرّفاتها لا تنتمي للحساب الحالي) تُضبط تلقائيًا كـ**خاصة**، مما يضمن استقلالية بيانات كل عضو
