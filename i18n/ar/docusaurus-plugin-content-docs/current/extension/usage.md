# كيفية استخدام إضافة المتصفح؟

يدعم ChatGPT Shortcut 13 لغة رئيسية، وسيتم تعيين لغة الإضافة تلقائياً وفقاً لبيئة متصفحك. يمكنك أيضاً تغيير اللغة يدوياً في صفحة الإعدادات.

![](/img/docs/extension-settings.png)

## الشريط الجانبي AiShort

تعمل إضافة AiShort حالياً كلوحة جانبية، مما يوفر تجربة تفاعلية سلسة.

إذا تم تفعيل "إبقاء اللوحة الجانبية مفتوحة" في الإعدادات، سيبقى الشريط الجانبي مفتوحاً عند التبديل بين علامات التبويب.

![](/img/docs/extension-sidebar.png)

## الوضع الداكن

تدعم الإضافة الوضع الداكن. فعّل "الوضع الداكن" في صفحة الإعدادات لتبديل الشريط الجانبي وصفحة الإعدادات إلى مظهر داكن.

## التفعيل بالاختصار

استخدم اختصار `Alt+Shift+S` لتفعيل الشريط الجانبي AiShort مباشرة.
