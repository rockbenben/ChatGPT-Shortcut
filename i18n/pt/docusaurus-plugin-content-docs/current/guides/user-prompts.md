# Prompts Personalizados

Após o login, você pode criar seus próprios prompts para acesso rápido no futuro. Os prompts personalizados serão salvos em sua conta e exibidos na visualização Minha Coleção.

## Criar Prompt

1. Clique no botão "Adicionar Prompt" no canto superior direito da página
2. Preencha as informações do prompt:
   - **Título**: Descrição curta do uso do prompt
   - **Descrição**: O conteúdo completo do prompt
   - **Observação**: Instruções de uso ou notas (opcional)
3. Escolha se deseja compartilhar na comunidade
4. Clique em Enviar

![Janela de Adicionar Prompt](/img/docs/user-prompts-create.png)

## Editar Prompt

Na visualização Minha Coleção, clique no cartão do prompt que você criou para:

- Modificar título, descrição e observação
- Alternar status de compartilhamento (Público/Privado)
- Salvar alterações

![Interface de Edição de Prompt](/img/docs/user-prompts-edit.png)

## Excluir Prompt

Clique no botão "Excluir" na interface de edição para excluir o prompt. A exclusão é irreversível, opere com cautela.

## Compartilhar na Comunidade

Ao criar ou editar um prompt, marque a opção "Compartilhar na Comunidade":

- **Público**: O prompt aparecerá na página da comunidade, outros usuários poderão vê-lo e favoritá-lo
- **Privado**: Visível apenas para você, não aparecerá na página da comunidade

Você pode alterar o status de compartilhamento a qualquer momento.

## Exportar Backup

Exporte todos os prompts que você criou:

1. Vá para o Centro Pessoal
2. Clique no botão "Exportar Prompts"
3. O sistema gera um arquivo JSON e baixa automaticamente

O conteúdo exportado inclui:

- Título do prompt, descrição, observação
- Horário de criação e atualização
- Status de compartilhamento

Recomenda-se exportar backups regularmente para evitar perda de dados.
