# Como usar a extensão do navegador?

A ChatGPT Shortcut suporta 18 idiomas principais. O idioma da extensão é definido automaticamente conforme o ambiente do seu navegador. Você também pode alterá-lo manualmente na página de configurações.

![](/img/docs/extension-settings.png)

## Modo de Exibição

A extensão suporta três modos de exibição. Escolha o que melhor se adapta ao seu uso:

- **Modo barra lateral**: abre na barra lateral do navegador — exibido lado a lado com a página, ideal para navegar pelos prompts enquanto consulta a página de diálogo da IA
- **Modo popup**: aparece como uma pequena janela ao clicar no ícone da extensão — para consulta rápida sem ocupar a tela
- **Modo janela independente**: abre em uma janela flutuante independente — pode ser arrastada para um monitor secundário ou posicionada de forma autônoma

## Manter Barra Lateral Aberta

Quando ativada, a barra lateral permanece aberta ao alternar entre abas do navegador, facilitando o uso contínuo.

## Modo Escuro

A extensão suporta modo escuro. Ative a opção "Modo escuro" nas configurações para mudar a interface para a aparência escura.

![](/img/docs/extension-sidebar.webp)

## Atalho de Teclado

Use o atalho `Alt+Shift+S` para ativar a extensão diretamente.
