# Perguntas Frequentes da Extensão

## Restrições da Página ChatGPT

O ChatGPT implementou várias restrições de acesso para páginas incorporadas por extensões, portanto, o login via autorização do Google não é suportado. Isso significa que os usuários devem fazer login usando um nome de usuário e senha.

Se sua conta foi criada inicialmente via autorização do Google, você pode usar o recurso "Esqueceu a senha" para definir uma nova senha. Isso garantirá que você possa fazer login sem problemas na página incorporada do ChatGPT.

## Aviso de Bloqueio de Conteúdo

Após fazer login com conta e senha, a página pode ocasionalmente exibir um aviso "Este conteúdo está bloqueado". Ao encontrar isso, geralmente uma simples **atualização da página** resolverá o problema, e a página retornará ao estado normal de login. Além disso, uma vez logado com sucesso, a menos que você saia ativamente, o status de login geralmente persistirá por um longo tempo, então não há necessidade de se preocupar muito com esses avisos frequentes.

Observe que alterar idiomas dentro da página integrada do ChatGPT também pode acionar o aviso "Este conteúdo está bloqueado". Portanto, é recomendável alterar os idiomas através das configurações da extensão, em vez de operar diretamente dentro da página integrada. Isso pode evitar erros desnecessários.
