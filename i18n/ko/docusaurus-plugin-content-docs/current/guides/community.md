# 커뮤니티 프롬프트

커뮤니티 프롬프트 섹션에는 사용자가 공유한 커스텀 프롬프트가 표시됩니다. 여기에서 영감을 얻거나 자신의 아이디어를 공유할 수 있습니다.

![](/img/docs/community-page.png)

## 정렬 및 필터링

커뮤니티 페이지는 다양한 필터링 방식을 제공합니다.

- **게시 시간**: 최신 공유 프롬프트 보기
- **투표 정렬**: 가장 많은 찬성표를 받은 프롬프트 보기
- **키워드 검색**: 특정 주제의 프롬프트 검색

## 투표 메커니즘

찬성/반대 투표를 통해 프롬프트의 노출도를 결정합니다.

- **찬성**: 양질의 콘텐츠가 더 많이 노출되도록 돕습니다
- **반대**: 반대 수가 일정 수량에 도달하면 프롬프트가 숨겨집니다

**컬렉션 연동**: 커뮤니티 프롬프트를 수집할 때 시스템이 자동으로 찬성표를 던집니다.

![투표 버튼 상태](/img/docs/community-vote.png)

> 투표는 로그인한 사용자만 가능하며, 동일한 프롬프트에 대한 각 사람의 투표는 가장 최근의 것을 기준으로 합니다.

## 비공개 프롬프트

공개하고 싶지 않은 아이디어는 비공개로 설정할 수 있습니다.

- 프롬프트 생성 또는 편집 시 "커뮤니티에 공유" 옵션 체크 해제
- 비공개 프롬프트는 자신만 볼 수 있습니다
- 공개/비공개 상태는 언제든지 전환 가능합니다
