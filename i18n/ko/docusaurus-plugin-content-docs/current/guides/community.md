# 커뮤니티 프롬프트

커뮤니티 프롬프트는 다른 사용자들이 직접 올린 결과물입니다. 여기서 영감을 얻고, 내가 다듬은 아이디어를 같이 나눠 볼 수 있습니다. **투표 기능**으로 좋은 프롬프트는 더 잘 보이게, 부족한 글은 자연스럽게 뒤로 밀리도록 정리됩니다.

![](/img/docs/community-page.png)

## 정렬과 필터

- **게시 시간**: 가장 최근에 올라온 프롬프트부터 살펴봅니다
- **투표 정렬**: 찬성표가 많은 인기 프롬프트를 모아 봅니다
- **키워드 검색**: 원하는 주제를 직접 입력해 찾습니다

## 투표 방식

찬성과 반대 투표가 프롬프트의 노출 정도를 정합니다.

- **찬성**: 좋은 글이 더 많은 사람에게 닿도록 밀어 줍니다
- **반대**: 반대표가 일정 수를 넘기면 해당 프롬프트는 목록에서 가려집니다

**즐겨찾기와 연동**: 커뮤니티 프롬프트를 즐겨찾기에 담으면 자동으로 찬성표가 함께 등록됩니다.

![투표 버튼 상태](/img/docs/community-vote.png)

> 투표는 로그인한 분만 할 수 있고, 같은 프롬프트에 여러 번 표를 던지면 가장 최근에 누른 결과만 반영됩니다.

## 비공개 프롬프트

아직 다듬는 중이거나 혼자만 쓰고 싶은 아이디어는 비공개로 두면 됩니다.

- 프롬프트를 만들거나 편집할 때 "커뮤니티에 공유"를 끕니다
- 비공개로 둔 프롬프트는 본인만 확인할 수 있습니다
- 공개와 비공개는 언제든지 다시 바꿀 수 있습니다
