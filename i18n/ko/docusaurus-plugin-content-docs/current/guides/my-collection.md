# 마이 컬렉션

로그인 후 좋아하는 프롬프트를 수집하고 개인적으로 관리할 수 있습니다. 마이 컬렉션 뷰는 프롬프트를 효율적으로 정리하고 사용할 수 있는 풍부한 관리 기능을 제공합니다.

## 뷰 전환

홈페이지 상단에는 뷰 전환 버튼이 있어 두 가지 모드를 지원합니다.

- **마이 컬렉션**: 수집한 프롬프트와 커스텀 프롬프트 표시
- **프롬프트 라이브러리**: 공용 프롬프트 라이브러리를 탐색하여 새로운 콘텐츠 발견

브라우저는 뷰 선택을 자동으로 기억하고 다음 방문 시 마지막 뷰로 자동 전환합니다.

![마이 컬렉션 뷰](/img/docs/my-collection-view.gif)

## 컬렉션에 추가

프롬프트 카드의 "즐겨찾기" 버튼을 클릭하여 마이 컬렉션에 추가합니다.

커뮤니티 프롬프트를 수집하면 시스템이 자동으로 찬성표를 던져 양질의 콘텐츠가 더 많이 노출되도록 돕습니다.

## 컬렉션에서 제거

마이 컬렉션 뷰에서 카드의 하트 아이콘을 클릭하여 즐겨찾기를 해제합니다.

## 커스텀 태그

수집한 프롬프트에 개인 태그를 추가하여 편리하게 분류 관리하세요.

### 태그 추가

1. 컨트롤 바의 태그 아이콘 클릭
2. 기존 태그 선택 또는 "+"를 클릭하여 새 태그 생성
3. 새 태그는 커스텀 이름과 색상을 가질 수 있습니다

### 태그 관리

"태그 관리"를 클릭하여 태그 관리 인터페이스로 진입합니다.

- 태그 이름 수정
- 태그 색상 변경 (11가지 사전 설정 색상 사용 가능)
- 불필요한 태그 삭제

![태그 관리 팝업](/img/docs/my-collection-tag.png)

### 태그로 필터링

마이 컬렉션 뷰의 필터링 바에서 커스텀 태그별로 콘텐츠를 필터링할 수 있습니다.

## 드래그 앤 드롭 정렬

마이 컬렉션 뷰에서는 드래그 앤 드롭으로 카드 순서를 조정할 수 있습니다.

1. 카드 왼쪽의 드래그 핸들을 드래그합니다
2. 목표 위치로 이동하여 놓습니다
3. 새 순서는 자동으로 서버에 저장됩니다

정렬 기능은 마이 컬렉션 뷰에서만 사용할 수 있습니다.

![드래그 앤 드롭 정렬](/img/docs/my-collection-drag.gif)
