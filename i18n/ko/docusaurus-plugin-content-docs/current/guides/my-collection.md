# 마이 컬렉션

로그인하면 **마이 컬렉션** 뷰에서 마음에 드는 프롬프트를 **북마크 정리하듯** 차곡차곡 모을 수 있습니다. 드래그로 순서를 바꾸고, 사용자 지정 태그를 붙이고, 태그별로 걸러 보며 필요할 때 빠르게 꺼내 쓰세요.

:::tip 온라인 버전과 오프라인 버전
온라인 버전의 즐겨찾기 데이터는 서버에 저장됩니다(로그인 필요). [오프라인 배포 버전](./offline)은 같은 데이터를 브라우저 안(localStorage)에 보관해 로그인 없이도 쓸 수 있습니다. 기능은 양쪽 모두 같습니다.
:::

## 뷰 전환

홈 화면 위쪽의 뷰 전환 버튼으로 두 가지 모드를 오갈 수 있습니다.

- **마이 컬렉션**: 즐겨찾기한 프롬프트와 직접 만든 프롬프트를 모아 봅니다
- **프롬프트 라이브러리**: 공용 프롬프트 라이브러리를 둘러봅니다

브라우저가 마지막에 본 뷰를 기억하므로, 다시 들어오면 그 화면이 자동으로 열립니다.

![마이 컬렉션 뷰](/img/docs/my-collection-view.gif)

## 즐겨찾기에 담기

프롬프트 카드의 "즐겨찾기" 버튼을 누르면 마이 컬렉션에 바로 담깁니다.

커뮤니티 프롬프트를 담으면 찬성표가 함께 등록되어, 좋은 글이 더 많은 사람에게 닿도록 도와줍니다.

## 즐겨찾기에서 빼기

마이 컬렉션 뷰에서 카드의 하트 아이콘을 한 번 더 누르면 즐겨찾기에서 빠집니다.

## 사용자 지정 태그

즐겨찾기한 프롬프트에 태그를 붙여 주제별로 깔끔하게 정리해 보세요.

### 태그 추가

1. 컨트롤 바의 태그 아이콘을 누릅니다
2. 이미 만들어 둔 태그를 고르거나, "+"를 눌러 새로 만듭니다
3. 새 태그에는 원하는 이름과 색상을 지정할 수 있습니다

### 태그 관리

"태그 관리"를 누르면 관리 화면이 열립니다.

- 태그 이름 바꾸기
- 태그 색상 바꾸기 (사전에 마련된 11가지 색상)
- 태그 삭제

![태그 관리 팝업](/img/docs/my-collection-tag.png)

### 태그로 거르기

마이 컬렉션 뷰의 필터 바에서 사용자 지정 태그별로 골라 볼 수 있습니다.

## 드래그로 순서 바꾸기

마이 컬렉션 뷰에서는 카드를 끌어다 놓아 순서를 마음대로 조정할 수 있습니다.

1. 카드 왼쪽의 드래그 핸들을 잡습니다
2. 원하는 자리로 옮긴 뒤 놓습니다
3. 바뀐 순서는 곧바로 저장됩니다(온라인 버전은 서버에, 오프라인 버전은 브라우저 안에)

순서 조정은 마이 컬렉션 뷰에서만 쓸 수 있습니다.

![드래그 앤 드롭 정렬](/img/docs/my-collection-drag.gif)
