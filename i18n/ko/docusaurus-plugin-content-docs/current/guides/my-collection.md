# 마이 컬렉션

로그인하면 **마이 컬렉션**에서 마음에 드는 프롬프트를 북마크처럼 모아둘 수 있습니다 — 드래그로 순서를 바꾸고, 사용자 지정 태그를 붙이고, 태그별로 걸러 보며 필요할 때 빠르게 꺼내 쓰세요. 직접 만든 프롬프트도 여기에 함께 표시됩니다.

:::tip[온라인 버전과 오프라인 버전]

온라인 버전의 즐겨찾기 데이터는 서버에 저장됩니다(로그인 필요). [오프라인 배포 버전](../deploy/offline)은 같은 데이터를 브라우저 안(localStorage)에 보관해 로그인 없이도 쓸 수 있습니다. 기능은 양쪽 모두 같습니다.

:::

## 뷰 전환

홈 화면 위쪽의 뷰 전환 버튼으로 두 가지 모드를 오갈 수 있습니다:

- **내 컬렉션**: 즐겨찾기한 프롬프트와 직접 만든 프롬프트를 모아 봅니다
- **탐색**: 공용 엄선 라이브러리를 둘러봅니다

브라우저가 마지막에 본 뷰를 기억하므로, 다시 들어오면 그 화면이 자동으로 열립니다.

![마이 컬렉션 뷰](/img/docs/my-collection-view.webp)

## 컬렉션 개요

마이 컬렉션 뷰 상단의 요약 카드에서 핵심 수치를 한눈에 확인할 수 있습니다:

- 네 가지 통계 — **총계** / **내 프롬프트** / **즐겨찾기** / **사용자 지정 태그**
- **레벨 뱃지** (L0–L9) — **커뮤니티에 프롬프트를 더 많이 공유할수록** 레벨이 올라갑니다

## 즐겨찾기에 담기

카드의 하트 "즐겨찾기" 버튼을 클릭하면 마이 컬렉션에 바로 담깁니다.

커뮤니티 프롬프트를 담으면 찬성표가 함께 등록되어, 좋은 글이 더 많은 사람에게 닿도록 도와줍니다.

## 즐겨찾기에서 빼기

마이 컬렉션 뷰에서 카드의 하트 아이콘을 한 번 더 클릭하면 즐겨찾기에서 빠집니다.

## 필터와 검색

컬렉션 뷰 상단의 툴바에서 빠르게 원하는 항목을 찾을 수 있습니다:

- **전체 / 프롬프트 / 즐겨찾기** 세그먼트 스위치로 유형별로 걸러 볼 수 있습니다
- 사용자 지정 태그 칩을 클릭해 태그별로 필터링할 수 있습니다
- 검색창(`Ctrl + K`으로 바로 활성화)에서 키워드로 검색할 수 있습니다

## 사용자 지정 태그

즐겨찾기한 프롬프트에 태그를 붙여 주제별로 깔끔하게 정리해 보세요.

### 태그 추가

1. 툴바의 "**태그 관리**"를 클릭합니다
2. 태그 이름을 입력하고 **11가지 사전 설정 색상** 중 하나를 고릅니다
3. 저장하면 필터 바에서 바로 사용할 수 있습니다

### 태그 관리

"태그 관리" 대화상자에서 다음을 수행할 수 있습니다:

- 태그 이름 바꾸기
- 태그 색상 바꾸기 (11가지 사전 설정 색상)
- 태그 삭제

![태그 관리 팝업](/img/docs/my-collection-tag.png)

## 드래그로 순서 바꾸기

마이 컬렉션 뷰에서는 드래그 앤 드롭으로 카드 순서를 자유롭게 조정할 수 있습니다:

1. 카드 왼쪽 가장자리의 드래그 핸들을 잡습니다
2. 원하는 위치로 옮긴 뒤 놓습니다
3. 바뀐 순서는 자동으로 저장됩니다(온라인 버전은 서버에, 오프라인 버전은 브라우저에)

![드래그 앤 드롭 정렬](/img/docs/my-collection-drag.webp)
