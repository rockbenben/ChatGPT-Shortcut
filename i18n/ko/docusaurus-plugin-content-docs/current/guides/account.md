# 계정 관리

AiShort는 로그인 없이도 자유롭게 둘러볼 수 있지만, **로그인하면** 마음에 드는 프롬프트를 즐겨찾기에 모으고, 나만의 프롬프트를 만들어 커뮤니티에 공유하며, 여러 기기에서 동기화할 수 있습니다. 이 페이지에서는 로그인과 등록, 비밀번호 변경, 캐시 정리까지 계정과 관련된 작업을 한 번에 정리했습니다.

## 로그인 방식

### 계정 비밀번호 로그인

사용자 이름 또는 등록된 이메일과 비밀번호를 사용하여 로그인합니다.

비밀번호를 잊은 경우: 로그인 페이지의 "비밀번호 찾기"를 클릭하고 등록된 이메일을 입력하면 재설정 링크가 발송됩니다.

### 구글 로그인

로그인 페이지의 구글 아이콘을 누르기만 하면 바로 로그인됩니다. 처음 사용한다면 별도 등록 절차 없이 계정이 자동으로 만들어집니다.

![](/img/docs/account-login.png)

### 비밀번호 없는 로그인

1. 로그인 페이지에 가입한 이메일 주소를 입력합니다
2. "로그인 링크 받기"를 누릅니다
3. 메일함을 열어 받은 링크를 클릭하면 바로 로그인됩니다

비밀번호를 따로 외울 필요가 없어 가끔씩만 들르는 분들께 잘 맞습니다.

## 계정 등록

1. 로그인 페이지의 "등록" 옵션 클릭
2. 사용자 이름, 이메일 및 비밀번호 입력
3. 제출 후 자동 로그인

![](/img/docs/account-register.png)

## 비밀번호 변경

로그인 후 "내 계정"에서 비밀번호를 변경할 수 있습니다.

![내 계정 페이지](/img/docs/account-profile.png)

1. 현재 비밀번호 입력
2. 새 비밀번호 입력 및 확인
3. "비밀번호 변경"을 클릭하여 제출

## 캐시 지우기

데이터 동기화가 어긋나거나 화면 표시가 이상할 때 사용합니다.

1. "내 계정" → "데이터 관리"에서 "캐시 지우기" 항목을 엽니다
2. "캐시 지우기" 버튼을 누르고 확인하면 모든 데이터가 다시 불러와집니다

즐겨찾기와 직접 만든 프롬프트는 사라지지 않으니 안심해도 됩니다. 서버에서 한 번 더 받아올 뿐입니다.
