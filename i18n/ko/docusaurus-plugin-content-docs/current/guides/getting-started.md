# 시작하기

AiShort 를 사용하면 선택한 프롬프트를 모든 AI 대화 모델에 복사하기만 하면 지정된 방향으로 출력이 나옵니다. 구체적인 단계는 다음과 같습니다.

1. 원하는 프롬프트를 검색하거나 선택한 다음 프롬프트에서 복사 버튼을 클릭합니다.

2. 프롬프트를 ChatGPT, Gemini 또는 모든 AI 대화 모델에 복사합니다.

3. 대화 프롬프트에 따라 요구 사항을 입력합니다.
