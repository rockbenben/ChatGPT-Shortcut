# 시작하기

AI와 대화할 때 좋은 프롬프트는 양질의 답변을 더 빨리 얻는 데 도움이 됩니다. AiShort는 복사하여 바로 사용할 수 있는 실용적인 프롬프트를 다수 정리했습니다.

## 기본 사용

1. 홈페이지에서 필요한 프롬프트를 검색하거나 선택합니다
2. 카드를 클릭하여 세부 정보를 확인하거나 "복사" 버튼을 직접 클릭합니다
3. 임의의 AI 모델을 열고 프롬프트를 붙여넣습니다
4. 프롬프트의 지침에 따라 구체적인 질문을 추가합니다
