# 자주 묻는 질문

## AiShort는 유료인가요?

AiShort 웹 버전은 **완전히 무료**입니다. 프롬프트 탐색, 복사, 검색에 비용이 들지 않으며 계정 등록조차 필요하지 않습니다. 로그인 후 사용할 수 있는 고급 기능(컬렉션, 커스텀 프롬프트, 커뮤니티 공유)도 모두 무료입니다.

참고: AiShort는 프롬프트만 제공하며 **AI 모델 자체의 호출 요금은 사용하는 AI 서비스에 따라 결정됩니다** (예: ChatGPT 무료/유료 버전, API 과금 등).

## 어떤 AI 모델을 지원하나요?

AiShort의 프롬프트는 범용적이며 **프롬프트를 입력할 수 있는 모든 AI 환경에 적용됩니다** — 채팅 페이지뿐 아니라 Cursor·Copilot 같은 코딩 도구, API 호출, AI 에이전트 등에서도 그대로 사용할 수 있습니다. 아래는 대표적인 대화형 모델입니다.

- 글로벌: ChatGPT, Gemini, Claude, Grok
- 국내: HyperCLOVA X, Solar, Gauss, A.X
- 중국 본토: DeepSeek, Tongyi Qianwen, Ernie Bot, Doubao, Kimi, ChatGLM, iFLYTEK Spark, Tencent Yuanbao
- API 플랫폼: OpenRouter, SiliconFlow, Groq

전체 모델 목록은 [시작하기 → 일반적인 AI 모델](./getting-started#일반적인-ai-모델)을 참조하세요.

## 왜 영어 프롬프트를 사용하나요?

대다수 글로벌 AI 모델은 학습 데이터의 비중상 영어 이해력과 추론 안정성이 더 높습니다. 또한 국내 IT·스타트업 현장에서는 이미 영문 기술 용어(deploy, sprint, async 등)가 표준으로 쓰이고 있어 영어 프롬프트와 자연스럽게 어울립니다.

한국어 프롬프트도 물론 사용 가능하지만 동일한 한국어 문장을 반복 실행하면 결과 편차가 큰 편입니다. **권장 패턴**은 「영어로 지시 + 한국어로 답변」 조합으로 안정성과 가독성을 모두 잡는 방식입니다. 한국어 출력의 자연스러움이 중요한 경우(공식 문서·고객 응대 메일 등)에는 한국어 특화 모델(HyperCLOVA X, Solar 등)도 고려해 보세요.

> 💡 한국어 답변이 필요하다면 프롬프트 끝에 `Respond in Korean.` 또는 `한국어로 답변해 주십시오.`를 추가하세요. 존댓말 톤을 유지하려면 `격식체(합쇼체)로 답변해 주십시오.`를 함께 명시하면 좋습니다.

## 매번 프롬프트를 입력해야 하나요?

**API 사용**: 프롬프트를 `system prompt`로 설정하면 후속 대화에 자동으로 적용됩니다.

**웹 버전**: 대화를 전환하지 않는 한 ChatGPT는 현재 프롬프트를 기억합니다. 답변이 예상에서 벗어나기 시작하면 프롬프트를 "잊은" 것이므로 다시 한 번 붙여넣으면 됩니다.

**팁**: 자주 사용하는 대화를 브라우저 북마크로 저장하면 다음에 바로 열 수 있습니다.

## 관련된 프롬프트를 찾을 수 없는 이유는 무엇입니까?

홈페이지의 검색 범위는 **엄선된 프롬프트 라이브러리**(로그인 후에는 개인 프롬프트 포함)이며 커뮤니티에 공유된 프롬프트는 **포함하지 않습니다**.

짧은 키워드로 홈페이지에서 찾을 수 없는 경우 [커뮤니티 프롬프트](./community) 페이지에서 다시 검색하세요. 그곳에는 더 많은 사용자 공유 콘텐츠가 있습니다.

## AI가 거짓 정보를 출력하면 어떻게 해야 할까요?

AI는 때때로 "환각"을 일으켜 그럴듯해 보이지만 실제로는 잘못된 정보를 출력합니다. 처리 방법:

1. **핵심 정보 확인**: 특히 데이터, 인용, 코드 등
2. **다중 최적화**: AI가 답변을 다시 확인하고 최적화하도록 합니다
3. **교차 검증**: 다른 프롬프트나 모델을 사용하여 중요한 결론을 검증합니다

적절한 프롬프트는 AI 환각의 발생을 줄이는 데 도움이 됩니다.

## 프롬프트를 백업하려면 어떻게 해야 하나요?

1. "내 계정" → "데이터 관리 → 프롬프트 내보내기" 항목 찾기
2. "데이터 내보내기" 버튼 클릭
3. 시스템이 자동으로 JSON 파일을 생성하여 다운로드

데이터 손실을 방지하기 위해 정기적으로 백업하는 것이 좋습니다.

## 프롬프트 효과가 이상적이지 않나요?

1. **AI에게 최적화 요청**: AI가 자신의 답변을 평가하고 개선하도록 하세요
2. **관점 변경**: 다른 프롬프트로 동일한 요구사항을 표현해 보세요
3. **커뮤니티 활용**: [커뮤니티 프롬프트](./community)에 더 적합한 버전이 있을 수 있습니다. 좋은 프롬프트가 있다면 그곳에 공유해 주세요
4. **피드백 제안**: 문제를 발견하거나 개선 제안이 있다면 [피드백](/feedback)을 환영합니다
