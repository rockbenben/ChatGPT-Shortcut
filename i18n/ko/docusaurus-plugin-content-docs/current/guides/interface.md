# 인터페이스 가이드

AiShort 페이지는 사용자가 프롬프트를 탐색하고 필터링할 수 있는 직관적인 인터페이스를 제공합니다. 페이지는 주로 태그 영역, 검색 영역 및 프롬프트 표시 영역으로 구성됩니다.

![](/img/docs/interface-home.png)

## 🏷︎ 태그 필터링

태그 영역은 사용자가 도메인 및 기능을 기준으로 프롬프트를 필터링하도록 돕습니다.

오른쪽 상단의 "태그 필터 규칙 전환" 버튼을 통해 더 정밀한 다중 태그 필터링을 구현할 수 있습니다.

- **OR 규칙** (기본값): 선택한 태그 중 하나라도 있는 프롬프트 표시
- **AND 규칙**: 선택한 모든 태그가 동시에 있는 프롬프트만 표시

![](/img/docs/interface-filter.png)

## 🔍 키워드 검색

키워드 검색 기능은 프롬프트 제목, 설명 및 내용 검색을 지원합니다.

- 이미 태그가 선택된 경우 검색 범위는 해당 태그로 제한됩니다
- 로그인 후 검색 범위에는 컬렉션 및 커스텀 프롬프트가 포함됩니다

![](/img/docs/interface-search.png)

## 🔬 프롬프트 카드

### 세부 정보 보기

프롬프트 카드를 클릭하면 세부 정보 팝업이 열려 제목, 설명, 전체 프롬프트 내용, 비고 및 복사 버튼을 볼 수 있습니다.

![](/img/docs/prompt-card.gif)

### 프롬프트 복사

카드 하단의 "복사" 버튼을 클릭하여 프롬프트를 가져옵니다. AI 모델에 붙여넣어 사용하세요.

### 프롬프트 즐겨찾기

로그인 후 카드에 즐겨찾기 버튼이 나타납니다. 클릭하여 프롬프트를 마이 컬렉션에 추가합니다.

## 🔥 인기 정렬

페이지는 인기도에 따라 프롬프트를 정렬하여 인기 있는 콘텐츠를 빠르게 발견할 수 있도록 합니다.

태그 정렬 또한 열기 값에 따라 조정되어 커뮤니티의 최신 선호도를 반영합니다.
