# 인터페이스 가이드

AiShort 홈 화면은 네 가지 영역으로 구성됩니다 — 상단의 **제목과 진입 버튼**, 그 아래의 **태그 필터**, **검색창**, 그리고 나머지 공간을 채우는 **프롬프트 카드**. 이 가이드에서는 이 네 영역을 함께 활용해 원하는 프롬프트를 빠르게 찾는 방법을 알려드립니다.

![](/img/docs/interface-home.png)

## 상단 진입 버튼

페이지 제목 아래의 버튼은 로그인 전후로 조금 달라집니다:

- **로그아웃 상태**: "**무료 로그인**" + "**커뮤니티 둘러보기**"
- **로그인 상태**: 뷰 전환 버튼("**탐색**" ⇄ "**내 컬렉션**"), "**프롬프트 만들기**", 오른쪽의 설정 메뉴("내 계정", "로그아웃")

> 💡 로그인 없이도 엄선된 프롬프트를 모두 둘러보고 복사할 수 있습니다. 로그인하면 즐겨찾기, 직접 만들기, 기기 간 동기화 기능이 추가됩니다.

## 태그 필터링

프롬프트를 분야와 기능에 따라 골라 볼 수 있습니다. 오른쪽 위의 **OR / AND** 전환 버튼이 여러 태그를 다루는 방식을 결정합니다:

- **OR** (기본값): 고른 태그 중 하나라도 맞으면 표시합니다
- **AND**: 고른 태그 전부와 맞아야 표시합니다

태그를 고르면 "**필터 지우기**" 버튼이 나타나 한 번에 모두 해제할 수 있습니다. 모바일에서는 태그가 기본적으로 접혀 있으니 "**태그 표시**"를 눌러 펼치세요.

![](/img/docs/interface-filter.png)

## 키워드 검색

프롬프트의 제목, 설명, 본문, 메모를 모두 훑어 줍니다.

- 데스크톱에서는 **`Ctrl + K`** (macOS는 `⌘ + K`)를 눌러 검색창을 바로 활성화할 수 있고, `Esc`로 빠져나옵니다
- 홈 화면 검색은 **엄선된 프롬프트**를 대상으로 합니다. 태그를 이미 골랐다면 그 범위 안에서만 검색합니다
- 로그인하면 **즐겨찾기**와 **사용자 정의 프롬프트**까지 검색 범위가 넓어집니다
- 짧은 단어로 홈에서 결과가 없으면 AiShort가 자동으로 [커뮤니티 프롬프트](./community)까지 검색해 드립니다

![](/img/docs/interface-search.png)

## 프롬프트 카드

엄선된 프롬프트는 누적 복사 횟수 순으로 정렬되므로, 가장 많이 쓰는 항목이 먼저 표시됩니다.

### 카드 정보

카드마다 프롬프트의 제목, 설명, 본문 미리보기, 태그가 표시됩니다. 오른쪽 위의 불꽃 아이콘과 숫자는 누적 복사 횟수입니다.

### 자세히 보기

카드를 클릭하면 상세 팝업이 열려 전체 본문과 메모를 확인하고 한 번에 복사할 수 있습니다.

![](/img/docs/prompt-card.webp)

### 프롬프트 복사

카드 또는 상세 팝업의 "**복사**" 버튼을 클릭한 뒤, 어떤 AI 도구에든 붙여넣어 바로 사용할 수 있습니다(채팅 페이지, 코딩 도구, API 호출 등).

### 프롬프트 즐겨찾기

로그인하면 카드마다 하트 버튼이 나타납니다 — 눌러두면 [마이 컬렉션](./my-collection)에 바로 담깁니다.
