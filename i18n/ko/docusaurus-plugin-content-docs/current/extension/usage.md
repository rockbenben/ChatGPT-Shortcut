# 브라우저 확장 프로그램 사용법

ChatGPT Shortcut은 13개 주요 언어를 지원하며, 확장 프로그램 언어는 브라우저 환경에 따라 자동으로 설정됩니다. 설정 페이지에서 수동으로 언어를 변경할 수도 있습니다.

![](/img/docs/extension-settings.png)

## AiShort 사이드바

AiShort 확장 프로그램은 현재 사이드 패널로 작동하여 원활한 인터랙티브 경험을 제공합니다.

설정에서 "사이드 패널 열린 상태 유지"를 활성화하면 브라우저 탭을 전환해도 사이드바가 열린 상태로 유지됩니다.

![](/img/docs/extension-sidebar.png)

## 다크 모드

확장 프로그램은 다크 모드를 지원합니다. 설정 페이지에서 "다크 모드"를 활성화하면 사이드바와 설정 페이지가 어두운 테마로 전환됩니다.

## 단축키 활성화

`Alt+Shift+S` 단축키를 사용하여 AiShort 사이드바를 직접 활성화할 수 있습니다.
