# 브라우저 확장 프로그램 사용법

ChatGPT Shortcut은 13개 주요 언어를 지원합니다. 확장 프로그램 언어는 브라우저 환경에 따라 자동 설정됩니다. 설정 페이지에서 수동으로 변경할 수도 있습니다.

![](/img/docs/extension-settings.png)

## 표시 모드

확장 프로그램은 세 가지 표시 모드를 지원합니다:

- **사이드바 모드**: 브라우저 사이드바에서 열며, 웹 콘텐츠와 나란히 표시
- **팝업 모드**: 확장 프로그램 아이콘 클릭 시 팝업으로 표시
- **창 모드**: 독립적인 플로팅 창에서 열기

## 사이드 패널 열어두기

활성화하면 브라우저 탭을 전환해도 사이드바가 열린 상태로 유지됩니다.

## 다크 모드

확장 프로그램은 다크 모드를 지원합니다. 설정 페이지에서 "다크 모드"를 활성화하면 인터페이스가 다크 모드로 전환됩니다.

![](/img/docs/extension-sidebar.png)

## 단축키 활성화

`Alt+Shift+S` 단축키로 확장 프로그램을 직접 활성화할 수 있습니다.
