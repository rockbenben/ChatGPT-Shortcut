# 브라우저 확장 프로그램 사용 방법은 무엇입니까?

ChatGPT Shortcut은 13개 주류 언어를 지원하며 확장 프로그램 언어는 브라우저 환경에 따라 자동으로 설정됩니다. 확장 프로그램 내의 ChatGPT 임베디드 페이지 및 사이드바 언어도 이 설정을 따릅니다. 타사 웹사이트의 권한 경고가 트리거되는 것을 방지하기 위해 **임베디드 페이지에서 언어를 변경하지 마세요**.

![](/img/docs/extension-settings.png)

## AiShort 사이드바

AiShort 확장 프로그램은 현재 사이드 패널(Side Panel) 형태로 실행되어 원활한 상호 작용 경험을 제공합니다.

설정에서 "사이드 패널 열어두기(Keep side panel open)" 옵션을 활성화하면 브라우저 탭을 전환할 때도 사이드바가 열린 상태로 유지되어 지속적으로 사용하기 편리합니다.

![](/img/docs/extension-sidebar.png)

다른 웹사이트에서 AiShort 사이드바를 사용하려면 [ChatGPT Shortcut Anywhere](https://greasyfork.org/scripts/482907-chatgpt-shortcut-anywhere) 스크립트를 설치하세요. Tampermonkey는 사용자가 사용자 지정 스크립트를 실행하여 웹 페이지를 수정할 수 있는 브라우저 확장 프로그램입니다. 기능 중복 및 잠재적 충돌을 피하기 위해 ChatGPT Shortcut Anywhere 스크립트는 기본적으로 확장 프로그램이 지원하는 웹사이트에서는 적용되지 않습니다.

## 단축키 활성화

`Alt+Shift+S` 단축키를 사용하면 AiShort 사이드바를 직접 활성화할 수 있습니다.
