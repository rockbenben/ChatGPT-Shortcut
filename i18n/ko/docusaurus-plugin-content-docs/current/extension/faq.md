# 확장 프로그램 자주 묻는 질문

## ChatGPT 페이지 제한

ChatGPT는 확장 프로그램에 포함된 페이지에 여러 권한 제한을 설정했으므로 Google 인증을 사용한 로그인은 지원되지 않습니다. 즉, 사용자는 계정과 비밀번호를 사용하여 로그인해야 합니다.

계정이 원래 Google 인증으로 생성된 경우 "비밀번호 찾기" 기능을 사용하여 새 비밀번호를 설정할 수 있습니다. 이렇게 하면 포함된 ChatGPT 페이지 내에서 원활하게 로그인할 수 있습니다.

## 콘텐츠 차단 프롬프트

계정과 비밀번호를 사용하여 로그인한 후 페이지에 "이 콘텐츠가 차단되었습니다"라는 프롬프트가 표시될 수 있습니다. 이러한 경우 일반적으로 **페이지를 새로 고침**하면 문제가 해결되고 페이지가 정상적인 로그인 상태로 돌아갑니다. 또한 한 번 로그인하면 적극적으로 로그아웃하지 않는 한 로그인 상태가 오랫동안 유지되므로 이러한 프롬프트가 자주 나타나는 것에 대해 너무 걱정할 필요가 없습니다.

통합 ChatGPT 페이지 내에서 언어를 전환하면 "이 콘텐츠가 차단되었습니다"라는 프롬프트가 트리거될 수 있습니다. 따라서 불필요한 오류 프롬프트를 피하기 위해 통합 페이지 내에서 직접 조작하는 대신 확장 프로그램의 설정 항목을 통해 언어를 전환하는 것이 좋습니다.
