# AI Short 소개

AI Short(ChatGPT Shortcut)는 시나리오별로 분류된 AI 프롬프트 라이브러리로 복사하여 바로 사용할 수 있습니다.

> **"프롬프트"란 무엇인가요?** 프롬프트(Prompt)는 AI에 입력하는 지시문으로 AI 출력의 품질을 결정합니다. 좋은 프롬프트는 AI가 전문적이고 정확한 답변을 내놓도록 하지만 모호한 프롬프트는 평범한 답변만을 가져옵니다. AiShort는 다양한 시나리오(글쓰기, 프로그래밍, 사무, 학습 등)의 우수한 프롬프트를 정리하여 어떻게 질문할지 고민하는 시간을 줄여줍니다.

## ⚡ 30초 빠른 시작

1. AI Short 열기
2. 필요한 프롬프트 검색 또는 탐색
3. "복사"를 클릭하고 어떤 AI 도구에든 붙여넣기 — ChatGPT·Gemini·Claude 같은 채팅 페이지, Cursor·Copilot 같은 코딩 도구, API 호출, AI 에이전트 등

정말 간단합니다. 더 많은 활용법은 [시작하기 가이드](./guides/getting-started)를 확인하세요.

## 핵심 기능

🚀 **엄선된 프롬프트**: 글쓰기, 프로그래밍, 사무 등 다양한 시나리오를 다루며 복사하여 바로 사용 가능.

🔍 **태그 + 키워드 검색**: 시나리오 태그로 필터링하거나 키워드로 직접 검색.

🌍 **18개 언어 지원**: 인터페이스와 프롬프트 번역이 완비되어 있으며 모국어 답변을 지정할 수 있습니다.

📦 **즉시 사용 가능**: 계정 등록 없이 열어서 바로 사용.

## 고급 기능 (로그인 후)

📂 **마이 컬렉션**: 드래그 앤 드롭 정렬 및 커스텀 태그 분류 지원.

✏️ **커스텀 프롬프트**: 나만의 프롬프트를 생성, 편집 및 관리.

🗳️ **커뮤니티 소통**: 프롬프트를 커뮤니티에 공유하고 투표에 참여.

📤 **데이터 내보내기**: 모든 프롬프트를 JSON 파일로 원클릭 내보내기.

🔐 **다양한 로그인 방식**: 계정 비밀번호, Google, 비밀번호 없는 이메일 링크 지원.

## 오프라인 배포 버전

[오프라인 배포 버전](./guides/offline)은 사내망, 정부망 등 외부 네트워크에 접속할 수 없는 시나리오를 위해 설계되었습니다. 백엔드 서버와 계정이 필요 없으며 배포 후 즉시 사용 가능합니다. 탐색, 검색, 컬렉션, 커스텀 등 핵심 기능을 유지하며 데이터 형식은 온라인 버전과 호환됩니다.
