# Panduan Instalasi Lokal Ekstensi Chrome CRX

> Panduan ini mengasumsikan Anda sudah mengunduh dan mengekstrak paket instalasi CRX dari [halaman instalasi](./README.md) (file .crx ada di dalam folder yang diekstrak).
>
> ⚠️ **Dua kesalahan umum**:
> 1. Anda harus menyeret **file `.crx`** (yang sudah diekstrak) ke halaman ekstensi, bukan paket instalasi zip itu sendiri
> 2. Harus **diseret** — jangan klik tombol "Muat ekstensi yang belum dikemas" (itu untuk versi zip)

## Aktifkan Mode Developer

Buka halaman "Kelola Ekstensi" Chrome dan aktifkan "Mode developer".

Salin alamat berikut dan tempel ke bilah alamat browser, lalu tekan Enter untuk membuka. Aktifkan "Mode developer" di kanan atas halaman.

```txt
chrome://extensions
```

![](https://img.newzone.top/2024-08-12-22-05-52.png?imageMogr2/format/webp)

## Instal Ekstensi

Instal ekstensi ChatGPT Shortcut (Catatan ⚠️: Anda harus menyeret file .crx, JANGAN klik "Muat ekstensi yang belum dikemas")

![](https://img.newzone.top/2024-08-12-22-16-38.png?imageMogr2/format/webp)

Setelah instalasi berhasil, periksa [Tutorial Penggunaan Plugin](./usage).

## Masalah Instalasi?

1. Pengguna Windows: Periksa apakah Anda sudah mengekstrak paket instalasi yang diunduh (bukan mengklik dua kali untuk membuka)?

2. Apakah "Mode developer" sudah diaktifkan? Jika belum, silakan lihat langkah kedua.

3. Apakah Anda menyeret file crx ke halaman "Ekstensi"? Catatan ⚠️: Jangan klik "Muat ekstensi yang belum dikemas", Anda harus menyeret file crx.

4. Browser tidak mengizinkan instalasi file crx? Coba instal file zip! [Klik di sini untuk tutorial instalasi zip](./manual-chrome-extension-zip).
