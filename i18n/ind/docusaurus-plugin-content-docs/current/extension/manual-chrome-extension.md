# Panduan Instalasi Lokal Ekstensi Chrome CRX

## Unduh Paket CRX

Unduh paket instalasi crx ChatGPT Shortcut (ChatGPT_Shortcut-crx-3.x.x.zip) dan ekstrak (file crx ada di dalam folder yang diekstrak).

- **GitHub**: [GitHub Release](https://github.com/rockbenben/ChatGPT-Shortcut/releases/latest)

![](https://img.newzone.top/2024-08-12-21-47-10.png?imageMogr2/format/webp)

## Aktifkan Mode Developer

Buka halaman "Kelola Ekstensi" Chrome dan aktifkan "Mode developer".

Salin alamat berikut dan tempel ke bilah alamat browser, lalu tekan Enter untuk membuka. Aktifkan "Mode developer" di kanan atas halaman.

```txt
chrome://extensions
```

![](https://img.newzone.top/2024-08-12-22-05-52.png?imageMogr2/format/webp)

## Instal Ekstensi

Instal ekstensi ChatGPT Shortcut (Catatan ⚠️: Anda harus menyeret file .crx, JANGAN klik "Muat yang belum dikemas")

![](https://img.newzone.top/2024-08-12-22-16-38.png?imageMogr2/format/webp)

Setelah instalasi berhasil, periksa [Tutorial Penggunaan Plugin](./usage.md).

## Masalah Instalasi?

1. Pengguna Windows: Periksa apakah Anda sudah mengekstrak paket instalasi yang diunduh (bukan mengklik dua kali untuk membuka)?

2. Apakah "Mode developer" diaktifkan? Jika tidak, lihat langkah 2.

3. Apakah Anda menyeret file crx ke halaman "Ekstensi"? Catatan ⚠️: Jangan klik "Muat yang belum dikemas", Anda harus menyeret file crx.

4. Browser tidak mengizinkan instalasi file crx? Coba instal file zip! [Klik di sini untuk tutorial instalasi zip](./manual-chrome-extension-zip.md).
