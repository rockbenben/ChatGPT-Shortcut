# Instal Ekstensi Browser

Ekstensi AiShort (ChatGPT Shortcut) membantu Anda mengakses perpustakaan prompt AIShort kapan saja, meningkatkan efisiensi kerja dan belajar. Mendukung browser utama seperti Chrome, Edge, Firefox, dan menyediakan fitur sidebar dan pintasan. Anda dapat dengan cepat memanggil ekstensi melalui `Alt + Shift + S`.
