# Instal Ekstensi Browser

Ekstensi AiShort (ChatGPT Shortcut) menyematkan perpustakaan prompt langsung ke sidebar halaman obrolan AI seperti ChatGPT, Gemini, Claude, Doubao, dll., sehingga Anda tidak perlu kembali ke aishort.top untuk menyalin dan menempel. Mendukung Chrome, Edge, Firefox, dan dapat dipanggil cepat dengan `Alt + Shift + S`.

## Cara Instalasi

### 1. Toko Aplikasi (Direkomendasikan, instal sekali klik)

- **Chrome**: [Chrome Web Store](https://chrome.google.com/webstore/detail/chatgpt-shortcut/blcgeoojgdpodnmnhfpohphdhfncblnj)
- **Edge**: [Microsoft Edge Addons](https://microsoftedge.microsoft.com/addons/detail/chatgpt-shortcut/hnggpalhfjmdhhmgfjpmhlfilnbmjoin)
- **Firefox**: [Firefox Browser ADD-ONS](https://addons.mozilla.org/addon/chatgpt-shortcut/) (setelah dipasang perlu melakukan [pengaturan izin](./firefox-extension-setting) satu kali, jika tidak sidebar tidak akan muncul di ChatGPT)

### 2. Toko Tidak Bisa Dibuka? Paket Instalasi Lokal

Unduh dari saluran berikut, lalu instal mengikuti panduan terkait:

- **GitHub**: [GitHub Release](https://github.com/rockbenben/ChatGPT-Shortcut/releases/latest)

> 📖 Panduan instalasi: [File CRX](./manual-chrome-extension) · [File ZIP](./manual-chrome-extension-zip)

## Setelah Terinstal

Tekan `Alt + Shift + S` untuk memanggil, atau klik ikon di toolbar. Lihat [panduan penggunaan](./usage) untuk detail.

## Script Tampermonkey

Selain ekstensi, kami juga menyediakan [**Script Tampermonkey ChatGPT Shortcut Anywhere**](https://greasyfork.org/scripts/482907-chatgpt-shortcut-anywhere).
Tampermonkey adalah ekstensi browser yang memungkinkan pengguna menjalankan script kustom untuk meningkatkan fungsi halaman web.

Dengan script ini, Anda dapat memanggil sidebar AiShort di situs web mana pun. Namun perlu dicatat, karena halaman resmi ChatGPT membatasi injeksi script, di halaman tersebut script akan berjalan dalam bentuk **popup** alih-alih sidebar.

Setelah mengaktifkan sidebar AiShort, Anda akan melihat ikon hijau di pojok kanan bawah halaman web yang didukung. Klik ikon ini untuk membuka atau menutup sidebar. Saat ini secara default mendukung ChatGPT, Gemini, Claude, Doubao, dll.
