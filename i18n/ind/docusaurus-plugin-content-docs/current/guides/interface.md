# Panduan Antarmuka

Beranda AiShort terdiri dari empat bagian — **judul dan tombol navigasi** di paling atas, **filter tag** di bawahnya, **kotak pencarian**, dan **kartu prompt** yang memenuhi sisa halaman. Bagian ini menjelaskan cara menggunakan keempatnya secara bersamaan untuk menemukan prompt yang Anda butuhkan.

![](/img/docs/interface-home.png)

## Tombol Navigasi Atas

Tombol di bawah judul halaman sedikit berbeda sebelum dan sesudah login:

- **Belum login**: "**Masuk Gratis**" + "**Jelajahi Komunitas**"
- **Sudah login**: tombol beralih tampilan ("**Jelajahi**" ⇄ "**Koleksi Saya**"), "**Buat Prompt**", dan menu pengaturan di kanan ("Akun Saya", "Keluar")

> 💡 Anda dapat menelusuri, mencari, dan menyalin setiap prompt pilihan tanpa login. Login hanya diperlukan untuk mengumpulkan prompt, membuat prompt baru, dan sinkronisasi lintas perangkat.

## Filter Tag

Saring prompt berdasarkan bidang dan fungsinya. Tombol **OR / AND** di kanan atas menentukan logika saat Anda memilih lebih dari satu tag:

- **OR** (default): Tampilkan prompt yang cocok dengan tag mana pun
- **AND**: Hanya tampilkan prompt yang cocok dengan semua tag sekaligus

Setelah tag dipilih, tombol "**Hapus filter**" akan muncul untuk menghapus semua pilihan sekaligus. Di perangkat mobile, tag dilipat secara default — ketuk "**Tampilkan Tag**" untuk meluaskannya.

![](/img/docs/interface-filter.png)

## Pencarian Kata Kunci

Mendukung pencarian judul, deskripsi, konten, dan catatan prompt.

- Di desktop, tekan **`Ctrl + K`** (`⌘ + K` di macOS) untuk fokus ke kotak pencarian; `Esc` untuk keluar
- Pencarian di beranda mencakup **prompt pilihan**. Bila tag sudah dipilih, hasil pencarian akan terbatas pada tag tersebut
- Setelah login, cakupan pencarian otomatis diperluas ke **koleksi** dan **prompt kustom** Anda
- Jika kata kunci pendek tidak membuahkan hasil di beranda, AiShort secara otomatis juga menelusuri [Prompt Komunitas](./community)

![](/img/docs/interface-search.png)

## Kartu Prompt

Prompt pilihan diurutkan berdasarkan jumlah salinan kumulatif, sehingga konten yang paling sering digunakan ditampilkan lebih dulu.

### Informasi Kartu

Setiap kartu menampilkan judul, deskripsi, pratinjau konten, dan tag prompt. Ikon api/nyala beserta angka di pojok kanan atas adalah jumlah salinan kumulatif.

### Lihat Detail

Klik kartu untuk membuka popup detail — baca konten lengkap dan catatan, lalu salin dengan satu klik.

![](/img/docs/prompt-card.webp)

### Salin Prompt

Klik tombol "**Salin**" di kartu atau di popup detail, lalu tempel ke alat AI mana pun (halaman percakapan, tool coding, panggilan API, dll.).

### Kumpulkan Prompt

Setelah login, tombol hati akan muncul di setiap kartu — klik untuk menambahkan prompt ke [Koleksi Saya](./my-collection).
