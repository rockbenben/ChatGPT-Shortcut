# Koleksi Saya

Setelah login, tampilan **Koleksi Saya** memungkinkan Anda menata prompt favorit layaknya bookmark — atur urutannya dengan seret-lepas, beri tag kustom, dan filter berdasarkan tag agar prompt yang Anda butuhkan selalu mudah ditemukan.

:::tip Versi Online vs Versi Offline
Data koleksi versi online tersimpan di server (butuh login); data koleksi [Versi Deploy Offline](./offline) tersimpan di browser lokal (localStorage, tanpa login) — fungsinya sama persis.
:::

## Beralih Tampilan

Tombol beralih tampilan di bagian atas halaman utama menyediakan dua mode:

- **Koleksi Saya**: Menampilkan koleksi + prompt kustom Anda
- **Perpustakaan Prompt**: Jelajahi perpustakaan prompt publik

Browser akan mengingat pilihan Anda — kunjungan berikutnya langsung terbuka di tampilan terakhir yang dipakai.

![Tampilan Koleksi Saya](/img/docs/my-collection-view.gif)

## Tambah ke Koleksi

Klik tombol "Kumpulkan" di kartu untuk menambahkannya ke Koleksi Saya.

Menyimpan prompt komunitas otomatis memberikan satu upvote — turut membantu prompt berkualitas menjangkau lebih banyak pengguna.

## Hapus dari Koleksi

Di tampilan Koleksi Saya, klik ikon hati pada kartu untuk membatalkan koleksi.

## Tag Kustom

Tambahkan tag pada prompt yang dikumpulkan untuk memudahkan klasifikasi:

### Tambah Tag

1. Klik ikon tag di bilah kontrol
2. Pilih tag yang ada atau klik "+" untuk membuat baru
3. Tag baru dapat memiliki nama dan warna kustom

### Kelola Tag

Klik "Kelola Tag" untuk masuk ke antarmuka manajemen:

- Ubah nama tag
- Ubah warna tag (11 warna preset)
- Hapus tag

![Popup Manajemen Tag](/img/docs/my-collection-tag.png)

### Filter berdasarkan Tag

Bilah filter tampilan Koleksi Saya dapat memfilter berdasarkan tag kustom.

## Pengurutan Seret-Lepas

Mendukung pengaturan urutan kartu dengan seret-lepas di tampilan Koleksi Saya:

1. Seret pegangan di sisi kiri kartu
2. Pindahkan ke posisi target lalu lepaskan
3. Urutan baru otomatis tersimpan (versi online tersimpan di server, versi offline tersimpan di lokal browser)

Pengurutan hanya tersedia di tampilan Koleksi Saya.

![Operasi Pengurutan Seret-Lepas](/img/docs/my-collection-drag.gif)
