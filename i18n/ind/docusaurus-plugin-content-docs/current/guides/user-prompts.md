# Prompt Kustom

Setelah login, Anda dapat membuat prompt sendiri untuk akses cepat nanti. Prompt kustom akan disimpan ke akun Anda dan ditampilkan di tampilan Koleksi Saya.

## Buat Prompt

1. Klik tombol "Tambah Prompt" di kanan atas halaman
2. Isi informasi prompt:
   - **Judul**: Deskripsi singkat tujuan prompt
   - **Deskripsi**: Konten lengkap prompt
   - **Catatan**: Petunjuk penggunaan atau catatan (opsional)
3. Pilih apakah akan dibagikan ke komunitas
4. Klik Kirim

![Popup Tambah Prompt](/img/docs/user-prompts-create.png)

## Edit Prompt

Di tampilan Koleksi Saya, klik kartu prompt yang Anda buat untuk:

- Ubah judul, deskripsi, dan catatan
- Alihkan status berbagi (Publik/Pribadi)
- Simpan perubahan

![Antarmuka Edit Prompt](/img/docs/user-prompts-edit.png)

## Hapus Prompt

Klik tombol "Hapus" di antarmuka edit untuk menghapus prompt. Penghapusan tidak dapat dibatalkan, harap lakukan dengan hati-hati.

## Bagikan ke Komunitas

Saat membuat atau mengedit prompt, centang opsi "Bagikan ke Komunitas":

- **Publik**: Prompt akan ditampilkan di halaman komunitas, pengguna lain dapat melihat dan mengumpulkannya
- **Pribadi**: Hanya terlihat oleh Anda sendiri, tidak akan muncul di halaman komunitas

Anda dapat mengubah status berbagi kapan saja.

## Ekspor Cadangan

Ekspor semua prompt yang Anda buat:

1. Masuk ke Pusat Pengguna
2. Klik tombol "Ekspor Prompt"
3. Sistem menghasilkan file JSON dan mengunduh secara otomatis

## Impor Prompt

Impor prompt dan favorit dari file JSON:

1. Masuk ke Pusat Pengguna
2. Klik tombol "Impor Prompt"
3. Pilih file JSON
4. Sistem secara otomatis menggabungkan data
