# ¿Cómo usar la extensión del navegador?

ChatGPT Shortcut soporta 18 idiomas principales. El idioma de la extensión se configura automáticamente según tu navegador. También puedes cambiarlo manualmente en la página de configuración.

![](/img/docs/extension-settings.png)

## Modo de visualización

La extensión admite tres modos de visualización — elige el que mejor se adapte a tu forma de trabajar:

- **Modo barra lateral**: se abre en la barra lateral del navegador — se muestra junto a la página web, ideal para consultar prompts mientras chateas con la IA
- **Modo popup**: se muestra como una ventana emergente al hacer clic en el icono — consulta rápida y se cierra, sin ocupar pantalla
- **Modo ventana**: se abre en una ventana flotante independiente — se puede arrastrar a un monitor secundario o colocar aparte

## Mantener panel lateral abierto

Al activar esta opción, la barra lateral permanecerá abierta al cambiar de pestaña.

## Modo oscuro

La extensión soporta modo oscuro. Activa "Modo oscuro" en configuración para cambiar la interfaz a apariencia oscura.

![](/img/docs/extension-sidebar.webp)

## Activación por atajo

Usa el atajo `Alt+Shift+S` para activar directamente la extensión.
