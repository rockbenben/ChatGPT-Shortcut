# 如何使用瀏覽器擴展？

ChatGPT Shortcut 支持 13 種主流語言，擴展語言將根據你的瀏覽器環境自動設置。你也可以在設置頁面手動更改語言。

![](/img/docs/extension-settings.png)

## 顯示模式

擴展支持三種顯示模式：

- **側邊欄模式**：在瀏覽器側邊欄中打開，可與網頁內容並排顯示
- **彈窗模式**：點擊擴展圖標後以彈窗形式顯示
- **獨立窗口模式**：在獨立的浮動窗口中打開

## 保持側邊欄打開

開啓此選項後，側邊欄將在瀏覽器標籤頁切換時保持開啓狀態，方便持續使用。

## 深色模式

擴展支持深色模式，可在設置頁面中開啓"深色模式"選項，啓用後界面將切換爲深色外觀。

![](/img/docs/extension-sidebar.png)

## 熱鍵激活

使用 `Alt+Shift+S` 快捷鍵可以直接激活擴展。
