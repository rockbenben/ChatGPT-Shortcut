# 如何使用瀏覽器擴展？

ChatGPT Shortcut 支持 18 種主流語言，擴展語言將根據你的瀏覽器環境自動設置。你也可以在設置頁面手動更改語言。

![](/img/docs/zh/extension-settings.png)

## 顯示模式

擴展支持三種顯示模式，按你的使用習慣選一種：

- **側邊欄模式**：在瀏覽器側邊欄打開——與網頁並排顯示，適合一邊看 AI 對話頁一邊瀏覽提示詞
- **彈窗模式**：點擴展圖標彈出小窗——快速查看後即關，不佔屏幕
- **獨立窗口模式**：在獨立的浮動窗口打開——可以拖到副屏或獨立擺放

## 保持側邊欄打開

開啓此選項後，側邊欄將在瀏覽器標籤頁切換時保持開啓狀態，方便持續使用。

## 深色模式

擴展支持深色模式，可在設置頁面中開啓「深色模式」選項，啓用後界面將切換爲深色外觀。

![](/img/docs/zh/extension-sidebar.webp)

## 熱鍵激活

使用 `Alt+Shift+S` 快捷鍵可以直接激活擴展。
