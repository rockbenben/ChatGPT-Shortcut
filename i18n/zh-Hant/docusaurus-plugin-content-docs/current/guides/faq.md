# 常見問題

## AiShort 是 AI 提示詞生成器嗎？

不是。AiShort 是依場景**精選 + 整理**的人工 prompt 範本庫，並不是用 LLM 自動生成 prompt 的工具。每條 prompt 經過篩選並依標籤分類，一鍵複製貼上到 ChatGPT / Claude / Gemini / DeepSeek 等任意 AI 對話工具即可。如果你想要「從需求自動生成 prompt」那類工具，PromptPerfect、ChatGPT prompt generator 是另一類產品。

## AiShort 需要付費嗎？需要 API Key 嗎？

完全免費，**無需 API Key、無需註冊**。瀏覽、搜尋、複製提示詞都不需要任何帳號。

註冊後可解鎖收藏與拖曳排序、自訂標籤、建立管理私人提示詞、社群分享與投票、JSON 資料匯出、跨裝置同步等擴充功能——一樣免費。程式碼全部開源於 [GitHub](https://github.com/rockbenben/ChatGPT-Shortcut)。

注意：AiShort 只提供提示詞，**呼叫 AI 模型本身是否收費由你使用的 AI 服務決定**（例如 ChatGPT 免費版/付費版、API 計費等）。

## 一共收錄了多少提示詞？涵蓋哪些場景？

AiShort 累計收錄 **5000+ 條提示詞**，分為兩個庫：

- **精選庫** —— 人工篩選，按 25+ 場景標籤分類，18 種語言翻譯完整。涵蓋標籤：寫作輔助、IT/程式設計、文章/報告、SEO/行銷、企業職能、學術/教師、教育/學生、語言/翻譯、心理/社交、思維訓練、效率工具、終端/直譯器、辯論/演講、點評/評鑑、趣味科普、生活百科、醫療健康、金融顧問、音樂藝術、哲學/宗教、文本/詞語、趣味遊戲、專業顧問等。
- **[社群庫](./community)** —— 占絕大多數，由使用者持續投稿，可按熱度或最新發佈排序，支援搜尋篩選；場景比精選庫更細更廣。

精選庫搜不到合適的，去社群庫再搜一次。

## 支援哪些 AI 模型？同一個提示詞跨模型通用嗎？

AiShort 的提示詞是通用的，**任何能輸入提示詞的 AI 場景都適用**——不只對話頁，還包括 Cursor 等程式設計工具、API 呼叫、AI 代理等。下面是常見的對話類模型：

- 國際：ChatGPT、Gemini、Claude、Grok
- 中文：DeepSeek、通義千問、文心一言、豆包、Kimi、智譜清言、訊飛星火、騰訊元寶、TAIDE
- API 平台：OpenRouter、矽基流動、Groq

更多模型清單見 [開始上手 → 常用 AI 模型](./getting-started#常用-ai-模型)。

**跨模型通用性**：絕大多數文本類 prompt 跨模型通用——寫作、翻譯、程式設計、問答這些通用任務，同一份 prompt 在主流大型語言模型上都能跑出有效結果，AiShort 每條 prompt 預設不綁定特定模型。但**表現有差異**：寫作類 Claude 通常細節更細膩、ChatGPT-5 跟指令更緊；程式設計類 GPT-5、Gemini Pro、DeepSeek 各有強項；推理密集題 Claude Opus / Gemini Deep Thinking / o-series 通常更穩。繪圖類 prompt（Midjourney / Stable Diffusion / DALL·E）則不通用，需要依各自語法調整。

## 為什麼提示詞使用英文？

AI 模型對英文理解更準，輸出更穩定。中文提示詞也能用，但多次執行同一條中文提示詞，結果差異會較大。重要場景建議使用英文。

> 💡 想要中文回覆？在提示詞結尾加一句 `respond in Chinese`；如需繁體中文，可改為 `respond in Traditional Chinese`。

## 每次都要輸入提示詞嗎？

**API 呼叫**：把提示詞設為 `system prompt`，後續對話自動生效。

**網頁版**：不切換對話的話，ChatGPT 會記住目前的提示詞。如果回覆開始走偏，代表它「忘」了，重新貼上一次即可。

**小技巧**：把常用對話儲存為瀏覽器書籤，下次直接打開。

## 怎麼挑到合適的提示詞？

依你要的輸出反向找：要寫文章用 [`write`](/?tags=write) 或 [`article`](/?tags=article) 標籤；要寫程式用 [`code`](/?tags=code) 或 [`interpreter`](/?tags=interpreter)；要翻譯用 [`language`](/?tags=language)；做角色扮演用 [`games`](/?tags=games)；不熟悉標籤時直接用關鍵字搜尋。每條 prompt 卡片會顯示熱度（複製次數），複製次數高的通常品質更穩。

## 為什麼搜尋不到相關的提示詞？

首頁搜尋範圍是**精選提示詞庫**（登入後還包含你的個人提示詞），**不包含**社群分享的提示詞。

如果用短關鍵字在首頁搜不到，去 [社群提示詞](./community) 頁面再搜一次——那裡有更多使用者分享的內容。

## AI 輸出了虛假資訊怎麼辦？

AI 有時會產生「幻覺」，輸出看似合理但實際錯誤的資訊。處理方法：

1. **核實關鍵資訊**：特別是資料、引用、程式碼等
2. **多輪優化**：讓 AI 對回答進行再次檢查與優化
3. **交叉驗證**：用不同的提示詞或模型驗證重要結論

合適的提示詞能減少 AI 幻覺。

## 提示詞效果不理想怎麼調？

不要直接換提示詞，先從這幾個方向調整：

1. **把方括號 `[佔位符]` 改得更具體**——加上風格、長度、領域、讀者輪廓等細節
2. **追問 AI 優化**：對不滿意的回覆追問「請改得更 X」或「請用 Y 風格重寫」，AI 通常一兩輪迭代後就能貼近目標，也可讓 AI 對自己的回答評分並改進
3. **切換模型對比**：同一 prompt 在 Claude Sonnet / ChatGPT / Gemini / DeepSeek 表現差異有時很大——寫作類 Claude 通常更細膩，程式設計類 GPT-5 / DeepSeek 更穩
4. **試試社群**：[社群提示詞](./community) 裡可能有更適合的版本——也歡迎把你的好提示詞分享到那裡
5. **回饋建議**：遇到問題或有改進建議？歡迎 [回饋](/feedback)

## 如何備份我的提示詞？

1. 進入「我的帳戶」→ 找到「資料管理 → 匯出提示詞」一欄
2. 點「匯出資料」按鈕
3. 系統自動產生 JSON 檔案下載

建議定期備份，避免資料遺失。

## AiShort 上的提示詞可以用於商業專案嗎？

可以，但要分情況看授權：

1. 來自 [Awesome ChatGPT Prompts](https://github.com/f/awesome-chatgpt-prompts) 的內容遵循 **CC0 公共領域**，沒有任何限制
2. AiShort 自建與社群投稿內容預設採用 **CC BY-SA 4.0** 開放授權——商用、二次散布、改編都允許，要求保留 AiShort / 原作者署名並允許下游同樣開放
3. 你**用 prompt 生成的 AI 內容**版權依你所使用 AI 服務商（OpenAI / Anthropic / Google 等）的使用條款歸屬，多數情況歸你
4. prompt 卡片說明裡明確標註「僅個人用」的極少數條目除外

## 用 AiShort 複製提示詞會洩漏我的資料嗎？

不會。AiShort 的提示詞以靜態 JSON 形式打包進站點資源，**複製動作本身是在你瀏覽器本地的剪貼簿完成**。

你**填入方括號佔位符的具體內容、AI 給你的回覆**——這些都直接在你與所選 AI 平台（ChatGPT / Claude / Gemini / DeepSeek 等）之間傳輸，AiShort 看不到。

需要說明的是：複製時會向後端傳送**匿名計數事件**（POST `/cards/<id>/copy`），僅用於統計每條提示詞的熱度（卡片上顯示的「複製次數」）。**只傳卡片 ID，不包含**你填的內容、不包含個人資訊、不關聯使用者身分。

**登入帳號後**的收藏清單、自訂提示詞、社群投稿才會同步到後端用於跨裝置同步，且支援隨時 JSON 匯出與一鍵刪除全部資料。
