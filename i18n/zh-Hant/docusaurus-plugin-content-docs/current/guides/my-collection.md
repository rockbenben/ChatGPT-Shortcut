# 我的收藏

登錄後，**我的收藏**視圖讓你像管理書籤一樣組織喜歡的提示詞——拖拽排序、自定義標籤、按標籤篩選，方便快速調用。

:::tip 在線版與離線版
在線版收藏數據存在服務器（需登錄）；[離線部署版](./offline) 收藏數據存在瀏覽器本地（localStorage，無需登錄），功能一致。
:::

## 視圖切換

首頁頂部的視圖切換按鈕提供兩種模式：

- **我的收藏**：展示你的收藏 + 自定義提示詞
- **提示詞庫**：瀏覽公共提示詞庫

瀏覽器會記住你的選擇，下次訪問自動回到上次的視圖。

![我的收藏視圖](/img/docs/my-collection-view.gif)

## 添加收藏

在卡片上點擊「收藏」按鈕即可加入我的收藏。

收藏社區提示詞會自動投一票贊成，幫優質內容獲得更多曝光。

## 移除收藏

在我的收藏視圖，點擊卡片上的愛心圖標即可取消收藏。

## 自定義標籤

給收藏的提示詞加標籤，方便分類：

### 添加標籤

1. 點擊控制欄的標籤圖標
2. 選已有標籤或點「+」新建
3. 新標籤可自定義名稱和顏色

### 管理標籤

點「管理標籤」進入管理界面：

- 改標籤名稱
- 換標籤顏色（11 種預設色）
- 刪除標籤

![標籤管理彈窗](/img/docs/my-collection-tag.png)

### 按標籤篩選

我的收藏視圖的篩選欄可按自定義標籤篩選。

## 拖拽排序

在我的收藏視圖支持拖拽調整卡片順序：

1. 拖動卡片左側的拖拽手柄
2. 移到目標位置後釋放
3. 新順序自動保存（在線版存服務器，離線版存瀏覽器本地）

排序僅在我的收藏視圖可用。

![拖拽排序操作](/img/docs/my-collection-drag.gif)
