# 開始上手

在與 AI 對話時，好的提示詞能幫助你更快獲得高質量的回答。AiShort 爲你整理了大量實用提示詞，複製粘貼即可使用。

## 基本使用

1. 在首頁搜索或選擇你需要的提示詞
2. 點擊卡片查看詳情，或直接點擊「複製」按鈕
3. 打開任意 AI 模型，將提示詞粘貼進去
4. 按照提示詞的指引，補充你的具體問題

![](/img/docs/how-to-use-aishort.gif)
