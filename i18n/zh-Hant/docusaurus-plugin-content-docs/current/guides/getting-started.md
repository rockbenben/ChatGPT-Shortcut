# 開始上手

好的提示詞決定 AI 輸出的質量。AiShort 收錄大量精選提示詞，複製粘貼即用。

> 📋 **開始前你需要**：一個瀏覽器 + 任意 AI 工具——ChatGPT 等對話頁、Cursor 等編程工具、API 調用等場景都適用。AiShort 網頁版無需註冊。

## 基本使用

1. 在首頁搜索或瀏覽感興趣的提示詞
2. 找到合適的提示詞，點「複製」按鈕（或先點開卡片看完整內容）
3. 粘貼到任意 AI 工具——對話頁、編程工具、API 調用均可
4. 按提示詞中的**佔位符**（如 `[內容]`、`[主題]`）補充你的具體需求，發送給 AI

![](/img/docs/zh/how-to-use-aishort.webp)

> 💡 **小貼士**：大部分提示詞用方括號 `[...]` 標記需要你補充的內容——把方括號裏的佔位文本換成你的具體內容即可。
