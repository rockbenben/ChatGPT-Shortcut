# 社區提示詞

社區提示詞版塊展示由用戶分享的自定義提示詞。您可以在這裏發現靈感，也可以分享自己的創意。

![](/img/docs/community-page.png)

## 排序和篩選

社區頁面提供多種篩選方式：

- **發佈時間**：查看最新分享的提示詞
- **投票排序**：查看獲得最多贊成票的提示詞
- **關鍵詞搜索**：搜索特定主題的提示詞

## 投票機制

通過贊/踩投票決定提示詞的曝光度：

- **贊成**：幫助優質內容獲得更多曝光
- **反對**：踩數達到一定數量後，提示詞會被隱藏

**收藏聯動**：收藏社區提示詞時，系統會自動投一票贊成。

![投票按鈕狀態](/img/docs/community-vote.png)

> 投票僅對登錄用戶開放，每人對同一提示詞的投票以最近一次爲準。

## 私密提示詞

不想公開的創意可以設爲私密：

- 創建或編輯提示詞時，取消「分享到社區」選項
- 私密提示詞僅自己可見
- 可隨時切換公開/私密狀態
