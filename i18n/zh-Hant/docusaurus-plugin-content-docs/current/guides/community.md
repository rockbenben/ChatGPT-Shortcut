# 社區提示詞

社區提示詞由用戶分享，你可以在這裏發現靈感，也能分享自己的創意。**投票機制**幫優質內容浮現——大家覺得好用的提示詞會被更多人看到，質量差的逐漸沉下去。

![](/img/docs/community-page.png)

## 排序和篩選

- **發佈時間**：看最新分享
- **投票排序**：看獲得最多贊的提示詞
- **關鍵詞搜索**：搜特定主題

## 投票機制

贊/踩決定提示詞的曝光：

- **贊成**：讓優質內容更多人看到
- **反對**：踩數達到閾值後，提示詞會被自動隱藏

**收藏聯動**：收藏社區提示詞會自動投一票贊成。

![投票按鈕狀態](/img/docs/community-vote.png)

> 投票僅對登錄用戶開放，同一提示詞以最近一次投票爲準。

## 私密提示詞

不想公開分享的創意可以設爲私密：

- 創建或編輯時取消「分享到社區」
- 私密提示詞僅自己可見
- 可隨時切換公開/私密
