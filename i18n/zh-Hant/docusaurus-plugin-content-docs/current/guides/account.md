# 賬戶管理

AiShort 不登入也能用——但**登入後**你可以收藏喜歡的提示詞、建立自己的提示詞、分享到社群，並跨裝置同步。下面介紹登入、註冊、密碼與資料的相關操作。

## 登入方式

登入彈窗提供「**密碼登入**」和「**免密登入**」兩種方式，外加 Google 一鍵登入。

### 密碼登入

用使用者名稱或註冊郵箱 + 密碼登入。

### 免密登入

1. 切到「免密登入」，輸入註冊郵箱
2. 點「**獲取登入連結**」——登入連結將發送至你的郵箱
3. 打開郵件中的連結即可登入

不用記密碼，適合不常登入的場景。

### Google 登入

點登入彈窗的 Google 按鈕，一鍵登入。首次會自動建立帳戶，不用單獨註冊。

![](/img/docs/zh/account-login.png)

## 註冊賬戶

1. 在登入彈窗點「**立即註冊**」切到註冊
2. 填郵箱、使用者名稱和密碼，並勾選同意「隱私政策」和「服務條款」
3. 提交後自動登入

![](/img/docs/zh/account-register.png)

## 我的帳戶

登入後進入「**我的帳戶**」頁面，可以管理資料、密碼、等級和資料。

![我的帳戶頁面](/img/docs/zh/account-profile.png)

### 修改使用者名稱

點使用者名稱旁的編輯圖示，改完儲存即可。

### 修改密碼

1. 輸入當前密碼
2. 輸入新密碼並確認
3. 點「修改密碼」提交

忘記密碼時，在登入彈窗點「忘記密碼」，輸入註冊郵箱點「發送重置郵件」，按郵件連結重置即可。

### 我的等級

帳戶頁展示你的**等級（L0–L9）**和升級進度——等級由你**分享到社群的提示詞數量**決定，分享越多檔位越高。

## 資料管理

「我的帳戶 → 資料管理」一欄提供三項操作：

- **清除快取** — 刷新本地快取資料；資料不同步或顯示異常時使用，收藏和自訂提示詞不會丟，只是強制重新同步
- **匯出提示詞** — 匯出為 JSON 檔案，方便備份（詳見[自定義提示詞](./user-prompts#導出備份)）
- **匯入提示詞** — 從 JSON 檔案匯入提示詞和收藏
