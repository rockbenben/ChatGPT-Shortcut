# 自定義提示詞

登錄後，您可以創建自己的提示詞，方便日後快速調用。自定義提示詞會保存到您的賬戶中，並顯示在我的收藏視圖。

## 創建提示詞

1. 點擊頁面右上角的「添加提示詞」按鈕
2. 填寫提示詞信息：
   - **標題**：簡短描述提示詞用途
   - **描述**：提示詞的完整內容
   - **備註**：使用說明或注意事項（可選）
3. 選擇是否分享到社區
4. 點擊提交

![添加提示詞彈窗](/img/docs/user-prompts-create.png)

## 編輯提示詞

在我的收藏視圖中，點擊您創建的提示詞卡片，可以：

- 修改標題、描述和備註
- 切換分享狀態（公開/私密）
- 保存更改

![編輯提示詞界面](/img/docs/user-prompts-edit.png)

## 刪除提示詞

在編輯界面中點擊「刪除」按鈕可刪除提示詞。刪除後無法恢復，請謹慎操作。

## 分享到社區

創建或編輯提示詞時，勾選「分享到社區」選項：

- **公開**：提示詞會顯示在社區頁面，其他用戶可以看到並收藏
- **私密**：僅自己可見，不會出現在社區頁面

您可以隨時修改分享狀態。

## 導出備份

導出您創建的所有提示詞：

1. 進入個人中心
2. 點擊「導出提示詞」按鈕
3. 系統生成 JSON 文件並自動下載

導出內容包括：

- 提示詞標題、描述、備註
- 創建和更新時間
- 分享狀態

建議定期導出備份，防止數據丟失。

## 導入提示詞

從 JSON 文件導入提示詞和收藏：

1. 進入個人中心
2. 點擊「導入提示詞」按鈕
3. 選擇 JSON 文件
4. 系統自動合併數據

### 團隊協作

在團隊內分享提示詞的推薦流程：

1. **篩選與分享**：導出數據後，可手動刪除收藏列表或篩選出需要分享的提示詞，再發送文件給團隊成員導入
2. **隱私保護**：導入他人分享的提示詞（ID 不屬於當前賬戶）會自動設爲**私密**，確保各成員數據互不干擾
