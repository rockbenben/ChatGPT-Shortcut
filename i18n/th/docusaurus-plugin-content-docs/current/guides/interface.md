# คู่มืออินเทอร์เฟซ

หน้าแรกของ AiShort ประกอบด้วยสี่ส่วนหลัก ได้แก่ **ชื่อหน้าและปุ่มนำทาง** ด้านบน **ตัวกรองแท็ก** ด้านล่าง **แถบค้นหา** และ **การ์ด prompt** ที่เรียงอยู่ส่วนที่เหลือของหน้า หัวข้อนี้จะแนะนำวิธีใช้งานทั้งสี่ส่วนร่วมกันเพื่อค้นหา prompt ที่ต้องการได้อย่างรวดเร็ว

![](/img/docs/interface-home.png)

## ปุ่มนำทางด้านบน

ปุ่มด้านล่างชื่อหน้าจะแตกต่างกันก่อนและหลังเข้าสู่ระบบ:

- **ยังไม่ได้เข้าสู่ระบบ**: "**เข้าสู่ระบบฟรี**" + "**ดูชุมชน**"
- **เข้าสู่ระบบแล้ว**: ปุ่มสลับมุมมอง ("**สำรวจ**" ⇄ "**คอลเลกชันของฉัน**"), "**สร้างพรอมต์**" และเมนูตั้งค่าทางขวา ("บัญชีของฉัน", "ออกจากระบบ")

> 💡 คุณสามารถเรียกดู ค้นหา และคัดลอก prompt ที่คัดสรรไว้ได้โดยไม่ต้องเข้าสู่ระบบ การเข้าสู่ระบบจะเพิ่มความสามารถในการรวบรวม สร้าง และซิงค์ข้ามอุปกรณ์

## การกรองแท็ก

กรอง prompt ตามโดเมนและฟังก์ชัน ตัวสลับ **OR / AND** ที่มุมขวาบนเป็นตัวกำหนดกฎการรวมแท็กหลายรายการ:

- **OR** (ค่าเริ่มต้น): แสดง prompt ที่ตรงกับแท็กใดแท็กหนึ่งที่เลือก
- **AND**: แสดงเฉพาะ prompt ที่ตรงกับทุกแท็กที่เลือกพร้อมกัน

เมื่อเลือกแท็กแล้ว ปุ่ม "**ล้างตัวกรอง**" จะปรากฏขึ้นเพื่อให้ล้างทั้งหมดได้ในคลิกเดียว บนอุปกรณ์มือถือ แท็กจะถูกย่อไว้เป็นค่าเริ่มต้น — แตะ "**แสดงแท็ก**" เพื่อขยายออก

![](/img/docs/interface-filter.png)

## ค้นหาคำสำคัญ

ค้นหาครอบคลุมชื่อ prompt คำอธิบาย เนื้อหา และหมายเหตุ

- บนเดสก์ท็อป กด **`Ctrl + K`** (`⌘ + K` บน macOS) เพื่อโฟกัสช่องค้นหา กด `Esc` เพื่อออก
- การค้นหาในหน้าแรกครอบคลุม **prompt ที่คัดสรร** เมื่อเลือกแท็กไว้แล้ว การค้นหาจะจำกัดอยู่ในแท็กเหล่านั้น
- หลังเข้าสู่ระบบ การค้นหาจะขยายไปครอบคลุม **คอลเลกชัน** และ **prompt ที่กำหนดเอง** ของคุณด้วย
- หากคำสำคัญสั้น ๆ ไม่พบผลในหน้าแรก AiShort จะค้นหาใน [Prompt ชุมชน](./community) ให้โดยอัตโนมัติ

![](/img/docs/interface-search.png)

## การ์ด Prompt

Prompt ที่คัดสรรจะจัดเรียงตามจำนวนครั้งที่คัดลอกสะสม ทำให้เนื้อหาที่ถูกใช้งานมากที่สุดขึ้นมาแสดงก่อน

### ข้อมูลบนการ์ด

แต่ละการ์ดจะแสดงชื่อ คำอธิบาย ตัวอย่างเนื้อหา และแท็กของ prompt ไอคอนเปลวไฟและตัวเลขที่มุมขวาบนคือจำนวนครั้งที่ถูกคัดลอกสะสม

### ดูรายละเอียด

คลิกการ์ดเพื่อเปิดป๊อปอัพรายละเอียด — อ่านเนื้อหาทั้งหมดและหมายเหตุ จากนั้นคัดลอกได้ด้วยคลิกเดียว

![](/img/docs/prompt-card.webp)

### คัดลอก Prompt

คลิกปุ่ม "**คัดลอก**" บนการ์ดหรือในป๊อปอัพรายละเอียด แล้ววางลงในเครื่องมือ AI ใดก็ได้ (หน้าสนทนา เครื่องมือเขียนโค้ด การเรียก API ฯลฯ)

### รวบรวม Prompt

หลังเข้าสู่ระบบ ปุ่มรูปหัวใจจะปรากฏบนแต่ละการ์ด — คลิกเพื่อเพิ่ม prompt เข้า [คอลเลกชันของฉัน](./my-collection)
