# คู่มืออินเทอร์เฟซ

หน้า AiShort ให้อินเทอร์เฟซที่ใช้งานง่ายแก่ผู้ใช้ในการเรียกดูและกรอง prompt หน้านี้ประกอบด้วยพื้นที่แท็ก พื้นที่ค้นหา และพื้นที่แสดง prompt

![](/img/docs/interface-home.png)

## 🏷︎ การกรองแท็ก

พื้นที่แท็กช่วยให้ผู้ใช้กรอง prompt ตามโดเมนและฟังก์ชัน

ผ่านปุ่ม "สลับกฎการกรองแท็ก" ที่มุมขวาบน คุณสามารถกรองหลายแท็กได้แม่นยำยิ่งขึ้น:

- **กฎ OR** (ค่าเริ่มต้น): แสดง prompt ที่มีแท็กใดก็ได้ที่เลือก
- **กฎ AND**: แสดงเฉพาะ prompt ที่มีแท็กทั้งหมดที่เลือกพร้อมกัน

![](/img/docs/interface-filter.png)

## 🔍 ค้นหาคำสำคัญ

ฟังก์ชันค้นหาคำสำคัญรองรับการค้นหาหัวข้อ คำอธิบาย และเนื้อหา prompt

- การค้นหาในหน้าแรกจะรวมถึง **พรอมต์ที่คัดสรรแล้ว** เป็นหลัก (ไม่รวมพรอมต์ของชุมชน) หากเลือกแท็กไว้แล้ว ขอบเขตการค้นหาจะจำกัดอยู่แค่แท็กเหล่านั้น
- หลังจากเข้าสู่ระบบ ขอบเขตการค้นหาจะขยายเพื่อครอบคลุม **คอลเลกชัน** และ **พรอมต์ที่กำหนดเอง** ของคุณ
- หากคุณไม่พบผลลัพธ์ที่คาดหวังในหน้าแรกด้วยคีย์เวิร์ดสั้นๆ โปรดไปค้นหาที่หน้า [พรอมต์ของชุมชน](./community)

![](/img/docs/interface-search.png)

## 🔬 การ์ด Prompt

### ดูรายละเอียด

คลิกการ์ด prompt เพื่อเปิดป๊อปอัพรายละเอียด ซึ่งคุณสามารถดูหัวข้อ คำอธิบาย เนื้อหา prompt ทั้งหมด หมายเหตุ และปุ่มคัดลอก

![](/img/docs/prompt-card.gif)

### คัดลอก Prompt

คลิกปุ่ม "คัดลอก" ที่ด้านล่างของการ์ดเพื่อรับ prompt วางลงในโมเดล AI เพื่อใช้งาน

### รวบรวม Prompt

หลังเข้าสู่ระบบ ปุ่มรวบรวมจะปรากฏบนการ์ด คลิกเพื่อเพิ่ม prompt ในคอลเลกชันของฉัน

## 🔥 จัดเรียงตามความนิยม

หน้าจัดเรียง prompt ตามความนิยมเพื่อช่วยให้คุณค้นพบเนื้อหาที่กำลังเป็นที่นิยมได้อย่างรวดเร็ว
