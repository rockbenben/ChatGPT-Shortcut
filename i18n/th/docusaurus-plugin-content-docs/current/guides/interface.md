# คู่มืออินเทอร์เฟซ

หัวข้อนี้จะพาคุณรู้จักสามพื้นที่หลักของหน้าแรก AiShort ได้แก่ **พื้นที่แท็ก** (แท็กสถานการณ์ด้านบน), **พื้นที่ค้นหา** (ช่องป้อนคำสำคัญ) และ **พื้นที่แสดง prompt** (รายการการ์ดด้านล่าง) พร้อมเคล็ดลับการใช้งานเพื่อค้นเจอ prompt ที่ต้องการได้อย่างรวดเร็ว

![](/img/docs/interface-home.png)

## 🏷︎ การกรองแท็ก

กรอง prompt ตามโดเมนและฟังก์ชัน ตัวสลับ **OR / AND** ที่มุมขวาบนเป็นตัวกำหนดกฎการกรองหลายแท็ก:

- **OR** (ค่าเริ่มต้น): แสดงเมื่อตรงกับแท็กใดก็ได้
- **AND**: ต้องตรงกับทุกแท็กพร้อมกัน

![](/img/docs/interface-filter.png)

## 🔍 ค้นหาคำสำคัญ

รองรับการค้นหาหัวข้อ คำอธิบาย และเนื้อหา prompt

- ขอบเขตการค้นหาในหน้าแรกคือ **prompt ที่คัดสรร** (ไม่รวมชุมชน) เมื่อเลือกแท็กไว้แล้ว ขอบเขตการค้นหาจะจำกัดอยู่ในแท็กเหล่านั้น
- หลังเข้าสู่ระบบ ขอบเขตการค้นหาจะขยายไปยัง **คอลเลกชัน** และ **prompt ที่กำหนดเอง** ของคุณ
- เมื่อใช้คำสำคัญสั้น ๆ ค้นหาที่หน้าแรกแล้วไม่เจอ ให้ไปค้นที่หน้า [Prompt ชุมชน](./community) อีกครั้ง

![](/img/docs/interface-search.png)

## 🔬 การ์ด Prompt

### ดูรายละเอียด

คลิกการ์ดเพื่อเปิดป๊อปอัพรายละเอียด ดูเนื้อหาทั้งหมด หมายเหตุ และคัดลอกได้ด้วยคลิกเดียว

![](/img/docs/prompt-card.gif)

### คัดลอก Prompt

คลิกปุ่ม "คัดลอก" ที่ด้านล่างของการ์ด แล้ววางลงในเครื่องมือ AI ใดก็ได้ (หน้าสนทนา, เครื่องมือเขียนโค้ด, การเรียก API ฯลฯ)

### รวบรวม Prompt

หลังเข้าสู่ระบบ ปุ่มรวบรวมจะปรากฏบนการ์ด คลิกเพื่อเพิ่มเข้าคอลเลกชันของฉัน

## 🔥 จัดเรียงตามความนิยม

จัดลำดับ prompt ตามความนิยม ช่วยให้คุณเจอเนื้อหามาแรงได้ง่ายขึ้น ลำดับของแท็กเองก็ปรับตามความนิยมเช่นกัน จึงสะท้อนเทรนด์ล่าสุดของชุมชนแบบเรียลไทม์
