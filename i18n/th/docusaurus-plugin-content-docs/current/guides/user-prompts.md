# Prompt ที่กำหนดเอง

หลังเข้าสู่ระบบ คุณสามารถสร้าง prompt ของตัวเองและบันทึกไว้ในบัญชีเพื่อเรียกใช้ได้ทุกเมื่อ Prompt ที่กำหนดเองจะปรากฏในมุมมอง "คอลเลกชันของฉัน"

## สร้าง Prompt

1. หลังเข้าสู่ระบบ คลิกปุ่ม "**สร้างพรอมต์**" ที่ด้านบนของหน้า
2. กรอกข้อมูลในหน้าต่าง "สร้างพรอมต์":
   - **ชื่อ พรอมต์** (จำเป็น): ตั้งชื่อให้ prompt
   - **เนื้อหา พรอมต์** (จำเป็น): เนื้อความของ prompt ข้อความในวงเล็บเหลี่ยม `[...]` จะถูกไฮไลต์เป็น placeholder ตอนแสดงผล
   - **หน้าที่/การใช้งาน** (ไม่บังคับ): อธิบายสั้น ๆ ว่า prompt นี้ทำอะไร
   - **หมายเหตุ** (ไม่บังคับ): แหล่งที่มา เวอร์ชันภาษาอื่น หรือข้อมูลเสริม
3. สวิตช์ "คุณต้องการแชร์ พรอมต์ นี้ไปยังหน้าสาธารณะหรือไม่?" ที่ด้านล่างเปิดอยู่เป็นค่าเริ่มต้น — ปิดเพื่อเก็บ prompt ไว้เป็นส่วนตัว
4. คลิก "สร้างพรอมต์" เพื่อส่ง

![ป๊อปอัพสร้าง prompt](/img/docs/user-prompts-create.png)

> 💡 **ตัวอย่าง** — prompt "ผู้ช่วยสรุปรายงานประจำวัน":
> - **ชื่อ พรอมต์**: ผู้ช่วยสรุปรายงานประจำวัน
> - **เนื้อหา พรอมต์**: คุณคือผู้ช่วยสื่อสารที่กระชับ โปรดแปลงโน้ตต่อไปนี้ให้เป็นรายงานประจำวันที่มีสามส่วน — สิ่งที่ทำเมื่อวาน สิ่งที่จะทำวันนี้ และสิ่งที่ติดขัด โดยแต่ละข้อให้สั้นและเน้นผลลัพธ์: [โน้ตดิบของฉัน]
> - **หน้าที่/การใช้งาน**: แปลงโน้ตยุ่งเหยิงให้เป็นรายงานประจำวันที่เป็นระเบียบ
> - **หมายเหตุ**: ใช้ก่อนการประชุมสรุปงานตอนเช้า

## แก้ไข Prompt

ในมุมมองคอลเลกชันของฉัน คลิกปุ่มแก้ไข (ดินสอ) บนการ์ด prompt ที่คุณสร้างเพื่อเปิดหน้าต่าง "แก้ไขพรอมต์" คุณสามารถ:

- แก้ไขชื่อ เนื้อหา หน้าที่/การใช้งาน และหมายเหตุ
- สลับสถานะการแบ่งปัน (สาธารณะ/ส่วนตัว)
- คลิก "บันทึก" เพื่ออัปเดต

![อินเทอร์เฟซแก้ไข prompt](/img/docs/user-prompts-edit.png)

## ลบ Prompt

คลิก "ลบ" ในหน้าต่างแก้ไข การลบไม่สามารถกู้คืนได้ โปรดดำเนินการด้วยความระมัดระวัง

## แบ่งปันไปยังชุมชน

เมื่อสร้างหรือแก้ไข สวิตช์ "คุณต้องการแชร์ พรอมต์ นี้ไปยังหน้าสาธารณะหรือไม่?" ที่ด้านล่างจะควบคุมการมองเห็น:

- **เปิด (ค่าเริ่มต้น)**: Prompt จะปรากฏในหน้า [Prompt ชุมชน](./community) ผู้ใช้คนอื่นสามารถมองเห็นและรวบรวมได้
- **ปิด**: Prompt ส่วนตัว มองเห็นได้เฉพาะตัวคุณเอง

สลับสถานะการแบ่งปันได้ตลอดเวลา

## ส่งออก Prompt

1. ไปที่ "บัญชีของฉัน" และหาส่วน "จัดการข้อมูล → ส่งออก พรอมต์"
2. คลิกปุ่ม "ส่งออกข้อมูล"
3. ระบบจะสร้างไฟล์ JSON และดาวน์โหลดให้อัตโนมัติ

เนื้อหาที่ส่งออกประกอบด้วย:

- ชื่อ เนื้อหา หน้าที่/การใช้งาน และหมายเหตุของ prompt
- สถานะการแบ่งปัน
- คอลเลกชัน การจัดเรียงคอลเลกชัน และแท็กที่กำหนดเองของคุณ

แนะนำให้ส่งออกข้อมูลสำรองเป็นประจำเพื่อป้องกันการสูญหาย

## นำเข้า Prompt

นำเข้า prompt และคอลเลกชันจากไฟล์ JSON:

1. ไปที่ "บัญชีของฉัน" และหาส่วน "จัดการข้อมูล → นำเข้า พรอมต์"
2. คลิกปุ่ม "นำเข้าข้อมูล"
3. เลือกไฟล์ JSON
4. ระบบจะรวมข้อมูลโดยอัตโนมัติ (ตรวจสอบความซ้ำตามชื่อ; prompt ที่ ID เป็นของบัญชีอื่นจะถูกตั้งเป็นส่วนตัวอัตโนมัติ)

### การทำงานร่วมกันเป็นทีม

ขั้นตอนแนะนำสำหรับการแบ่งปัน prompt ภายในทีม:

1. **กรองและแบ่งปัน**: หลังส่งออก ลบรายการคอลเลกชันหรือกรองเฉพาะ prompt ที่ต้องการแบ่งปันด้วยตนเอง แล้วส่งไฟล์ให้สมาชิกในทีมเพื่อนำเข้า
2. **ปกป้องความเป็นส่วนตัว**: prompt ที่นำเข้าจากคนอื่น (ID ไม่ใช่ของบัญชีปัจจุบัน) จะถูกตั้งเป็น **ส่วนตัว** อัตโนมัติ ข้อมูลของสมาชิกแต่ละคนจะแยกจากกัน
