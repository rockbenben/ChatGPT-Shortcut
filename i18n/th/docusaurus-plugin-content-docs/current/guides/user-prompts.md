# Prompt ที่กำหนดเอง

หลังเข้าสู่ระบบ คุณสามารถสร้าง prompt ของตัวเอง บันทึกไว้ในบัญชีเพื่อเรียกใช้ได้ทุกเมื่อ Prompt ที่กำหนดเองจะแสดงในมุมมอง "คอลเลกชันของฉัน"

## สร้าง Prompt

1. คลิกปุ่ม "เพิ่ม" ที่มุมขวาบน
2. ในหน้าต่าง "สร้าง prompt" ที่ปรากฏ ให้กรอกฟอร์ม:
   - **ชื่อ prompt** (จำเป็น): ตั้งชื่อให้ prompt
   - **เนื้อหา prompt** (จำเป็น): เนื้อความของ prompt
   - **บทบาท/การใช้งาน** (ไม่บังคับ): อธิบายสั้น ๆ ว่า prompt นี้ทำอะไรได้
   - **หมายเหตุ** (ไม่บังคับ): แหล่งที่มา เวอร์ชันภาษาอื่น หรือคำอธิบายเพิ่มเติม
3. สวิตช์ "ยินดีที่จะแบ่งปัน prompt นี้ไปยังหน้าสาธารณะหรือไม่?" ที่ด้านล่างเปิดอยู่เป็นค่าเริ่มต้น หากปิดจะมองเห็นได้เฉพาะคุณคนเดียว
4. คลิก "สร้าง prompt" เพื่อส่ง

![ป๊อปอัพสร้าง prompt](/img/docs/user-prompts-create.png)

> 💡 **ตัวอย่างการกรอก** — prompt "ผู้ช่วยสรุปการประชุม":
> - **ชื่อ prompt**: ผู้ช่วยสรุปการประชุม
> - **เนื้อหา prompt**: คุณคือผู้ช่วยมืออาชีพในการสรุปการประชุม โปรดสรุปบันทึกการประชุมต่อไปนี้ให้กระชับ โดยแบ่งเป็น 3 ส่วน ได้แก่ ประเด็นหลักที่หารือ ข้อตัดสินใจสำคัญ และรายการสิ่งที่ต้องดำเนินการต่อ (ระบุผู้รับผิดชอบและกำหนดส่งหากมี): [บันทึกการประชุม]
> - **บทบาท/การใช้งาน**: เปลี่ยนบันทึกการประชุมแบบดิบให้เป็นสรุปที่นำไปใช้งานต่อได้ทันที
> - **หมายเหตุ**: ใช้หลังประชุมทีมหรือประชุมกับลูกค้า

## แก้ไข Prompt

ในมุมมองคอลเลกชันของฉัน คลิกการ์ด prompt ที่คุณสร้างขึ้นเพื่อเปิดหน้าต่าง "แก้ไข prompt" คุณสามารถ:

- แก้ไขชื่อ เนื้อหา บทบาท หมายเหตุ
- สลับสถานะการแบ่งปัน (สาธารณะ/ส่วนตัว)
- คลิก "บันทึก" เพื่ออัปเดต

![อินเทอร์เฟซแก้ไข prompt](/img/docs/user-prompts-edit.png)

## ลบ Prompt

คลิก "ลบ" ในหน้าต่างแก้ไข การลบไม่สามารถกู้คืนได้ โปรดดำเนินการด้วยความระมัดระวัง

## แบ่งปันไปยังชุมชน

เมื่อสร้างหรือแก้ไข สวิตช์ "ยินดีที่จะแบ่งปัน prompt นี้ไปยังหน้าสาธารณะหรือไม่?" ที่ด้านล่างจะควบคุมการมองเห็น:

- **เปิด (ค่าเริ่มต้น)**: Prompt จะแสดงในหน้า [Prompt ชุมชน](./community) ผู้ใช้คนอื่นมองเห็นและรวบรวมได้
- **ปิด**: Prompt ส่วนตัว มองเห็นได้เฉพาะตัวคุณเอง

สลับสถานะการแบ่งปันได้ตลอดเวลา

## ส่งออกข้อมูลสำรอง

1. เข้าไปที่ "บัญชีของฉัน" → หา "จัดการข้อมูล → ส่งออก Prompt"
2. คลิกปุ่ม "ส่งออกข้อมูล"
3. ระบบจะสร้างไฟล์ JSON และดาวน์โหลดอัตโนมัติ

เนื้อหาที่ส่งออกประกอบด้วย:

- ชื่อ เนื้อหา บทบาท/การใช้งาน หมายเหตุของ prompt
- เวลาที่สร้างและอัปเดต
- สถานะการแบ่งปัน

แนะนำให้ส่งออกข้อมูลสำรองเป็นประจำเพื่อป้องกันการสูญหาย

## นำเข้า Prompt

นำเข้า prompt และคอลเลกชันจากไฟล์ JSON:

1. เข้าไปที่ "บัญชีของฉัน" → หา "จัดการข้อมูล → นำเข้า Prompt"
2. คลิกปุ่ม "นำเข้าข้อมูล"
3. เลือกไฟล์ JSON
4. ระบบจะรวมข้อมูลโดยอัตโนมัติ (ตรวจสอบความซ้ำตามหัวข้อ; prompt ที่มี ID ของคนอื่นจะถูกตั้งเป็นส่วนตัวอัตโนมัติ)

### การทำงานร่วมกันเป็นทีม

ขั้นตอนแนะนำสำหรับการแบ่งปัน prompt ภายในทีม:

1. **กรองและแบ่งปัน**: หลังส่งออก ลบรายการคอลเลกชันหรือกรอง prompt ที่ต้องการแบ่งปันด้วยตนเอง ส่งไฟล์ให้สมาชิกในทีมเพื่อนำเข้า
2. **ปกป้องความเป็นส่วนตัว**: prompt ที่นำเข้าจากคนอื่น (ID ไม่ใช่ของบัญชีปัจจุบัน) จะถูกตั้งเป็น **ส่วนตัว** อัตโนมัติ ข้อมูลของสมาชิกแต่ละคนจะไม่ปะปนกัน
