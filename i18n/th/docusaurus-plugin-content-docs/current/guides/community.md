# Prompt ชุมชน

ส่วน Prompt ชุมชนแสดง prompt ที่กำหนดเองที่แบ่งปันโดยผู้ใช้ คุณสามารถหาแรงบันดาลใจที่นี่หรือแบ่งปันความคิดสร้างสรรค์ของคุณเอง

![](/img/docs/community-page.png)

## การจัดเรียงและการกรอง

หน้าชุมชนมีหลายวิธีการกรอง:

- **เวลาเผยแพร่**: ดู prompt ที่แบ่งปันล่าสุด
- **การจัดเรียงตามคะแนนโหวต**: ดู prompt ที่มีคะแนนโหวตมากที่สุด
- **ค้นหาคำสำคัญ**: ค้นหา prompt ในหัวข้อเฉพาะ

## กลไกการโหวต

ตัดสินการเปิดเผยของ prompt โดยการโหวตขึ้น/ลง:

- **โหวตขึ้น**: ช่วยให้เนื้อหาคุณภาพสูงได้รับการเปิดเผยมากขึ้น
- **โหวตลง**: หลังจากจำนวนโหวตลงถึงจำนวนหนึ่ง prompt จะถูกซ่อน

**ลิงก์คอลเลกชัน**: เมื่อรวบรวม prompt ชุมชน ระบบจะโหวตขึ้นให้อัตโนมัติ

![สถานะปุ่มโหวต](/img/docs/community-vote.png)

> การโหวตเปิดเฉพาะผู้ใช้ที่เข้าสู่ระบบ และคะแนนโหวตของแต่ละคนสำหรับ prompt เดียวกันอ้างอิงจากคะแนนล่าสุด

## Prompt ส่วนตัว

ความคิดสร้างสรรค์ที่คุณไม่ต้องการเปิดเผยสามารถตั้งเป็นส่วนตัว:

- เมื่อสร้างหรือแก้ไข prompt ให้ยกเลิกการเลือกตัวเลือก "แบ่งปันไปยังชุมชน"
- Prompt ส่วนตัวมองเห็นได้เฉพาะตัวคุณเอง
- คุณสามารถสลับระหว่างสถานะสาธารณะ/ส่วนตัวได้ตลอดเวลา
