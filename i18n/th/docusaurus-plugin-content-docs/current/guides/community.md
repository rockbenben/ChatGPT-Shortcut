# Prompt ชุมชน

Prompt ชุมชนคือ prompt ที่ผู้ใช้แบ่งปันกัน คุณสามารถหาแรงบันดาลใจและแชร์ไอเดียของตัวเองได้ที่นี่ **ระบบโหวต** จะช่วยดัน prompt ที่คนใช้งานได้ดีให้คนอื่นเห็นมากขึ้น ส่วน prompt คุณภาพต่ำก็จะค่อย ๆ จมหายไป

![](/img/docs/community-page.png)

## การจัดเรียงและการกรอง

- **เวลาเผยแพร่**: ดู prompt ที่แบ่งปันล่าสุด
- **จัดเรียงตามคะแนนโหวต**: ดู prompt ที่ได้รับการโหวตขึ้นมากที่สุด
- **ค้นหาคำสำคัญ**: ค้นหาหัวข้อเฉพาะ

## กลไกการโหวต

โหวตขึ้น/ลงเป็นตัวกำหนดว่า prompt จะถูกมองเห็นมากแค่ไหน:

- **โหวตขึ้น**: ช่วยดัน prompt คุณภาพให้ปรากฏเด่นขึ้น
- **โหวตลง**: เมื่อคะแนนโหวตลงถึงเกณฑ์ที่กำหนด prompt จะถูกซ่อนจากรายการ

**เชื่อมโยงกับคอลเลกชัน**: การเก็บ prompt ชุมชนจะนับเป็นการโหวตขึ้นให้โดยอัตโนมัติ

![สถานะปุ่มโหวต](/img/docs/community-vote.png)

> เฉพาะผู้ใช้ที่เข้าสู่ระบบเท่านั้นจึงจะโหวตได้ และระบบจะยึดผลโหวตล่าสุดของคุณต่อหนึ่ง prompt

## Prompt ส่วนตัว

ความคิดสร้างสรรค์ที่ไม่ต้องการเปิดเผยสามารถตั้งเป็นส่วนตัว:

- เมื่อสร้างหรือแก้ไข ให้ยกเลิก "แบ่งปันไปยังชุมชน"
- Prompt ส่วนตัวมองเห็นได้เฉพาะตัวคุณเอง
- สลับสาธารณะ/ส่วนตัวได้ตลอดเวลา
