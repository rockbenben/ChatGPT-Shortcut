# Prompt ชุมชน

Prompt ชุมชนคือ prompt ที่ผู้ใช้คนอื่นแบ่งปัน — เรียกดูเพื่อหาแรงบันดาลใจ แล้วนำสิ่งที่คุณสร้างมาแบ่งปันด้วยกัน **ระบบโหวต** จะดูแลให้เนื้อหาดีอยู่บนสุด: prompt ที่มีประโยชน์จะปีนขึ้น ส่วน prompt ที่อ่อนแอก็จะค่อย ๆ จมหายไป

![](/img/docs/community-page.png)

## การจัดเรียงและการกรอง

- **ใหม่ล่าสุด**: ดู prompt ที่แบ่งปันล่าสุด (ค่าเริ่มต้น)
- **ถูกโหวตมากที่สุด**: ดู prompt ที่ได้รับการโหวตขึ้นมากที่สุด
- **ค้นหาคำสำคัญ**: ค้นหาหัวข้อเฉพาะ

## กลไกการโหวต

โหวตขึ้น/ลงเป็นตัวกำหนดว่า prompt จะถูกมองเห็นมากแค่ไหน:

- **โหวตขึ้น**: ดัน prompt คุณภาพให้ปรากฏต่อผู้คนมากขึ้น
- **โหวตลง**: เมื่อ prompt ได้รับโหวตลงถึงเกณฑ์ที่กำหนด จะถูกซ่อนจากรายการ

**เก็บ = โหวตขึ้น**: การเพิ่ม prompt ชุมชนเข้าคอลเลกชันของคุณจะนับเป็นการโหวตขึ้นให้โดยอัตโนมัติ

![สถานะปุ่มโหวต](/img/docs/community-vote.png)

> การโหวตเปิดให้เฉพาะผู้ใช้ที่เข้าสู่ระบบ และระบบจะยึดผลโหวตล่าสุดของคุณต่อหนึ่ง prompt

## Prompt ส่วนตัว

ไอเดียที่ไม่ต้องการเปิดเผยสามารถตั้งเป็นส่วนตัวได้:

- ปิด "แบ่งปันไปยังชุมชน" ขณะสร้างหรือแก้ไข
- Prompt ส่วนตัวมองเห็นได้เฉพาะตัวคุณเอง
- สลับระหว่างสาธารณะและส่วนตัวได้ตลอดเวลา

## การสนทนา

เปิดหน้ารายละเอียดของ prompt ชุมชนใด ๆ นอกจากเนื้อหาครบถ้วนแล้ว หลังเข้าสู่ระบบยังสามารถแสดงความคิดเห็นและร่วมสนทนาได้ (รองรับอิโมจิและ GIF)
