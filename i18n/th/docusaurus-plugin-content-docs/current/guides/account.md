# จัดการบัญชี

AiShort ใช้งานได้แม้ไม่ได้เข้าสู่ระบบ แต่เมื่อ **เข้าสู่ระบบ** แล้วคุณจะรวบรวม prompt ที่ชอบ สร้าง prompt ของตัวเอง แบ่งปันในชุมชน และซิงค์ข้ามอุปกรณ์ได้ หน้านี้ครอบคลุมการเข้าสู่ระบบ การลงทะเบียน รหัสผ่าน และการจัดการข้อมูล

## วิธีการเข้าสู่ระบบ

หน้าต่างเข้าสู่ระบบมีแท็บ "**เข้าสู่ระบบด้วยรหัสผ่าน**" และ "**เข้าสู่ระบบแบบไม่ใช้รหัสผ่าน**" พร้อมการเข้าสู่ระบบด้วย Google แบบคลิกเดียว

### เข้าสู่ระบบด้วยรหัสผ่าน

เข้าสู่ระบบด้วยชื่อผู้ใช้หรืออีเมลที่ลงทะเบียนพร้อมรหัสผ่าน

### เข้าสู่ระบบแบบไม่ใช้รหัสผ่าน

1. สลับไปที่แท็บ "เข้าสู่ระบบแบบไม่ใช้รหัสผ่าน" และป้อนอีเมลที่ลงทะเบียน
2. คลิก "**รับลิงก์เข้าสู่ระบบ**" — ลิงก์เข้าสู่ระบบจะถูกส่งไปยังกล่องจดหมายของคุณ
3. เปิดลิงก์เพื่อเข้าสู่ระบบ

ไม่ต้องจำรหัสผ่าน — เหมาะมากหากคุณเข้าสู่ระบบไม่บ่อยนัก

### เข้าสู่ระบบด้วย Google

คลิกปุ่ม Google ในหน้าต่างเข้าสู่ระบบเพื่อเข้าสู่ระบบแบบคลิกเดียว ครั้งแรกที่ใช้งาน ระบบจะสร้างบัญชีให้โดยอัตโนมัติ ไม่ต้องลงทะเบียนแยก

![](/img/docs/account-login.png)

## ลงทะเบียนบัญชี

1. ในหน้าต่างเข้าสู่ระบบ คลิก "**ลงทะเบียนเลย**" เพื่อสลับไปหน้าลงทะเบียน
2. กรอกอีเมล ชื่อผู้ใช้ และรหัสผ่าน พร้อมยอมรับนโยบายความเป็นส่วนตัวและข้อกำหนดการให้บริการ
3. ระบบจะเข้าสู่ระบบให้โดยอัตโนมัติหลังจากส่ง

![](/img/docs/account-register.png)

## บัญชีของฉัน

หลังเข้าสู่ระบบ เปิดหน้า "**บัญชีของฉัน**" เพื่อจัดการโปรไฟล์ รหัสผ่าน ระดับ และข้อมูลของคุณ

![หน้าบัญชีของฉัน](/img/docs/account-profile.png)

### เปลี่ยนชื่อผู้ใช้

คลิกไอคอนแก้ไขถัดจากชื่อผู้ใช้ แล้วบันทึก

### เปลี่ยนรหัสผ่าน

1. ป้อนรหัสผ่านปัจจุบัน
2. ป้อนรหัสผ่านใหม่และยืนยัน
3. คลิก "เปลี่ยนรหัสผ่าน" เพื่อส่ง

ลืมรหัสผ่าน? ในหน้าต่างเข้าสู่ระบบ คลิก "ลืมรหัสผ่าน?" ป้อนอีเมลที่ลงทะเบียน คลิก "ส่งอีเมลรีเซ็ต" แล้วทำตามลิงก์

### ระดับของฉัน

หน้าบัญชีจะแสดง **ระดับ (L0–L9)** และความคืบหน้าไปยังระดับถัดไป — ระดับของคุณขึ้นอยู่กับ **จำนวน prompt ที่คุณแบ่งปันสู่ชุมชน**

## การจัดการข้อมูล

ส่วน "บัญชีของฉัน → การจัดการข้อมูล" มีสามตัวเลือก:

- **ล้างแคช** — รีเฟรชข้อมูลที่แคชไว้ในเครื่อง ใช้เมื่อข้อมูลไม่ซิงค์หรือแสดงผลผิดปกติ คอลเลกชันและ prompt ที่กำหนดเองของคุณจะไม่หายไป เป็นเพียงการบังคับให้ซิงค์ข้อมูลใหม่
- **ส่งออก พรอมต์** — ส่งออกไฟล์ JSON เพื่อสำรองข้อมูล (ดู [Prompt ที่กำหนดเอง](./user-prompts#ส่งออก-prompt))
- **นำเข้า พรอมต์** — นำเข้า prompt และคอลเลกชันจากไฟล์ JSON
