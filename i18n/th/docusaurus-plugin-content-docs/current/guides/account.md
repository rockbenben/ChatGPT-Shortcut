# จัดการบัญชี

AiShort ใช้งานได้แม้ไม่ได้เข้าสู่ระบบ แต่เมื่อ **เข้าสู่ระบบ** แล้วคุณจะรวบรวม prompt ที่ชอบ สร้าง prompt ของตัวเอง แบ่งปันในชุมชน และซิงค์ข้ามอุปกรณ์ได้ หัวข้อด้านล่างจะแนะนำวิธีเข้าสู่ระบบ ลงทะเบียน รวมถึงการจัดการรหัสผ่านและแคช

## วิธีการเข้าสู่ระบบ

### เข้าสู่ระบบด้วยรหัสผ่าน

ใช้ชื่อผู้ใช้หรืออีเมลที่ลงทะเบียน + รหัสผ่านในการเข้าสู่ระบบ

ลืมรหัสผ่าน: คลิก "ลืมรหัสผ่าน" บนหน้าเข้าสู่ระบบ ป้อนอีเมลที่ลงทะเบียนแล้วระบบจะส่งลิงก์รีเซ็ตให้

### เข้าสู่ระบบด้วย Google

คลิกไอคอน Google บนหน้าเข้าสู่ระบบเพื่อเข้าสู่ระบบด้วยคลิกเดียว ครั้งแรกจะสร้างบัญชีโดยอัตโนมัติ ไม่ต้องลงทะเบียนแยก

![](/img/docs/account-login.png)

### เข้าสู่ระบบแบบไม่มีรหัสผ่าน

1. ป้อนอีเมลที่ลงทะเบียนบนหน้าเข้าสู่ระบบ
2. คลิก "ส่งลิงก์เข้าสู่ระบบ"
3. เปิดลิงก์ในอีเมลเพื่อเข้าสู่ระบบ

ไม่ต้องจำรหัสผ่าน เหมาะกับสถานการณ์ที่ไม่ได้เข้าสู่ระบบบ่อย

## ลงทะเบียนบัญชี

1. คลิก "ลงทะเบียน" บนหน้าเข้าสู่ระบบ
2. กรอกชื่อผู้ใช้ อีเมล และรหัสผ่าน
3. เข้าสู่ระบบโดยอัตโนมัติหลังจากส่ง

![](/img/docs/account-register.png)

## เปลี่ยนรหัสผ่าน

หลังเข้าสู่ระบบ ไปที่ "บัญชีของฉัน" เพื่อเปลี่ยนรหัสผ่าน:

![หน้าบัญชีของฉัน](/img/docs/account-profile.png)

1. ป้อนรหัสผ่านปัจจุบัน
2. ป้อนรหัสผ่านใหม่และยืนยัน
3. คลิก "เปลี่ยนรหัสผ่าน" เพื่อส่ง

## ล้างแคช

เมื่อข้อมูลไม่ซิงค์หรือแสดงผลผิดปกติ:

1. เข้าไปที่ "บัญชีของฉัน" → หา "จัดการข้อมูล → ล้างแคช"
2. คลิกปุ่ม "ล้างแคช" ยืนยันแล้วจะโหลดข้อมูลใหม่ทั้งหมดอัตโนมัติ

คอลเลกชันและ prompt ที่กำหนดเองจะไม่หายไปไหน เป็นเพียงการบังคับให้ซิงค์ข้อมูลใหม่เท่านั้น
