# คำถามที่พบบ่อย

## ทำไมต้องใช้ prompt ภาษาอังกฤษ?

โมเดล AI โดยทั่วไปเข้าใจภาษาอังกฤษได้ดีกว่า และการใช้ prompt ภาษาอังกฤษสามารถให้ผลลัพธ์ที่เสถียรกว่า

แม้ว่าการใช้ prompt ภาษาท้องถิ่นจะได้ผลลัพธ์ที่ดี แต่การป้อน prompt ภาษาท้องถิ่นเดียวกันซ้ำๆ อาจทำให้ผลลัพธ์แตกต่างกันอย่างมาก ดังนั้นแนะนำให้ใช้ prompt ภาษาอังกฤษในสถานการณ์สำคัญ

**ระบุการตอบกลับเป็นภาษาไทย**: เพิ่ม `respond in Thai` ที่ท้าย prompt เพื่อรับการตอบกลับเป็นภาษาไทย

## ต้องป้อน prompt ทุกครั้งหรือไม่?

**การใช้ API**: คุณสามารถตั้ง prompt เป็น "system prompt" ซึ่งจะถูกนำไปใช้กับการสนทนาครั้งต่อไปโดยอัตโนมัติ

**เวอร์ชันเว็บ**: หากคุณยังไม่ได้เปลี่ยนหัวข้อ ChatGPT จะจำการตั้งค่า prompt ของการสนทนาปัจจุบัน เมื่อการตอบกลับเริ่มเบี่ยงเบนจากที่คาดหวัง หมายความว่า AI "ลืม" prompt และคุณต้องป้อนใหม่

**เคล็ดลับ**: บันทึกการสนทนาที่ใช้บ่อยเป็น bookmark สำหรับการเข้าถึงอย่างรวดเร็วในภายหลัง

## AI ส่งข้อมูลผิดพลาดจะทำอย่างไร?

AI บางครั้ง "ภาพหลอน" ส่งข้อมูลที่ดูเหมือนสมเหตุสมผลแต่จริงๆ แล้วไม่ถูกต้อง คำแนะนำในการใช้:

1. **ตรวจสอบข้อมูลสำคัญ**: โดยเฉพาะข้อมูล การอ้างอิง โค้ด ฯลฯ
2. **การเพิ่มประสิทธิภาพหลายรอบ**: ให้ AI ตรวจสอบและเพิ่มประสิทธิภาพคำตอบอีกครั้ง
3. **การตรวจสอบข้าม**: ใช้ prompt หรือโมเดลที่แตกต่างกันเพื่อยืนยันข้อสรุปที่สำคัญ

Prompt ที่เหมาะสมสามารถช่วยคุณลดการเกิดภาพหลอน AI

## จะสำรองข้อมูล prompt ของฉันอย่างไร?

1. เข้าศูนย์ผู้ใช้
2. คลิกปุ่ม "ส่งออก Prompt"
3. ระบบจะสร้างไฟล์ JSON สำหรับดาวน์โหลดโดยอัตโนมัติ

แนะนำให้สำรองข้อมูลเป็นประจำเพื่อป้องกันการสูญหายของข้อมูล

## ผลลัพธ์ prompt ไม่เป็นที่น่าพอใจ?

1. **ลองเพิ่มประสิทธิภาพ**: ให้ AI เพิ่มประสิทธิภาพคำตอบอีกครั้ง
2. **เปลี่ยนมุมมอง**: ใช้ prompt ที่แตกต่างกันเพื่อแสดงความต้องการเดียวกัน
3. **ความคิดเห็นและการแบ่งปัน**: หากคุณพบปัญหาหรือมี prompt ที่ดี ยินดีต้อนรับที่จะแจ้งเราผ่าน [ความคิดเห็น](/feedback) หรือแบ่งปันในชุมชน
