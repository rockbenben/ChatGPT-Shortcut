# เริ่มต้นใช้งาน

เมื่อสนทนากับ AI prompt ที่ดีสามารถช่วยให้คุณได้รับคำตอบคุณภาพสูงได้เร็วขึ้น AiShort ได้คัดสรร prompt ที่ใช้งานได้จริงจำนวนมากสำหรับคุณ พร้อมคัดลอกและใช้งาน

## การใช้งานพื้นฐาน

1. ค้นหาหรือเลือก prompt ที่คุณต้องการบนหน้าแรก
2. คลิกการ์ดเพื่อดูรายละเอียด หรือคลิกปุ่ม "คัดลอก" โดยตรง
3. เปิดโมเดล AI ใดก็ได้แล้ววาง prompt
4. ทำตามคำแนะนำของ prompt เพื่อเพิ่มคำถามเฉพาะของคุณ
