# เริ่มต้นใช้งาน

Prompt ที่ดีเป็นตัวกำหนดคุณภาพของผลลัพธ์ AI AiShort รวบรวม prompt คุณภาพไว้ให้คุณ คัดลอกแล้ววางใช้งานได้ทันที

> 📋 **ก่อนเริ่มต้นคุณต้องมี**: เบราว์เซอร์ + เครื่องมือ AI ใดก็ได้ — หน้าแชท (ChatGPT, Gemini, Claude, Doubao ฯลฯ), เครื่องมือเขียนโค้ดอย่าง Cursor หรือการเรียก API ของคุณเอง ตัว AiShort เวอร์ชันเว็บไม่ต้องลงทะเบียน

## การใช้งานพื้นฐาน

1. ค้นหาหรือเรียกดู prompt ที่สนใจบนหน้าแรก
2. เมื่อเจอ prompt ที่เหมาะสม คลิกปุ่ม "คัดลอก" (หรือคลิกการ์ดเพื่อดูเนื้อหาทั้งหมดก่อน)
3. วางลงในเครื่องมือ AI ใดก็ได้ — หน้าแชท, เครื่องมือเขียนโค้ดอย่าง Cursor หรือการเรียก API ของคุณเอง
4. ตามที่ **ตัวยึดตำแหน่ง** ใน prompt ระบุไว้ (เช่น `[เนื้อหา]`, `[หัวข้อ]`) เติมความต้องการของคุณ แล้วส่งให้ AI

![](/img/docs/how-to-use-aishort.gif)

> 💡 **เคล็ดลับ**: Prompt ส่วนใหญ่ใช้วงเล็บเหลี่ยม `[...]` เพื่อทำเครื่องหมายส่วนที่คุณต้องเติม เพียงแทนที่ข้อความตัวยึดในวงเล็บด้วยเนื้อหาเฉพาะของคุณ
