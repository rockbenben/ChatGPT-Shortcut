# คอลเลกชันของฉัน

หลังเข้าสู่ระบบแล้ว มุมมอง **คอลเลกชันของฉัน** จะช่วยให้คุณจัดระเบียบ prompt ที่ชอบไว้ใช้งานสะดวก **คล้ายกับการจัดบุ๊กมาร์กในเบราว์เซอร์** ทั้งลากวางจัดลำดับ ตั้งแท็กเอง และกรองตามแท็กเพื่อหยิบใช้ได้อย่างรวดเร็ว

:::tip เวอร์ชันออนไลน์และออฟไลน์
เวอร์ชันออนไลน์เก็บข้อมูลคอลเลกชันไว้บนเซิร์ฟเวอร์ (ต้องเข้าสู่ระบบ) ส่วน [เวอร์ชันออฟไลน์](./offline) เก็บข้อมูลคอลเลกชันไว้ในเบราว์เซอร์ (localStorage ไม่ต้องเข้าสู่ระบบ) ฟังก์ชันการใช้งานเหมือนกัน
:::

## การสลับมุมมอง

ปุ่มสลับมุมมองที่ด้านบนของหน้าแรกมีสองโหมด:

- **คอลเลกชันของฉัน**: แสดงคอลเลกชันของคุณ + prompt ที่กำหนดเอง
- **คลัง Prompt**: เรียกดูคลัง prompt สาธารณะ

ระบบจะจำมุมมองที่คุณเลือกไว้ล่าสุด และเปิดมุมมองเดิมให้อัตโนมัติเมื่อคุณกลับเข้ามาครั้งถัดไป

![มุมมองคอลเลกชันของฉัน](/img/docs/my-collection-view.gif)

## เพิ่มในคอลเลกชัน

คลิกปุ่ม "รวบรวม" บนการ์ดเพื่อเพิ่มในคอลเลกชันของฉัน

การรวบรวม prompt ชุมชนจะโหวตขึ้นให้อัตโนมัติ ช่วยให้เนื้อหาคุณภาพได้รับการมองเห็นมากขึ้น

## ลบจากคอลเลกชัน

ในมุมมองคอลเลกชันของฉัน คลิกไอคอนหัวใจบนการ์ดเพื่อยกเลิกการรวบรวม

## แท็กที่กำหนดเอง

เพิ่มแท็กให้ prompt ที่รวบรวม เพื่อสะดวกในการจัดหมวดหมู่:

### เพิ่มแท็ก

1. คลิกไอคอนแท็กบนแถบควบคุม
2. เลือกแท็กที่มีอยู่ หรือคลิก "+" เพื่อสร้างใหม่
3. แท็กใหม่กำหนดชื่อและสีได้

### จัดการแท็ก

คลิก "จัดการแท็ก" เพื่อเข้าสู่อินเทอร์เฟซการจัดการ:

- แก้ไขชื่อแท็ก
- เปลี่ยนสีแท็ก (มี 11 สีที่ตั้งไว้)
- ลบแท็ก

![ป๊อปอัพการจัดการแท็ก](/img/docs/my-collection-tag.png)

### กรองตามแท็ก

แถบกรองในมุมมองคอลเลกชันของฉัน สามารถกรองตามแท็กที่กำหนดเองได้

## การลากวางจัดเรียง

มุมมองคอลเลกชันของฉันรองรับการลากวางเพื่อปรับลำดับการ์ด:

1. ลากที่จับด้านซ้ายของการ์ด
2. ย้ายไปยังตำแหน่งเป้าหมายแล้วปล่อย
3. ลำดับใหม่จะถูกบันทึกอัตโนมัติ (เวอร์ชันออนไลน์เก็บที่เซิร์ฟเวอร์ เวอร์ชันออฟไลน์เก็บที่เบราว์เซอร์)

การจัดเรียงมีเฉพาะในมุมมองคอลเลกชันของฉันเท่านั้น

![การลากวางจัดเรียง](/img/docs/my-collection-drag.gif)
