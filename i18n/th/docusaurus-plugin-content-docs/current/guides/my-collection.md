# คอลเลกชันของฉัน

หลังเข้าสู่ระบบ คุณสามารถรวบรวม prompt ที่ชื่นชอบและจัดการส่วนตัว มุมมองคอลเลกชันของฉันมีฟังก์ชันการจัดการที่หลากหลายเพื่อช่วยคุณจัดระเบียบและใช้ prompt อย่างมีประสิทธิภาพ

## การสลับมุมมอง

ด้านบนของหน้าแรกมีปุ่มสลับมุมมอง รองรับสองโหมด:

- **คอลเลกชันของฉัน**: แสดง prompt ที่รวบรวมและ prompt ที่กำหนดเองของคุณ
- **คลัง Prompt**: เรียกดูคลัง prompt สาธารณะเพื่อค้นพบเนื้อหาใหม่

เบราว์เซอร์จะจดจำการเลือกมุมมองของคุณโดยอัตโนมัติและสลับไปยังมุมมองล่าสุดเมื่อเข้าชมครั้งต่อไป

![มุมมองคอลเลกชันของฉัน](/img/docs/my-collection-view.gif)

## เพิ่มในคอลเลกชัน

คลิกปุ่ม "รวบรวม" บนการ์ด prompt เพื่อเพิ่มในคอลเลกชันของฉัน

เมื่อรวบรวม prompt ชุมชน ระบบจะโหวตขึ้นให้อัตโนมัติ ช่วยให้เนื้อหาคุณภาพได้รับการเปิดเผยมากขึ้น

## ลบจากคอลเลกชัน

ในมุมมองคอลเลกชันของฉัน คลิกไอคอนหัวใจบนการ์ดเพื่อยกเลิกการรวบรวม

## แท็กที่กำหนดเอง

เพิ่มแท็กส่วนตัวให้ prompt ที่รวบรวมเพื่อการจัดการการจำแนกที่สะดวก:

### เพิ่มแท็ก

1. คลิกไอคอนแท็กบนแถบควบคุม
2. เลือกแท็กที่มีอยู่หรือคลิก "+" เพื่อสร้างแท็กใหม่
3. แท็กใหม่สามารถมีชื่อและสีที่กำหนดเอง

### จัดการแท็ก

คลิก "จัดการแท็ก" เพื่อเข้าสู่อินเทอร์เฟซการจัดการแท็ก:

- แก้ไขชื่อแท็ก
- เปลี่ยนสีแท็ก (มี 11 สีที่ตั้งไว้ล่วงหน้า)
- ลบแท็กที่ไม่ต้องการ

![ป๊อปอัพการจัดการแท็ก](/img/docs/my-collection-tag.png)

### กรองตามแท็ก

ในแถบกรองของมุมมองคอลเลกชันของฉัน คุณสามารถกรองเนื้อหาตามแท็กที่กำหนดเอง

## การลากวางจัดเรียง

รองรับการปรับลำดับการ์ดด้วยลากวางในมุมมองคอลเลกชันของฉัน:

1. ลากที่จับด้านซ้ายของการ์ด
2. ย้ายไปยังตำแหน่งเป้าหมายแล้วปล่อย
3. ลำดับใหม่จะถูกบันทึกไปยังเซิร์ฟเวอร์โดยอัตโนมัติ

ฟังก์ชันการจัดเรียงมีเฉพาะในมุมมองคอลเลกชันของฉัน

![การลากวางจัดเรียง](/img/docs/my-collection-drag.gif)
