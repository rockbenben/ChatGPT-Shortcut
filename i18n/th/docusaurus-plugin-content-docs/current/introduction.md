# แนะนำ AI Short

AI Short (ChatGPT Shortcut) คือไลบรารี prompt AI ที่จัดหมวดหมู่ตามสถานการณ์การใช้งาน คัดลอกแล้วใช้ได้ทันที

> **"prompt" คืออะไร?** Prompt คือคำสั่งที่คุณป้อนให้ AI ซึ่งเป็นตัวกำหนดคุณภาพของผลลัพธ์ Prompt ที่ดีทำให้ AI ตอบได้อย่างมืออาชีพและแม่นยำ ขณะที่ prompt ที่กำกวมจะได้คำตอบธรรมดา ๆ เท่านั้น AiShort รวบรวม prompt คุณภาพสำหรับสถานการณ์ต่าง ๆ (การเขียน เขียนโค้ด งานสำนักงาน การเรียน ฯลฯ) ไว้ให้พร้อม คุณไม่ต้องเสียเวลาคิดเองว่าจะถามอย่างไร

## ⚡ เริ่มต้นใน 30 วินาที

1. เปิด AI Short
2. ค้นหาหรือเรียกดู prompt ที่คุณต้องการ
3. คลิก "คัดลอก" แล้ววางลงในเครื่องมือ AI ใดก็ได้ — หน้าแชทอย่าง ChatGPT, Gemini หรือ Claude, เครื่องมือเขียนโค้ดอย่าง Cursor, การเรียก API เป็นต้น

ง่ายแค่นี้เอง สำหรับวิธีใช้งานเพิ่มเติม ดูได้ที่[คู่มือเริ่มต้นใช้งาน](./guides/getting-started)

## คุณสมบัติหลัก

🚀 **Prompt ที่คัดสรร**: ครอบคลุมสถานการณ์การเขียน การเขียนโค้ด งานสำนักงาน และอื่น ๆ คัดลอกใช้งานได้ทันที

🔍 **ค้นหาด้วยแท็ก + คำสำคัญ**: กรองตามแท็กสถานการณ์ หรือค้นหาด้วยคำสำคัญโดยตรง

🌍 **18 ภาษา**: รองรับการแปลทั้งอินเทอร์เฟซและ prompt ระบุภาษาแม่ในการตอบกลับได้

📦 **พร้อมใช้งานทันที**: ไม่ต้องลงทะเบียน เปิดแล้วใช้ได้เลย

## คุณสมบัติขั้นสูง (หลังเข้าสู่ระบบ)

📂 **คอลเลกชันของฉัน**: ลากวางจัดเรียง พร้อมแท็กที่กำหนดเองเพื่อจัดหมวดหมู่

✏️ **Prompt ที่กำหนดเอง**: สร้าง แก้ไข และจัดการ prompt ของคุณเอง

🗳️ **การโต้ตอบกับชุมชน**: แบ่งปัน prompt ไปยังชุมชน และมีส่วนร่วมในการโหวต

📤 **ส่งออกข้อมูล**: ส่งออก prompt ทั้งหมดเป็นไฟล์ JSON ด้วยคลิกเดียว

🔐 **วิธีการเข้าสู่ระบบหลายแบบ**: รหัสผ่านบัญชี, Google และลิงก์อีเมลแบบไม่มีรหัสผ่าน

## เวอร์ชันออฟไลน์

[เวอร์ชันออฟไลน์](./guides/offline) ออกแบบมาสำหรับสถานการณ์ที่ไม่สามารถเข้าถึงอินเทอร์เน็ตได้ เช่น อินทราเน็ตขององค์กรหรือเครือข่ายภาครัฐ ไม่ต้องใช้เซิร์ฟเวอร์หลังบ้านหรือบัญชี เปิดแล้วใช้งานได้ทันที พร้อมฟังก์ชันหลักทั้งการเรียกดู ค้นหา รวบรวม และกำหนดเอง รูปแบบข้อมูลใช้งานร่วมกับเวอร์ชันออนไลน์ได้

## แหล่งที่มาของ Prompt

Prompt มาจากการคัดสรรจากอินเทอร์เน็ต การแบ่งปันในชุมชน และ [Awesome ChatGPT Prompts](https://github.com/f/awesome-chatgpt-prompts) อัปเดตอย่างต่อเนื่อง
