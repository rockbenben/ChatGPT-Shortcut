# วิธีใช้ส่วนขยายเบราว์เซอร์?

ChatGPT Shortcut รองรับ 13 ภาษาหลัก และภาษาส่วนขยายจะถูกตั้งค่าโดยอัตโนมัติตามสภาพแวดล้อมเบราว์เซอร์ของคุณ คุณยังสามารถเปลี่ยนภาษาด้วยตนเองในหน้าการตั้งค่า

![](/img/docs/extension-settings.png)

## โหมดการแสดงผล

ส่วนขยายรองรับสามโหมดการแสดงผล:

- **โหมดแถบด้านข้าง**: เปิดในแถบด้านข้างของเบราว์เซอร์ แสดงควบคู่กับเนื้อหาเว็บ
- **โหมดป๊อปอัพ**: แสดงเป็นป๊อปอัพเมื่อคลิกไอคอนส่วนขยาย
- **โหมดหน้าต่าง**: เปิดในหน้าต่างลอยแยกต่างหาก

## เปิดแผงด้านข้างค้างไว้

เมื่อเปิดใช้งาน แถบด้านข้างจะยังคงเปิดอยู่เมื่อสลับแท็บเบราว์เซอร์

## โหมดมืด

ส่วนขยายรองรับโหมดมืด เปิดใช้งาน "โหมดมืด" ในหน้าการตั้งค่าเพื่อสลับอินเทอร์เฟซเป็นรูปลักษณ์มืด

![](/img/docs/extension-sidebar.png)

## การเปิดใช้งานด้วยปุ่มลัด

ใช้ทางลัด `Alt+Shift+S` เพื่อเปิดใช้งานส่วนขยายโดยตรง
