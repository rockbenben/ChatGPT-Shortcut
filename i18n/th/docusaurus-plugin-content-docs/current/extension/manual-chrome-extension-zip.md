# คู่มือติดตั้งส่วนขยาย Chrome ZIP ในเครื่อง

> คู่มือนี้สมมติว่าคุณได้ดาวน์โหลดและแตกไฟล์แพ็คเกจ ZIP จาก[หน้าติดตั้ง](./README.md)เรียบร้อยแล้ว
>
> ⚠️ **ข้อผิดพลาด 2 ข้อที่พบบ่อย**:
> 1. โฟลเดอร์ที่แตกออกมาต้อง **เก็บไว้ระยะยาว** (ส่วนขยายจะอ่านจากไดเรกทอรีนี้ตอนทำงาน) ให้ย้ายไปยังตำแหน่งถาวรก่อนดำเนินการต่อ
> 2. สิ่งที่ลากเข้าไปคือ **โฟลเดอร์ที่แตกแล้วทั้งโฟลเดอร์** (ไม่ใช่ไฟล์ zip หรือไฟล์เดี่ยวในโฟลเดอร์)

## เปิดโหมดนักพัฒนา

เปิดหน้า "จัดการส่วนขยาย" ของ Chrome และเปิด "โหมดนักพัฒนา"

คัดลอกที่อยู่ต่อไปนี้และวางในแถบที่อยู่ของเบราว์เซอร์ จากนั้นกด Enter เพื่อเปิด เปิด "โหมดนักพัฒนา" ที่มุมขวาบนของหน้า

```txt
chrome://extensions
```

## ติดตั้งส่วนขยาย

ลากโฟลเดอร์ที่แตกแล้วเข้าไปในหน้าส่วนขยายเพื่อทำการติดตั้งให้เสร็จ

![](https://img.newzone.top/2024-08-12-22-27-47.png?imageMogr2/format/webp)

หลังจากติดตั้งสำเร็จ ตรวจสอบ [บทเรียนการใช้งานปลั๊กอิน](./usage)
