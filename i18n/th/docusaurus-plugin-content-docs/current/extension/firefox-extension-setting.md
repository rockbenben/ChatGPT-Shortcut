# การตั้งค่าส่วนขยาย Firefox

ก่อนใช้ส่วนขยาย Firefox แนะนำให้ทำตามสองขั้นตอนนี้เพื่อตั้งค่า

## 1. ปักหมุดส่วนขยายและเข้าถึงการตั้งค่า

ขั้นแรก เลือก "ปักหมุดส่วนขยาย ChatGPT Shortcut" (ปักหมุดไปที่แถบเครื่องมือ) บนแถบเครื่องมือ Firefox จากนั้นเข้าศูนย์ส่วนขยาย ค้นหารายการส่วนขยาย ChatGPT Shortcut และเลือก "ตัวเลือก" เข้าสู่อินเทอร์เฟซการตั้งค่าของส่วนขยาย ขั้นตอนเฉพาะแสดงด้านล่าง:

![การตั้งค่า Firefox](https://img.newzone.top/2023-12-25-05-51-47.png?imageMogr2/format/webp)

## 2. อนุญาตส่วนขยายให้ทำงาน

ประการที่สอง เพื่อให้แน่ใจว่าส่วนขยายทำงานปกติบนเว็บไซต์เช่น ChatGPT, Gemini คุณต้องคลิกขวาที่ไอคอนส่วนขยายบนเว็บไซต์เหล่านี้และเลือก "อนุญาตเสมอบน \*\*\*" การดำเนินการนี้จะให้สิทธิ์ส่วนขยายในการเพิ่มแถบด้านข้างบนโดเมนที่ระบุ

![สิทธิ์ส่วนขยาย Firefox](https://img.newzone.top/2023-12-25-05-59-48.png?imageMogr2/format/webp)

หลังจากติดตั้งสำเร็จ ตรวจสอบ [บทเรียนการใช้งานปลั๊กอิน](./usage.md)
