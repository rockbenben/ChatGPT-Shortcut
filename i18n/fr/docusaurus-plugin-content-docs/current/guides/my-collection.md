# Ma collection

Une fois connecté, la vue **Ma collection** vous permet d'organiser vos prompts favoris comme une collection de signets&nbsp;: tri par glisser-déposer, balises personnalisées, filtrage ciblé — pour les retrouver d'un coup d'œil.

:::tip Version en ligne et version hors ligne
Dans la version en ligne, les données de collection sont stockées sur le serveur (connexion requise)&nbsp;; dans la [version déployée hors ligne](./offline), elles restent dans le navigateur (localStorage, sans connexion). Les fonctionnalités sont identiques.
:::

## Changement de vue

Le bouton de changement de vue en haut de la page d'accueil propose deux modes&nbsp;:

- **Ma collection** : affiche votre collection + vos prompts personnalisés
- **Bibliothèque de prompts** : parcourir la bibliothèque publique de prompts

Le navigateur retient votre choix et restaure la dernière vue à votre prochaine visite.

![Vue Ma collection](/img/docs/my-collection-view.gif)

## Ajouter à la collection

Cliquez sur "Collectionner" sur une carte pour l'ajouter à votre collection.

Collectionner un prompt de la communauté équivaut automatiquement à un vote positif — un bon moyen d'aider les contenus de qualité à gagner en visibilité.

## Retirer de la collection

Dans la vue Ma collection, cliquez sur l'icône en forme de cœur sur la carte pour la retirer de la collection.

## Balises personnalisées

Ajoutez des balises à vos prompts collectionnés pour les classer :

### Ajouter une balise

1. Cliquez sur l'icône de balise dans la barre de contrôle
2. Sélectionnez une balise existante, ou cliquez sur "+" pour en créer une nouvelle
3. Les nouvelles balises ont un nom et une couleur personnalisables

### Gérer les balises

Cliquez sur "Gérer les balises" pour accéder à l'interface de gestion :

- Modifier le nom d'une balise
- Changer la couleur d'une balise (11 couleurs prédéfinies)
- Supprimer une balise

![Fenêtre de gestion des balises](/img/docs/my-collection-tag.png)

### Filtrer par balise

La barre de filtrage de la vue Ma collection permet de filtrer par balise personnalisée.

## Tri par glisser-déposer

La vue Ma collection permet d'ajuster l'ordre des cartes par glisser-déposer :

1. Faites glisser la poignée à gauche de la carte
2. Relâchez à la position cible
3. Le nouvel ordre est enregistré automatiquement (sur le serveur pour la version en ligne, en local pour la version hors ligne)

Le tri n'est disponible que dans la vue Ma collection.

![Tri par glisser-déposer](/img/docs/my-collection-drag.gif)
