# Gestion de compte

AiShort s'utilise sans compte — mais **une fois connecté**, vous pouvez constituer votre collection de prompts favoris, créer vos propres prompts, les partager avec la communauté et synchroniser vos données entre tous vos appareils. Cette page couvre la connexion, l'inscription, les mots de passe et les données.

## Méthodes de connexion

Le dialogue de connexion propose les onglets «&nbsp;**Connexion par mot de passe**&nbsp;» et «&nbsp;**Connexion sans mot de passe**&nbsp;», ainsi qu'une connexion Google en un clic.

### Connexion par mot de passe

Connectez-vous avec votre nom d'utilisateur ou votre e-mail enregistré ainsi que votre mot de passe.

### Connexion sans mot de passe

1. Basculez sur «&nbsp;Connexion sans mot de passe&nbsp;» et saisissez votre e-mail enregistré
2. Cliquez sur «&nbsp;**Obtenir le lien de connexion**&nbsp;» — un lien de connexion est envoyé dans votre boîte de réception
3. Ouvrez le lien pour vous connecter

Rien à mémoriser — pratique si vous ne vous connectez qu'occasionnellement.

### Connexion Google

Cliquez sur le bouton Google dans le dialogue de connexion pour vous connecter en un clic. La première fois, nous créons automatiquement un compte pour vous — aucune inscription séparée n'est nécessaire.

![](/img/docs/account-login.png)

## Inscription au compte

1. Dans le dialogue de connexion, cliquez sur «&nbsp;**S'inscrire maintenant**&nbsp;» pour passer à l'inscription
2. Remplissez l'e-mail, le nom d'utilisateur et le mot de passe, et acceptez la Politique de confidentialité et les Conditions d'utilisation
3. La connexion est automatique après validation

![](/img/docs/account-register.png)

## Mon Compte

Après connexion, ouvrez la page «&nbsp;**Mon Compte**&nbsp;» pour gérer votre profil, votre mot de passe, votre niveau et vos données.

![Page Mon Compte](/img/docs/account-profile.png)

### Changer de nom d'utilisateur

Cliquez sur l'icône de modification à côté de votre nom d'utilisateur, puis enregistrez.

### Changer de mot de passe

1. Saisissez votre mot de passe actuel
2. Saisissez votre nouveau mot de passe et confirmez-le
3. Cliquez sur «&nbsp;Modifier le mot de passe&nbsp;» pour valider

Mot de passe oublié&nbsp;? Dans le dialogue de connexion, basculez sur «&nbsp;Mot de passe oublié&nbsp;», saisissez votre e-mail enregistré, cliquez sur «&nbsp;Envoyer l'email de réinitialisation&nbsp;», puis suivez le lien.

### Mon niveau

La page de compte affiche votre **niveau (L0–L9)** et votre progression vers le palier suivant — votre niveau est déterminé par le **nombre de prompts que vous avez partagés avec la communauté**.

## Gestion des données

La rubrique «&nbsp;Mon Compte → Gestion des Données&nbsp;» propose trois actions&nbsp;:

- **Vider le cache** — actualise les données mises en cache localement&nbsp;; utilisez cette option lorsque les données ne se synchronisent pas ou que l'affichage semble anormal. Vos collections et prompts personnalisés ne sont pas perdus, cela force simplement une resynchronisation
- **Exporter les prompts** — exporte un fichier JSON pour la sauvegarde (voir [Prompts personnalisés](./user-prompts#exporter-les-prompts))
- **Importer les prompts** — importe des prompts et des collections depuis un fichier JSON
