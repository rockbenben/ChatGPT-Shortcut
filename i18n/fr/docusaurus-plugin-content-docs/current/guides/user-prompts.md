# Prompts personnalisés

Après connexion, vous pouvez créer vos propres prompts pour un accès rapide plus tard. Les prompts personnalisés seront enregistrés dans votre compte et affichés dans la vue Ma collection.

## Créer un prompt

1. Cliquez sur le bouton "Ajouter un prompt" en haut à droite de la page
2. Remplissez les informations du prompt :
   - **Titre** : Brève description de l'objectif du prompt
   - **Description** : Le contenu complet du prompt
   - **Remarque** : Instructions d'utilisation ou notes (optionnel)
3. Choisissez de partager avec la communauté
4. Cliquez sur Soumettre

![Fenêtre contextuelle d'ajout de prompt](/img/docs/user-prompts-create.png)

## Iditer un prompt

Dans la vue Ma collection, cliquez sur la carte de prompt que vous avez créée pour :

- Modifier le titre, la description et la remarque
- Changer le statut de partage (Public/Privé)
- Enregistrer les modifications

![Interface d'édition de prompt](/img/docs/user-prompts-edit.png)

## Supprimer un prompt

Cliquez sur le bouton "Supprimer" dans l'interface d'édition pour supprimer le prompt. La suppression est irréversible, veuillez opérer avec prudence.

## Partager avec la communauté

Lors de la création ou de l'édition d'un prompt, cochez l'option "Partager avec la communauté" :

- **Public** : Le prompt sera affiché sur la page de la communauté, et les autres utilisateurs peuvent le voir et le collectionner
- **Privé** : Visible uniquement par vous, n'apparaîtra pas sur la page de la communauté

Vous pouvez modifier le statut de partage à tout moment.

## Exporter une sauvegarde

Exportez tous les prompts créés par vous :

1. Entrez dans le Centre utilisateur
2. Cliquez sur le bouton "Exporter les prompts"
3. Le système génère un fichier JSON et le télécharge automatiquement

Le contenu de l'exportation comprend :

- Titre du prompt, description, remarque
- Heure de création et de mise à jour
- Statut de partage

Il est recommandé d'exporter des sauvegardes régulièrement pour éviter la perte de données.
