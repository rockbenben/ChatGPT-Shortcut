# Paramètres de l'extension Firefox

Le modèle de permissions de Firefox est plus strict que celui de Chrome — après installation, deux étapes supplémentaires sont nécessaires pour que la barre latérale s'affiche correctement sur ChatGPT, Gemini et d'autres sites.

## 1. Épingler l'extension et configurer l'accès

Commencez par sélectionner "Épingler l'extension ChatGPT Shortcut" (Pin to Toolbar) dans la barre d'outils de Firefox. Allez ensuite dans le Gestionnaire d'extensions, trouvez l'entrée ChatGPT Shortcut et sélectionnez "Options" (Options) pour accéder aux paramètres de l'extension. Les étapes détaillées sont illustrées ci-dessous :

![Firefox settings](https://img.newzone.top/2023-12-25-05-51-47.png?imageMogr2/format/webp)

## 2. Autoriser l'exécution de l'extension

Ensuite, pour que l'extension fonctionne correctement sur des sites comme ChatGPT, Gemini, etc., faites un clic droit sur l'icône de l'extension sur ces sites et sélectionnez "Toujours autoriser sur ce site web" (Always allow on \*\*\*). Cette opération accorde à l'extension la permission d'ajouter une barre latérale sur le domaine spécifié.

![Firefox Extension Permission](https://img.newzone.top/2023-12-25-05-59-48.png?imageMogr2/format/webp)

Après une installation réussie, vous pouvez consulter le [Tutoriel d'utilisation de l'extension](./usage.md).
