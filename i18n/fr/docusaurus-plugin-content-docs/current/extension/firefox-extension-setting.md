# Paramètres de l'extension Firefox

Avant d'utiliser l'extension Firefox, il est recommandé de suivre ces deux étapes pour la configuration.

## 1. Épingler l'extension et paramétrer l'accès

Tout d'abord, sélectionnez "Épingler l'extension ChatGPT Shortcut" (Pin to Toolbar) dans la barre d'outils Firefox. Ensuite, entrez dans le Centre d'extensions, trouvez l'entrée de l'extension ChatGPT Shortcut et sélectionnez "Options" (Options). Entrez dans l'interface de configuration de l'extension. Les étapes spécifiques sont indiquées ci-dessous :

![Firefox settings](https://img.newzone.top/2023-12-25-05-51-47.png?imageMogr2/format/webp)

## 2. Autoriser l'exécution de l'extension

Deuxièmement, pour s'assurer que l'extension fonctionne normalement sur des sites web comme ChatGPT, Gemini, vous devez faire un clic droit sur l'icône de l'extension sur ces sites web et sélectionner "Toujours autoriser sur ce site web" (Always allow on \*\*\*). Cette opération accorde à l'extension la permission d'ajouter une barre latérale sur le domaine spécifié.

![Firefox Extension Permission](https://img.newzone.top/2023-12-25-05-59-48.png?imageMogr2/format/webp)

Après une installation réussie, vous pouvez consulter le [Tutoriel d'utilisation du plugin](./usage.md).
