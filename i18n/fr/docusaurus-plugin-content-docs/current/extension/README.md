# Installer l'extension de navigateur

L'extension AiShort (ChatGPT Shortcut) intègre la bibliothèque de prompts directement dans la barre latérale des pages de discussion IA telles que ChatGPT, Gemini, Claude ou Doubao, vous évitant de revenir sur aishort.top pour copier-coller. Compatible avec Chrome, Edge et Firefox, elle s'invoque rapidement avec `Alt + Shift + S`.

## Installation

### 1. Boutiques d'extensions (recommandé, installation en un clic)

- **Chrome** : [Chrome Web Store](https://chrome.google.com/webstore/detail/chatgpt-shortcut/blcgeoojgdpodnmnhfpohphdhfncblnj)
- **Edge** : [Microsoft Edge Addons](https://microsoftedge.microsoft.com/addons/detail/chatgpt-shortcut/hnggpalhfjmdhhmgfjpmhlfilnbmjoin)
- **Firefox** : [Firefox Browser ADD-ONS](https://addons.mozilla.org/addon/chatgpt-shortcut/) (une [autorisation ponctuelle](./firefox-extension-setting) est requise après l'installation, sinon la barre latérale ne s'affichera pas sur ChatGPT)

### 2. La boutique est inaccessible ? Paquet d'installation local

Téléchargez depuis l'une des sources ci-dessous, puis suivez le tutoriel correspondant :

- **GitHub** : [GitHub Release](https://github.com/rockbenben/ChatGPT-Shortcut/releases/latest)

> 📖 Guides d'installation : [fichier CRX](./manual-chrome-extension) · [fichier ZIP](./manual-chrome-extension-zip)

## Une fois installé

Appuyez sur `Alt + Shift + S` pour l'invoquer, ou cliquez sur l'icône dans la barre d'outils. Consultez le [guide d'utilisation](./usage) pour plus de détails.

## Script Tampermonkey

En plus du format d'extension, nous fournissons également le [**Script Tampermonkey ChatGPT Shortcut Anywhere**](https://greasyfork.org/scripts/482907-chatgpt-shortcut-anywhere).
Tampermonkey est une extension de navigateur qui permet aux utilisateurs d'exécuter des scripts personnalisés pour améliorer les fonctions des pages web.

Avec ce script, vous pouvez invoquer la barre latérale AiShort sur n'importe quel site web.
Cependant, en raison des restrictions de la page officielle de ChatGPT sur l'injection de scripts, sur cette page, le script s'exécutera sous forme de **fenêtre contextuelle** au lieu d'une barre latérale.

Une fois la barre latérale AiShort activée, vous verrez une icône verte en bas à droite des pages prises en charge. Cliquez sur cette icône pour ouvrir ou fermer la barre latérale. Les sites pris en charge par défaut incluent actuellement ChatGPT, Gemini, Claude, Doubao, etc.
