# Comment utiliser l'extension ?

ChatGPT Shortcut prend en charge 18 langues principales. La langue de l'extension est définie automatiquement selon l'environnement de votre navigateur. Vous pouvez aussi la changer manuellement dans les paramètres.

![](/img/docs/extension-settings.png)

## Mode d'affichage

L'extension propose trois modes d'affichage — choisissez celui qui correspond le mieux à votre usage :

- **Mode barre latérale** : s'ouvre dans la barre latérale du navigateur — affiché côte à côte avec la page web, idéal pour parcourir des prompts tout en consultant une page de conversation IA
- **Mode popup** : cliquez sur l'icône de l'extension pour ouvrir une petite fenêtre — consultation rapide puis fermeture, sans encombrer l'écran
- **Mode fenêtre indépendante** : s'ouvre dans une fenêtre flottante autonome — peut être déplacée sur un second écran ou placée librement

## Garder la barre latérale ouverte

Une fois cette option activée, la barre latérale reste ouverte lors du changement d'onglet, pour un usage continu.

## Mode sombre

L'extension prend en charge le mode sombre : activez l'option "Mode sombre" dans les paramètres pour basculer l'interface en apparence sombre.

![](/img/docs/extension-sidebar.png)

## Raccourci clavier

Utilisez le raccourci `Alt+Shift+S` pour activer directement l'extension.
