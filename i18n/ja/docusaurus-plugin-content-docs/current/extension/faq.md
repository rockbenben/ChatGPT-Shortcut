# 拡張機能に関するよくある質問

## ChatGPT ページの制限

ChatGPT は拡張機能に埋め込まれたページに複数の権限制限を設定しているため、Google 認証を使用したログインはサポートされていません。つまり、ユーザーはアカウントとパスワードを使用してログインする必要があります。

アカウントが元々 Google 認証で作成された場合は、「パスワードを忘れた場合」機能を使用して新しいパスワードを設定できます。これにより、埋め込まれた ChatGPT ページ内でスムーズにログインできるようになります。

## コンテンツブロックのプロンプト

アカウントとパスワードを使用してログインした後、ページに「このコンテンツはブロックされました」というプロンプトが表示されることがあります。このような場合、通常は**ページを更新**するだけで問題が解決し、ページは正常なログイン状態に戻ります。また、一度ログインすると、積極的にログアウトしない限り、ログイン状態は通常長時間維持されるため、このようなプロンプトが頻繁に表示されることを過度に心配する必要はありません。

ChatGPT の統合ページ内で言語を切り替えると、「このコンテンツはブロックされました」というプロンプトがトリガーされる可能性があることに注意してください。トラブルを避けるために、統合ページ内で直接操作するのではなく、拡張機能の設定項目を通じて言語を切り替えることをお勧めします。これにより、不要なエラープロンプトを回避できます。
