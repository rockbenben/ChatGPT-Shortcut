# ブラウザ拡張機能の使い方

ChatGPT Shortcut は 18 の主要言語をサポートしています。拡張機能の言語はブラウザ環境に応じて自動設定されます。設定ページで手動変更も可能です。

![](/img/docs/extension-settings.png)

## 表示モード

拡張機能は 3 つの表示モードをサポートしており、使い方に合わせて選択できます：

- **サイドバーモード**：ブラウザのサイドバーで開き、Web ページと並べて表示 — AI 対話ページを見ながらプロンプトを閲覧するのに最適
- **ポップアップモード**：拡張機能アイコンをクリックして小窓をポップアップ表示 — サッと確認してすぐ閉じられ、画面を占有しない
- **ウィンドウモード**：独立したフローティングウィンドウで開く — サブモニターへ移したり、独立配置できる

## サイドパネルを開いたままにする

このオプションを有効にすると、ブラウザタブを切り替えてもサイドバーが開いたままになり、継続利用に便利です。

## ダークモード

拡張機能はダークモードをサポートしています。設定ページで「ダークモード」を有効にすると、インターフェースがダーク外観に切り替わります。

![](/img/docs/extension-sidebar.webp)

## ホットキー起動

`Alt+Shift+S` ショートカットで拡張機能を直接起動できます。
