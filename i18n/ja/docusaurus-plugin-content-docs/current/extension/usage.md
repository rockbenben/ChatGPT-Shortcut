# ブラウザ拡張機能の使用方法は？

ChatGPT Shortcut は 13 の主要言語をサポートしており、拡張機能の言語はブラウザ環境に応じて自動的に設定されます。拡張機能内の ChatGPT 埋め込みページとサイドバーの言語もこの設定に従います。サードパーティの Web サイトからの権限警告がトリガーされるのを避けるために、**埋め込みページで言語を変更しないでください**。

![](/img/docs/extension-settings.png)

## AiShort サイドバー

AiShort 拡張機能は現在、サイドパネル（Side Panel）として動作し、スムーズな対話体験を提供します。

設定で「サイドパネルを開いたままにする（Keep side panel open）」オプションを有効にすると、ブラウザのタブを切り替えてもサイドバーが開いたままになり、継続的に使用するのに便利です。

![](/img/docs/extension-sidebar.png)

他の Web サイトで AiShort サイドバーを使用したい場合は、[ChatGPT Shortcut Anywhere](https://greasyfork.org/scripts/482907-chatgpt-shortcut-anywhere) スクリプトをインストールしてください。Tampermonkey は、ユーザーがカスタムスクリプトを実行して Web ページを変更できるブラウザ拡張機能です。機能の重複と潜在的な競合を避けるために、ChatGPT Shortcut Anywhere スクリプトはデフォルトで拡張機能がサポートする Web サイトでは有効にならないことに注意してください。

## ホットキーアクティブ化

`Alt+Shift+S` ショートカットを使用すると、AiShort サイドバーを直接アクティブにできます。
