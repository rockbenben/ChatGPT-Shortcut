# ブラウザ拡張機能の使い方

ChatGPT Shortcut は 13 の主要言語をサポートし、拡張機能の言語はブラウザ環境に応じて自動的に設定されます。設定ページで手動で言語を変更することもできます。

![](/img/docs/extension-settings.png)

## AiShort サイドバー

AiShort 拡張機能は現在サイドパネルとして動作し、シームレスなインタラクティブ体験を提供します。

設定で「サイドパネルを開いたままにする」を有効にすると、ブラウザタブを切り替えてもサイドバーは開いたままになります。

![](/img/docs/extension-sidebar.png)

## ダークモード

拡張機能はダークモードをサポートしています。設定ページで「ダークモード」を有効にすると、サイドバーと設定ページがダークテーマに切り替わります。

## ホットキー起動

`Alt+Shift+S` ショートカットを使用して、AiShort サイドバーを直接起動できます。
