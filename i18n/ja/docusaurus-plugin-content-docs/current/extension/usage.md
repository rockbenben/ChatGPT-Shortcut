# ブラウザ拡張機能の使い方

ChatGPT Shortcut は 13 の主要言語をサポートしています。拡張機能の言語はブラウザ環境に応じて自動設定されます。設定ページで手動変更も可能です。

![](/img/docs/extension-settings.png)

## 表示モード

拡張機能は 3 つの表示モードをサポートしています：

- **サイドバーモード**：ブラウザのサイドバーで開き、Web コンテンツと並べて表示
- **ポップアップモード**：拡張機能アイコンをクリックするとポップアップで表示
- **ウィンドウモード**：独立したフローティングウィンドウで開く

## サイドパネルを開いたままにする

有効にすると、ブラウザタブを切り替えてもサイドバーは開いたままになります。

## ダークモード

拡張機能はダークモードをサポートしています。設定ページで「ダークモード」を有効にすると、インターフェースがダーク外観に切り替わります。

![](/img/docs/extension-sidebar.png)

## ホットキー

`Alt+Shift+S` ショートカットで拡張機能を直接起動できます。
