# 始め方

AI と会話する際、良いプロンプトは質の高い回答をより早く得るのに役立ちます。AiShort は、コピーしてすぐに使える実用的なプロンプトを多数整理しました。

## 基本的な使用方法

1. ホームページで必要なプロンプトを検索または選択します
2. カードをクリックして詳細を確認するか、直接「コピー」ボタンをクリックします
3. 任意の AI モデルを開き、プロンプトを貼り付けます
4. プロンプトの指示に従って、具体的な質問を追加します
