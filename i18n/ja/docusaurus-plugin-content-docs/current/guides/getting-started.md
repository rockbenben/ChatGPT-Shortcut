# 始め方

良いプロンプトが AI の出力品質を決めます。AiShort は厳選された大量のプロンプトを収録しており、コピー＆ペーストですぐに使えます。

> 📋 **事前に必要なもの**：ブラウザ 1 つ＋利用できる AI ツール 1 つ (ChatGPT などのチャットページ、Cursor などのコーディングツール、API 呼び出しなど何でも OK)。AiShort の Web 版は登録不要です。

## 基本的な使用方法

1. ホームページで気になるプロンプトを検索または閲覧する
2. 適したプロンプトを見つけたら「コピー」ボタンをクリック (またはカードを開いて全文を確認)
3. 任意の AI ツールに貼り付ける——ChatGPT などのチャットページ、Cursor などのコーディングツール、API 呼び出しなど
4. プロンプト内の**プレースホルダー** (例：`[内容]`、`[テーマ]`) を、自分の具体的な要件に置き換えて AI に送信する

![](/img/docs/how-to-use-aishort.gif)

> 💡 **ヒント**：多くのプロンプトは大括弧 `[...]` で補足すべき内容をマークしています — 大括弧内のプレースホルダーテキストを、自分の具体的な内容に置き換えるだけです。
