# はじめに

AiShort を使用すると、選択したプロンプトを任意の AI ダイアログ モデルにコピーするだけで、指定した方向の出力を取得できます。具体的な手順は次のとおりです。

1. 必要なプロンプトを検索または選択し、プロンプトのコピー ボタンをクリックします。

2. プロンプトを ChatGPT、Gemini、または任意の AI ダイアログ モデルにコピーします。

3. ダイアログ プロンプトに従って要件を入力します。
