# マイコレクション

ログイン後、**マイコレクション**はお気に入りのプロンプトをブックマーク感覚で管理できるスペースです。ドラッグで並べ替え、タグで分類し、タグでフィルタリングすれば必要なプロンプトをすばやく呼び出せます。自分が作成したプロンプトもここに表示されます。

:::tip[オンライン版とオフライン版]

オンライン版のコレクションデータはサーバーに保存されます（ログインが必要）。[オフラインデプロイ版](../deploy/offline)ではコレクションデータがブラウザのローカルストレージ（localStorage、ログイン不要）に保存されます。機能はどちらも同等です。

:::

## ビューの切り替え

ホームページ上部のビュー切り替えボタンで 2 つのモードを使い分けられます：

- **マイコレクション**：コレクションしたプロンプト＋カスタムプロンプトを表示
- **見つける**：公開の厳選プロンプトライブラリを閲覧

ブラウザは最後に使ったビューを記憶しているので、次回アクセス時に前回の状態から再開できます。

![マイコレクションビュー](/img/docs/my-collection-view.webp)

## コレクションの概要

マイコレクションビューの上部にはサマリーカードが表示され、一目で状況を把握できます：

- 4 つの統計 — **合計** / **マイプロンプト** / **お気に入り** / **カスタムタグ**
- **レベルバッジ**（L0〜L9）— コミュニティへのプロンプト共有数に応じて上昇します

## コレクションに追加

カード上の「コレクション」ハートボタンをクリックするとマイコレクションに追加されます。

コミュニティプロンプトをコレクションに追加すると自動的に賛成票が 1 票投じられ、優れたコンテンツの普及に貢献できます。

## コレクションから削除

マイコレクションビューでカードのハートアイコンをクリックするとコレクションを解除できます。

## フィルターと検索

コレクションビューの上部ツールバーで素早く目的のものを探せます：

- **すべて** / **プロンプト** / **お気に入り** のセグメント切り替えでタイプ別にフィルタリング
- カスタムタグチップ — クリックしてタグでフィルタリング
- 検索ボックス（`Ctrl + K` でフォーカス）でキーワード検索

## カスタムタグ

コレクションしたプロンプトにタグを付けて分類できます：

### タグを追加

1. ツールバーの「**タグ管理**」をクリックします
2. タグ名を入力し、**11 色のプリセットカラー**から色を選びます
3. 保存すると、フィルターバーでそのタグが使えるようになります

### タグを管理

「タグ管理」ダイアログでは以下の操作が行えます：

- タグ名を変更
- タグの色を変更（11 色のプリセットカラー）
- タグを削除

![タグ管理ポップアップ](/img/docs/my-collection-tag.png)

## ドラッグ並べ替え

マイコレクションビューではドラッグ＆ドロップで並べ替えができます：

1. カード左端のドラッグハンドルをつかみます
2. 目的の位置にドロップします
3. 新しい順序は自動的に保存されます（オンライン版はサーバー、オフライン版はブラウザ）

![ドラッグ並べ替え操作](/img/docs/my-collection-drag.webp)
