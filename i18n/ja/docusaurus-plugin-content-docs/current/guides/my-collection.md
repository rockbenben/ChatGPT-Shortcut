# マイコレクション

ログイン後、お気に入りのプロンプトをコレクションして個人的に管理できます。マイコレクションビューは、プロンプトを効率的に整理して使用するための豊富な管理機能を提供します。

## ビューの切り替え

ホームページの上部にビュー切り替えボタンがあり、2 つのモードをサポートしています：

- **マイコレクション**：コレクションしたプロンプトとカスタムプロンプトを表示
- **プロンプトライブラリ**：パブリックプロンプトライブラリを閲覧して新しいコンテンツを発見

ブラウザはビューの選択を自動的に記憶し、次回の訪問時に最後のビューに切り替わります。

![マイコレクションビュー](/img/docs/my-collection-view.gif)

## コレクションに追加

プロンプトカードの「コレクション」ボタンをクリックして、マイコレクションに追加します。

コミュニティプロンプトをコレクションすると、システムは自動的にそれに賛成票を投じ、質の高いコンテンツの露出を増やすのを助けます。

## コレクションから削除

マイコレクションビューで、カードのハートアイコンをクリックしてコレクションを解除します。

## カスタムタグ

コレクションしたプロンプトに個人タグを追加して、分類管理を便利にします：

### タグを追加

1. コントロールバーのタグアイコンをクリックします
2. 既存のタグを選択するか、「+」をクリックして新しいタグを作成します
3. 新しいタグにはカスタムの名前と色を設定できます

### タグを管理

「タグを管理」をクリックして、タグ管理インターフェースに入ります：

- タグ名を変更
- タグの色を変更（11 色のプリセットカラーが利用可能）
- 不要なタグを削除

![タグ管理ポップアップ](/img/docs/my-collection-tag.png)

### タグでフィルタリング

マイコレクションビューのフィルタリングバーで、カスタムタグ別にコンテンツをフィルタリングできます。

## ドラッグアンドドロップ並べ替え

マイコレクションビューでは、カードの順序をドラッグアンドドロップで調整できます：

1. カードの左側にあるドラッグハンドルをドラッグします
2. 目的の位置に移動して放します
3. 新しい順序は自動的にサーバーに保存されます

並べ替え機能は、マイコレクションビューでのみ利用可能です。

![ドラッグアンドドロップ並べ替え](/img/docs/my-collection-drag.gif)
