# カスタムプロンプト

ログイン後、独自のプロンプトを作成してアカウントに保存し、いつでも呼び出せます。カスタムプロンプトは「マイコレクション」ビューに表示されます。

## プロンプトを作成

1. 右上の「追加」ボタンをクリックします
2. ポップアップで開く「プロンプトを作成」ウィンドウのフォームに入力します：
   - **プロンプト名** (必須)：プロンプトに名前を付けます
   - **プロンプト内容** (必須)：プロンプトの本文
   - **役割・用途** (任意)：プロンプトが何をするかの簡単な説明
   - **備考説明** (任意)：出典、他言語版、補足説明など
3. 下部の「このプロンプトを公開ページに共有しますか？」スイッチはデフォルトでオン — オフにすれば自分だけが見られます
4. 「プロンプトを作成」をクリックして送信します

![プロンプト作成ポップアップ](/img/docs/user-prompts-create.png)

> 💡 **入力例** — 「日報作成アシスタント」プロンプト：
> - **プロンプト名**：日報作成アシスタント
> - **プロンプト内容**：あなたはビジネス文書作成のアシスタントです。以下の内容をもとに、本日の業務日報を作成してください。「本日の業務」「進捗状況」「明日の予定」「課題・所感」の構成で、簡潔かつ要点を押さえた文章にまとめてください：[本日の業務メモ]
> - **役割・用途**：箇条書きのメモを上司への提出にふさわしい日報に整える
> - **備考説明**：毎日終業前に 1 回使用

## プロンプトを編集

マイコレクションビューで、自分が作成したプロンプトカードをクリックすると「プロンプトを編集」ウィンドウが開き、以下のことができます：

- 名前、内容、役割、備考を変更
- 共有ステータスを切り替え (公開／非公開)
- 「保存」をクリックして更新

![プロンプト編集インターフェース](/img/docs/user-prompts-edit.png)

## プロンプトを削除

編集ウィンドウで「削除」をクリックします。削除後は元に戻せませんので、慎重に操作してください。

## コミュニティに共有

作成または編集時、下部の「このプロンプトを公開ページに共有しますか？」スイッチで可視性を制御します：

- **オン (デフォルト)**：プロンプトは [コミュニティプロンプト](./community) ページに表示され、他のユーザーが見たりコレクションしたりできます
- **オフ**：非公開プロンプトとなり、自分だけが見られます

共有ステータスはいつでも切り替え可能です。

## データのエクスポート

1. 「マイアカウント」に入り、「データ管理 → プロンプトのエクスポート」を探します
2. 「データをエクスポート」ボタンをクリックします
3. システムが JSON ファイルを生成し自動でダウンロードします

エクスポート内容：

- プロンプトの名前、内容、役割・用途、備考説明
- 作成および更新時間
- 共有ステータス

データの損失を防ぐため、定期的なエクスポートをおすすめします。

## データのインポート

JSON ファイルからプロンプトとコレクションをインポートします：

1. 「マイアカウント」に入り、「データ管理 → プロンプトのインポート」を探します
2. 「データをインポート」ボタンをクリックします
3. JSON ファイルを選択します
4. システムが自動でデータをマージします (タイトルで重複排除。他のユーザーが作成したプロンプトは自動で非公開に設定)

### チームコラボレーション

チーム内でプロンプトを共有する際の推奨ワークフロー：

1. **フィルター＆共有**：エクスポート後、コレクションリストを手動で削除するか共有したいプロンプトをフィルターし、チームメンバーにファイルを送ってインポートしてもらいます
2. **プライバシー保護**：他のメンバーから受け取ったプロンプト (現在のアカウントに属さない ID のもの) をインポートすると自動で**非公開**に設定され、メンバー間でのデータの混在を防ぎます
