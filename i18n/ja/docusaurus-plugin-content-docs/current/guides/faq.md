# よくある質問

## AiShort は AI プロンプトジェネレーターですか？

いいえ。AiShort はシーン別に**精選・整理**された人手によるプロンプトテンプレート集で、LLM で自動的にプロンプトを生成するツールではありません。各プロンプトは厳選のうえタグで分類されており、ワンクリックでコピーして ChatGPT / Claude / Gemini / DeepSeek などお好きな AI チャットツールに貼り付けるだけで使えます。「要件から自動でプロンプトを生成する」タイプをお探しなら、PromptPerfect や ChatGPT prompt generator などが別カテゴリの製品として存在します。

## AiShort は無料ですか？API Key が必要ですか？

完全無料で、**API Key も登録も不要**です。プロンプトの閲覧・検索・コピーにアカウントは一切いりません。

登録するとコレクションとドラッグ並べ替え、カスタムタグ、プライベートプロンプトの作成・管理、コミュニティ共有と投票、JSON データのエクスポート、デバイス間同期などの拡張機能が解放されます——こちらも無料です。コードはすべて [GitHub](https://github.com/rockbenben/ChatGPT-Shortcut) でオープンソースとして公開しています。

注意：AiShort はプロンプトを提供するのみで、**AI モデル自体の利用が有料かどうかは、ご利用の AI サービスによって決まります** (例：ChatGPT 無料版/有料版、API 課金など)。

## プロンプトは何件収録？どんなシーンをカバー？

AiShort には **合計 5000+ 件のプロンプト** が登録されており、2 つのライブラリに分かれています：

- **厳選ライブラリ** —— 人手で選定、25+ シーンタグで分類、18 言語の翻訳が完備。タグ：ライティング補助、IT/プログラミング、記事/レポート、SEO/マーケティング、企業職能、学術/教師、教育/学生、言語/翻訳、心理/対人、思考トレーニング、生産性ツール、ターミナル/インタプリタ、討論/スピーチ、レビュー/評価、面白い豆知識、生活百科、医療健康、金融アドバイザー、音楽芸術、哲学/宗教、文章/単語、ゲーム、専門アドバイザーなど。
- **[コミュニティライブラリ](./community)** —— 大多数を占め、ユーザーが継続的に投稿、人気順や新着順で並び替え可能、検索フィルタにも対応；シーンの粒度と広さで厳選を上回ります。

厳選で見つからないときは、コミュニティで検索してみてください。

## どの AI モデルに対応？同じプロンプトで使い回せる？

AiShort のプロンプトは汎用的で、**プロンプトを入力できる AI シーンならどこでも使えます**——チャットページだけでなく、Cursor などのコーディングツール、API 呼び出し、AI エージェントなども含みます。代表的なチャット系モデルは以下のとおりです。

- 海外：ChatGPT、Gemini、Claude、Grok
- 中国：DeepSeek、通義千問、文心一言、Doubao、Kimi、ChatGLM、iFLYTEK Spark、テンセント元宝
- API プラットフォーム：OpenRouter、SiliconFlow、Groq

より多くのモデル一覧は [始め方 → 一般的な AI モデル](./getting-started#一般的な-ai-モデル) をご覧ください。

**モデル横断の汎用性**：ほとんどのテキスト系プロンプトはモデル間で共通して使えます——ライティング、翻訳、プログラミング、Q&A といった一般的なタスクなら、同じプロンプトで主要な大規模言語モデルから有効な結果を引き出せます。AiShort の各プロンプトはデフォルトで特定モデルに縛られません。ただし**得意分野には差があります**：ライティング系は Claude がディテール表現に優れ、ChatGPT-5 は指示への追従が高め。プログラミング系は GPT-5、Gemini Pro、DeepSeek にそれぞれ強みがあり、推論密度の高いタスクでは Claude Opus / Gemini Deep Thinking / o-series が安定する傾向です。画像生成系プロンプト (Midjourney / Stable Diffusion / DALL·E) は汎用ではなく、それぞれの構文に合わせた調整が必要です。

## なぜプロンプトは英語で書かれている？

AI モデルは英語の理解力がより正確で、出力も安定する傾向があります。日本語のプロンプトでも使用できますが、同じ日本語プロンプトを繰り返し実行すると結果のばらつきが大きくなりがちです。重要な場面では英語のプロンプトをおすすめします。

> 💡 日本語で返信が欲しい場合は、プロンプトの末尾に `respond in Japanese` を追加してください。

## 毎回プロンプトを入力する必要は？

**API 利用時**：プロンプトを `system prompt` として設定すれば、以降の会話に自動的に適用されます。

**Web 版**：会話を切り替えない限り、ChatGPT は現在のプロンプトを記憶します。返信が期待から外れ始めたら、AI が「忘れた」状態なので、もう一度貼り付け直してください。

**ヒント**：よく使う会話をブラウザのブックマークに保存しておくと、次回すぐに開けて便利です。

## 自分に合うプロンプトの選び方は？

欲しい出力から逆引きで探しましょう。記事を書くなら [`write`](/?tags=write) または [`article`](/?tags=article) タグ、コードを書くなら [`code`](/?tags=code) または [`interpreter`](/?tags=interpreter)、翻訳なら [`language`](/?tags=language)、ロールプレイなら [`games`](/?tags=games) が便利です。タグに不慣れなときはキーワード検索を直接使ってください。各プロンプトカードには人気度 (コピー回数) が表示され、コピー数が多いものは品質が安定している傾向があります。

## 関連するプロンプトが見つからないのはなぜ？

ホームページの検索範囲は**厳選プロンプトライブラリ** (ログイン後は個人プロンプトも含む) で、コミュニティで共有されたプロンプトは**含まれません**。

短いキーワードでホームページ検索に出てこない場合は、[コミュニティプロンプト](./community) ページでもう一度検索してください——そこにはユーザー投稿のより豊富なコンテンツがあります。

## AI が誤情報を出力したらどうする？

AI は時として「ハルシネーション」を起こし、もっともらしく見えて実際には誤った情報を出力することがあります。対処方法：

1. **重要情報の確認**：特にデータ、引用、コードなど
2. **多段階の最適化**：AI に回答を再チェックさせ、最適化させる
3. **クロス検証**：異なるプロンプトやモデルを使って重要な結論を検証する

適切なプロンプトは AI のハルシネーションを減らすのに役立ちます。

## プロンプトの効果がいまいち、どう調整？

プロンプト自体を別物に取り替える前に、まず次の方向で調整してみてください。

1. **角括弧 `[プレースホルダ]` をより具体的に書く**——スタイル、長さ、領域、読者像などのディテールを追加します
2. **AI に追加質問して最適化**：満足できない返信には「もっと X にしてください」「Y のスタイルで書き直してください」と追問すると、1〜2 ラウンドで目標に近づくことが多いです。AI に自分の回答を採点させて改善させるのも有効です
3. **モデルを切り替えて比較**：同じプロンプトでも Claude Sonnet / ChatGPT / Gemini / DeepSeek で結果が大きく異なる場合があります——ライティング系は Claude がディテールに強く、プログラミング系は GPT-5 / DeepSeek が安定する傾向です
4. **コミュニティを試す**：[コミュニティプロンプト](./community) により適したバージョンがあるかもしれません——あなたの良いプロンプトもぜひ共有してください
5. **フィードバック**：問題や改善案があれば [フィードバック](/feedback) までお寄せください

## プロンプトのバックアップ方法は？

1. 「マイアカウント」に入り、「データ管理 → プロンプトのエクスポート」を探します
2. 「データをエクスポート」ボタンをクリックします
3. システムが JSON ファイルを自動生成しダウンロードします

データの損失を防ぐため、定期的なバックアップをおすすめします。

## AiShort のプロンプトは商用利用できる？

可能ですが、ライセンスの場合分けがあります。

1. [Awesome ChatGPT Prompts](https://github.com/f/awesome-chatgpt-prompts) 由来のコンテンツは **CC0 (パブリックドメイン)** に従い、制限はありません
2. AiShort 独自作成およびコミュニティ投稿コンテンツはデフォルトで **CC BY-SA 4.0** オープンライセンスを採用しています——商用利用、再配布、改変いずれも許可されますが、AiShort / 原作者のクレジット表示を残し、下流の利用者にも同じ開放ライセンスを引き継ぐことが条件です
3. **プロンプトを使って生成された AI コンテンツ**の著作権は、ご利用の AI サービスプロバイダ (OpenAI / Anthropic / Google など) の利用規約に従って帰属が決まります。多くの場合はあなたに帰属します
4. プロンプトカードの説明に明示的に「個人利用のみ」と記載されたごく一部の項目は除きます

## プロンプトをコピーするとデータは漏れる？

漏れません。AiShort のプロンプトは静的 JSON としてサイト資産にまとめてバンドルされており、**コピー操作はブラウザのローカルクリップボード上で完結します**。

あなたが**角括弧プレースホルダに入力した具体的な内容、AI からの返信**——これらはあなたと選択した AI プラットフォーム (ChatGPT / Claude / Gemini / DeepSeek など) の間で直接やり取りされ、AiShort からは見えません。

補足として、コピー時にはバックエンドに**匿名のカウントイベント** (POST `/cards/<id>/copy`) を送信していますが、これは各プロンプトの人気度 (カードに表示される「コピー回数」) を集計する目的のみに使われます。**送信されるのはカード ID だけ**で、入力内容も個人情報も含まず、ユーザー識別子とも紐づきません。

**ログイン後**のコレクションリスト、カスタムプロンプト、コミュニティ投稿のみがデバイス間同期のためにバックエンドに保存され、いつでも JSON エクスポートとワンクリック全削除に対応しています。
