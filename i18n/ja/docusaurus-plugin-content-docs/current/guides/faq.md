# よくある質問

## AiShort は有料ですか？

AiShort の Web 版は**完全無料**です。プロンプトの閲覧、コピー、検索のいずれも料金は不要で、アカウント登録さえ必要ありません。ログイン後の高度な機能 (コレクション、カスタムプロンプト、コミュニティ共有) も無料で利用できます。

注意：AiShort はプロンプトを提供するのみで、**AI モデル自体の利用が有料かどうかは、ご利用の AI サービスによって決まります** (例：ChatGPT 無料版／有料版、API 課金など)。

## 対応している AI モデルは？

AiShort のプロンプトは汎用的で、**あらゆる AI 利用シーン** (チャット対話、Cursor などのコーディングツール、API 呼び出し、AI エージェントなど) で使用できます。例：

- 海外の主要モデル：ChatGPT、Gemini、Claude、Grok
- 中国モデル：DeepSeek、通義千問、文心一言、Doubao、Kimi、ChatGLM、iFLYTEK Spark、テンセント元宝
- API プラットフォーム：OpenRouter、SiliconFlow、Groq

より多くのモデルについては [始め方 → 一般的な AI モデル](./getting-started#一般的な-ai-モデル) をご覧ください。

## なぜ英語のプロンプトを使うのですか？

AI モデルは英語の理解力がより正確で、出力も安定する傾向があります。日本語のプロンプトでも使用は可能ですが、同じプロンプトを繰り返し実行すると結果のばらつきが大きくなりがちです。重要な場面では英語のプロンプトをおすすめします。

> 💡 日本語で返信が欲しい場合は、プロンプトの末尾に `respond in Japanese` (または「日本語で回答してください」) を追加してください。英語のプロンプト＋日本語回答という組み合わせは、精度と読みやすさを両立できる定番の使い方です。

## 毎回プロンプトを入力する必要がありますか？

**API 利用時**：プロンプトを `system prompt` として設定すれば、以降の会話に自動的に適用されます。

**Web 版**：会話を切り替えない限り、ChatGPT は現在のプロンプトを記憶します。返信が期待から外れ始めたら、AI が「忘れた」状態なので、もう一度貼り付け直してください。

**ヒント**：よく使う会話をブラウザのブックマークに保存しておくと、次回すぐに開けて便利です。

## 関連するプロンプトが見つからないのはなぜですか？

ホームページの検索範囲は**厳選プロンプトライブラリ** (ログイン後は個人プロンプトも含む) で、コミュニティで共有されたプロンプトは**含まれません**。

短いキーワードでホームページ検索でヒットしない場合は、[コミュニティプロンプト](./community)ページでもう一度検索してください — そこにはユーザーが共有したより豊富なコンテンツがあります。

## AI が偽情報を出力した場合はどうすればよいですか？

AI は時として「ハルシネーション」を起こし、もっともらしく見えて実際には誤った情報を出力することがあります。対処方法：

1. **重要情報の確認**：特にデータ、引用、コードなど
2. **多段階の最適化**：AI に回答を再チェックさせ、最適化させる
3. **クロス検証**：異なるプロンプトやモデルを使って重要な結論を検証する

適切なプロンプトは AI のハルシネーションを減らすのに役立ちます。

## プロンプトをバックアップするには？

1. 「マイアカウント」に入り、「データ管理 → プロンプトのエクスポート」を探します
2. 「データをエクスポート」ボタンをクリックします
3. システムが JSON ファイルを自動生成しダウンロードします

データの損失を防ぐため、定期的なバックアップをおすすめします。

## プロンプトの効果が理想的でない？

1. **AI に追加質問して最適化**：AI に自分の回答を採点させ改善させる
2. **角度を変えて質問**：異なるプロンプトで同じ要件を表現してみる
3. **コミュニティを試す**：[コミュニティプロンプト](./community) により適したバージョンがあるかもしれません — あなたの良いプロンプトもぜひ共有してください
4. **フィードバック**：問題や改善案があれば[フィードバック](/feedback)までお寄せください
