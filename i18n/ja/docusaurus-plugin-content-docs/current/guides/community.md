# コミュニティプロンプト

コミュニティプロンプトは他のユーザーが投稿したものです。インスピレーションを探したり、自分が作ったものを共有したりできます。**投票の仕組み**によって優れたコンテンツが上位に浮かび上がり、有用なプロンプトは注目を集め、評価の低いものは静かに後退していきます。

![](/img/docs/community-page.png)

## 並べ替えとフィルタリング

- **最新発表**：最新の共有プロンプトを表示（デフォルト）
- **最多いいね**：最も賛成票を集めたプロンプトを表示
- **キーワード検索**：特定のトピックを検索

## 投票の仕組み

賛成・反対の投票によって、プロンプトの表示順位が決まります：

- **賛成**：優れたコンテンツをより多くの人の目に届けます
- **反対**：反対票が一定数に達すると、プロンプトはフィードから非表示になります

**コレクション = 賛成票**：コミュニティプロンプトをコレクションに追加すると、自動的に賛成票が 1 票投じられます。

![投票ボタンの状態](/img/docs/community-vote.png)

> 投票はログインユーザーのみ可能で、同一プロンプトに対する投票は最新のものが有効です。

## 非公開プロンプト

自分だけに留めておきたいアイデアは非公開に設定できます：

- 作成または編集時に「コミュニティに共有」をオフにします
- 非公開プロンプトは自分だけが見られます
- 公開・非公開はいつでも切り替え可能です

## ディスカッション

コミュニティプロンプトの詳細ページを開くと、全文のほかにログイン後はコメントを投稿して会話に参加できます（絵文字・GIF 対応）。
