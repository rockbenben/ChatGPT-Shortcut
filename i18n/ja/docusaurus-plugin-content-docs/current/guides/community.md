# コミュニティプロンプト

コミュニティプロンプトはユーザーから投稿されたもので、新しいインスピレーションを得たり、ご自身のアイデアを共有したりできます。**投票の仕組み**によって優れたコンテンツが上位に表示され、多くの方に役立つと評価されたプロンプトほど目に触れやすくなり、評価の低いものは次第に表示されにくくなります。

![](/img/docs/community-page.png)

## 並べ替えとフィルタリング

- **公開時間**：最新の共有プロンプトを表示
- **投票順**：賛成票が最も多いプロンプトを表示
- **キーワード検索**：特定のトピックを検索

## 投票の仕組み

賛成／反対の投票によって、プロンプトの表示順位が決まります：

- **賛成**：優れたコンテンツがより多くの方に届くよう後押しします
- **反対**：反対票が一定数に達すると、プロンプトは非表示になります

**コレクションとの連動**：コミュニティプロンプトをコレクションに追加すると、自動的に賛成票が 1 票投じられます。

![投票ボタンの状態](/img/docs/community-vote.png)

> 投票はログインユーザーのみ可能で、同一プロンプトに対する投票は最新のものが有効です。

## 非公開プロンプト

公開したくないアイデアは非公開に設定できます：

- 作成または編集時に「コミュニティに共有」のチェックを外します
- 非公開プロンプトは自分だけが見られます
- 公開／非公開はいつでも切り替え可能です
