# コミュニティプロンプト

コミュニティプロンプトセクションには、ユーザーが共有したカスタムプロンプトが表示されます。ここでインスピレーションを得たり、自分のアイデアを共有したりできます。

![](/img/docs/community-page.png)

## 並べ替えとフィルタリング

コミュニティページでは、複数のフィルタリング方法を提供しています：

- **公開時間**：最新の共有プロンプトを表示
- **投票順**：賛成票が最も多いプロンプトを表示
- **キーワード検索**：特定のトピックのプロンプトを検索

## 投票メカニズム

賛成/反対の投票でプロンプトの露出度を決定します：

- **賛成**：質の高いコンテンツの露出を増やすのを助けます
- **反対**：反対票が一定数に達すると、プロンプトは非表示になります

**コレクション連携**：コミュニティプロンプトをコレクションすると、システムは自動的に賛成票を投じます。

![投票ボタンの状態](/img/docs/community-vote.png)

> 投票はログインユーザーのみ利用可能で、同一プロンプトに対する各人の投票は最新のものが有効になります。

## 非公開プロンプト

公開したくないアイデアは非公開に設定できます：

- プロンプトの作成または編集時に、「コミュニティに共有」オプションのチェックを外します
- 非公開プロンプトは自分だけが見ることができます
- 公開/非公開の状態はいつでも切り替えることができます
