# インターフェースガイド

AiShort ページは、ユーザーがプロンプトを閲覧およびフィルタリングするための直感的なインターフェースを提供します。ページは主にタグエリア、検索エリア、プロンプト表示エリアで構成されています。

![](/img/docs/interface-home.png)

## 🏷︎ タグフィルタリング

タグエリアは、ユーザーがドメインや機能に基づいてプロンプトをフィルタリングするのを容易にします。

右上の「タグフィルタリングルール切り替え」ボタンを使用すると、より詳細なマルチタグフィルタリングを実現できます：

- **OR ルール**（デフォルト）：選択したタグのいずれかを持つプロンプトを表示
- **AND ルール**：選択したすべてのタグを同時に持つプロンプトのみを表示

![](/img/docs/interface-filter.png)

## 🔍 キーワード検索

キーワード検索機能は、プロンプトのタイトル、説明、コンテンツの検索をサポートしています。

- すでにタグが選択されている場合、検索範囲はそれらのタグの下に限定されます
- ログイン後、検索範囲にはコレクションとカスタムプロンプトが含まれます

![](/img/docs/interface-search.png)

## 🔬 プロンプトカード

### 詳細を見る

プロンプトカードをクリックすると、詳細ポップアップが開き、タイトル、説明、完全なプロンプト内容、備考、コピーボタンが表示されます。

![](/img/docs/prompt-card.gif)

### プロンプトをコピー

カードの下部にある「コピー」ボタンをクリックして、プロンプトを取得します。AI モデルに貼り付けて使用してください。

### プロンプトをコレクション

ログイン後、カードにコレクションボタンが表示されます。クリックして、プロンプトをマイコレクションに追加します。

## 🔥 人気順並べ替え

ページは人気の度合いに基づいてプロンプトを並べ替え、トレンドのコンテンツを素早く発見できるようにします。

タグの並べ替えも熱量値に基づいて調整され、コミュニティの最新の好みを反映するようにしています。
