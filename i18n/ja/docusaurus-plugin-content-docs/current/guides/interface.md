# インターフェースガイド

AiShort のホームページは 4 つのパーツで構成されています。上部の**タイトルとエントリーボタン**、その下の**タグフィルター**、**検索バー**、そしてページの大部分を占める**プロンプトカード**です。このセクションでは、これらを組み合わせて目的のプロンプトを見つける方法をご紹介します。

![](/img/docs/interface-home.png)

## 上部エントリーボタン

ページタイトルの下に表示されるボタンは、ログイン前後で少し異なります：

- **ログイン前**：「**無料ログイン**」+「**コミュニティ投稿を見る**」
- **ログイン後**：ビュー切り替え（「**見つける**」 ⇄ 「**マイコレクション**」）、「**プロンプトを作成**」、右側の設定メニュー（「マイアカウント」、「ログアウト」）

> 💡 ログインしなくても、厳選プロンプトの閲覧・検索・コピーはすべて行えます。ログインすると、コレクション・作成・デバイス間同期の機能が追加されます。

## タグフィルタリング

プロンプトの領域や機能でフィルタリングできます。右上の **OR / AND** 切り替えスイッチで複数タグの組み合わせ方を制御します：

- **OR**（デフォルト）：選択したタグのいずれかに一致するプロンプトを表示
- **AND**：選択したすべてのタグに一致するプロンプトのみを表示

タグを選択すると「**絞り込みをクリア**」ボタンが表示され、ワンクリックでリセットできます。モバイルでは、タグはデフォルトで折り畳まれています。「**タグを表示**」をタップすると展開されます。

![](/img/docs/interface-filter.png)

## キーワード検索

プロンプトのタイトル、説明、内容、備考を横断して検索します。

- デスクトップでは **`Ctrl + K`**（macOS は `⌘ + K`）で検索ボックスにフォーカス、`Esc` で終了
- ホームページの検索範囲は**厳選プロンプト**です。タグが選択されている場合、そのタグ内に絞り込まれます
- ログイン後は検索範囲があなたの**コレクション**と**カスタムプロンプト**にも拡張されます
- 短いキーワードでホームページ検索でヒットしない場合は、AiShort が自動的に [コミュニティプロンプト](./community) も検索します

![](/img/docs/interface-search.png)

## プロンプトカード

厳選プロンプトは累計コピー回数で並べ替えられるため、最もよく使われるコンテンツが上位に表示されます。

### カード情報

各カードにはプロンプトのタイトル、説明、内容プレビュー、タグが表示されます。右上の炎アイコンと数字は累計コピー回数です。

### 詳細を見る

カードをクリックすると詳細ポップアップが開き、全文と備考を確認したうえで、ワンクリックでコピーできます。

![](/img/docs/prompt-card.webp)

### プロンプトをコピー

カードまたは詳細ポップアップの「**コピー**」ボタンをクリックし、任意の AI ツール（チャット画面、コーディングツール、API 呼び出しなど）に貼り付けて使用します。

### プロンプトをコレクション

ログイン後、各カードにハートボタンが表示されます。クリックすると [マイコレクション](./my-collection) に追加されます。
