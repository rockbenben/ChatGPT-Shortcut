# インターフェースガイド

このセクションでは、AiShort のホームページを構成する 3 つのエリア、すなわち **タグエリア** (上部のシーンタグ)、**検索エリア** (キーワード入力ボックス)、**プロンプト表示エリア** (下部のカードリスト) について解説し、これらを活用して目的のプロンプトを素早く見つける方法をご紹介します。

![](/img/docs/interface-home.png)

## 🏷︎ タグフィルタリング

プロンプトの領域や機能でフィルタリングできます。右上の **OR / AND** 切り替えスイッチで複数タグのフィルタリングルールを決定します：

- **OR** (デフォルト)：いずれかのタグに一致すれば表示
- **AND**：選択したすべてのタグに同時に一致するもののみ表示

![](/img/docs/interface-filter.png)

## 🔍 キーワード検索

プロンプトのタイトル、説明、内容を検索できます。

- ホームページの検索範囲は**厳選プロンプト** (コミュニティは含まない)。タグが選択されている場合、検索範囲はそれらのタグ内に限定されます
- ログイン後、検索範囲はあなたの**コレクション**と**カスタムプロンプト**にも拡張されます
- 短いキーワードでホームページ検索でヒットしない場合は、[コミュニティプロンプト](./community)ページでもう一度検索してください

![](/img/docs/interface-search.png)

## 🔬 プロンプトカード

### 詳細を見る

カードをクリックすると詳細ポップアップが開き、プロンプトの全文と備考を確認したうえで、ワンクリックでコピーできます。

![](/img/docs/prompt-card.gif)

### プロンプトをコピー

カード下部の「コピー」ボタンをクリックし、あらゆる AI ツール (チャット対話、コーディングツール、API 呼び出しなど) に貼り付けて使用します。

### プロンプトをコレクションに追加

ログイン後はカードにコレクションボタンが表示され、クリックするだけでマイコレクションに追加できます。

## 🔥 人気順並べ替え

プロンプトの人気度に基づいて並べ替え、トレンドのコンテンツを発見できます。タグの順序も人気度に応じて調整され、コミュニティの最新の好みを反映します。
