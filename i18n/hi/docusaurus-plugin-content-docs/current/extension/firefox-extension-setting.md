# Firefox एक्सटेंशन सेटिंग्स

फ़ायरफ़ॉक्स एक्सटेंशन का उपयोग करने से पहले, सेटिंग्स के लिए इन दो चरणों का पालन करने की अनुशंसा की जाती है।

## 1. एक्सटेंशन पिन करें और एक्सेस सेटिंग्स

सबसे पहले, फ़ायरफ़ॉक्स टूलबार पर "ChatGPT Shortcut एक्सटेंशन पिन करें" (Pin to Toolbar) चुनें। इसके बाद, एक्सटेंशन सेंटर में प्रवेश करें, ChatGPT Shortcut एक्सटेंशन प्रविष्टि खोजें और "विकल्प" (Options) चुनें। एक्सटेंशन सेटिंग्स इंटरफ़ेस दर्ज करें। विशिष्ट चरण नीचे दिखाए गए हैं:

![फ़ायरफ़ॉक्स सेटिंग्स](https://img.newzone.top/2023-12-25-05-51-47.png?imageMogr2/format/webp)

## 2. एक्सटेंशन रन को अधिकृत करें

दूसरे, यह सुनिश्चित करने के लिए कि एक्सटेंशन ChatGPT, Gemini, आदि जैसी वेबसाइटों पर ठीक से काम करता है, आपको इन वेबसाइटों पर एक्सटेंशन आइकन पर राइट-क्लिक करना होगा और "इस साइट पर हमेशा अनुमति दें" (Always allow on \*\*\*) चुनना होगा। यह ऑपरेशन निर्दिष्ट डोमेन पर साइडबार जोड़ने के लिए एक्सटेंशन को अनुमति देगा।

![फ़ायरफ़ॉक्स एक्सटेंशन अनुमति](https://img.newzone.top/2023-12-25-05-59-48.png?imageMogr2/format/webp)

सफल स्थापना के बाद, आप [एक्सटेंशन उपयोग ट्यूटोरियल](./usage.md) देख सकते हैं।
