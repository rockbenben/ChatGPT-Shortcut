# Chrome ZIP एक्सटेंशन स्थानीय इंस्टॉलेशन गाइड

> यह गाइड मानती है कि आपने [इंस्टॉलेशन पेज](./README.md) से ZIP पैकेज पहले ही डाउनलोड और अनज़िप कर लिया है।
>
> ⚠️ **दो आम गलतियाँ**:
>
> 1. अनज़िप किए गए फ़ोल्डर को **दीर्घकालिक रूप से रखें** (एक्सटेंशन चलते समय इस डायरेक्ट्री से पढ़ता है) — आगे बढ़ने से पहले इसे किसी स्थायी स्थान पर रखें
> 2. खींचा जाने वाला **पूरा अनज़िप फ़ोल्डर** है (न कि zip फ़ाइल स्वयं, न ही फ़ोल्डर के अंदर की कोई एकल फ़ाइल)

## डेवलपर मोड सक्षम करें

क्रोम का "एक्सटेंशन प्रबंधित करें" पृष्ठ खोलें, और "डेवलपर मोड" चालू करें।

नीचे दिए गए पते को कॉपी करें और ब्राउज़र एड्रेस बार में पेस्ट करें, खोलने के लिए एंटर दबाएं। पृष्ठ के ऊपरी दाएँ कोने में "डेवलपर मोड" चालू करें।

```txt
chrome://extensions
```

## प्लगइन स्थापित करें

स्थापना को पूरा करने के लिए अनज़िप किए गए फ़ोल्डर को एक्सटेंशन पेज पर खींचें।

![](https://img.newzone.top/2024-08-12-22-27-47.png?imageMogr2/format/webp)

सफल स्थापना के बाद, आप [एक्सटेंशन उपयोग ट्यूटोरियल](./usage) देख सकते हैं।
