# Chrome ZIP एक्सटेंशन स्थानीय इंस्टॉलेशन गाइड

## प्लगइन डाउनलोड करें

ChatGPT Shortcut प्लगइन ज़िप इंस्टॉलेशन पैकेज (ChatGPT_Shortcut-zip-3.x.x.zip) डाउनलोड करें और इसे अनज़िप करें। (अनज़िप किए गए फ़ोल्डर को कंप्यूटर पर रखने की आवश्यकता है, इसलिए इसे स्थायी स्थान पर अनज़िप करना याद रखें)

- **GitHub**: [GitHub Release](https://github.com/rockbenben/ChatGPT-Shortcut/releases/latest)
- **घरेलू डाउनलोड**: [Alist Cloud](https://alist.newzone.top:9003/apps/ChatGPT%20Shortcut%20Extension); [Lanzou Cloud](https://wwva.lanzouq.com/b01lsc9vi), पासवर्ड: 1qow

## डेवलपर मोड सक्षम करें

क्रोम का "एक्सटेंशन प्रबंधित करें" पृष्ठ खोलें, और "डेवलपर मोड" चालू करें।

नीचे दिए गए पते को कॉपी करें और ब्राउज़र एड्रेस बार में पेस्ट करें, खोलने के लिए एंटर दबाएं। पृष्ठ के ऊपरी दाएँ कोने में "डेवलपर मोड" चालू करें।

```txt
chrome://extensions
```

## प्लगइन स्थापित करें

स्थापना को पूरा करने के लिए अनज़िप किए गए फ़ोल्डर को एक्सटेंशन पेज पर खींचें।

![](https://img.newzone.top/2024-08-12-22-27-47.png?imageMogr2/format/webp)

सफल स्थापना के बाद, आप [एक्सटेंशन उपयोग ट्यूटोरियल](./usage.md) देख सकते हैं।
