# কাস্টম প্রম্পট

লগইন করার পরে, আপনি আপনার নিজস্ব প্রম্পট তৈরি করতে পারেন, অ্যাকাউন্টে সংরক্ষণ করে যেকোনো সময় কল করতে পারেন। কাস্টম প্রম্পটগুলি "আমার সংগ্রহ" ভিউতে প্রদর্শিত হবে।

## প্রম্পট তৈরি করুন

1. উপরের ডানদিকে "যোগ করুন" বাটনে ক্লিক করুন
2. পপআপ "প্রম্পট তৈরি করুন" উইন্ডোতে ফর্ম পূরণ করুন:
   - **প্রম্পট নাম** (আবশ্যক): প্রম্পটের জন্য একটি নাম দিন
   - **প্রম্পট বিষয়বস্তু** (আবশ্যক): প্রম্পটের মূল টেক্সট
   - **কাজ/উদ্দেশ্য** (ঐচ্ছিক): প্রম্পট কী করতে পারে তার সংক্ষিপ্ত বিবরণ
   - **মন্তব্য** (ঐচ্ছিক): উৎস, অন্যান্য ভাষার সংস্করণ বা পরিপূরক তথ্য
3. নিচের "এই প্রম্পটটি পাবলিক পৃষ্ঠায় শেয়ার করতে ইচ্ছুক?" সুইচ ডিফল্টভাবে চালু — বন্ধ করলে শুধুমাত্র আপনার কাছে দৃশ্যমান
4. "প্রম্পট তৈরি করুন" ক্লিক করে জমা দিন

![প্রম্পট তৈরি পপআপ](/img/docs/user-prompts-create.png)

> 💡 **পূরণের উদাহরণ** — একটি "পেশাদার ইমেইল সম্পাদক" প্রম্পট:
> - **প্রম্পট নাম**: পেশাদার ইমেইল সম্পাদক
> - **প্রম্পট বিষয়বস্তু**: আপনি একজন অভিজ্ঞ যোগাযোগ পরামর্শদাতা। অনুগ্রহ করে নিম্নলিখিত খসড়াটি একটি সংক্ষিপ্ত, ভদ্র এবং পেশাদার ইংরেজি ইমেইলে রূপান্তর করুন, যেখানে স্পষ্ট বিষয় (Subject), উপযুক্ত সম্ভাষণ এবং সুনির্দিষ্ট কর্মপ্রস্তাব (Call to Action) থাকবে: [আপনার খসড়া বা মূল বার্তা]
> - **কাজ/উদ্দেশ্য**: অনানুষ্ঠানিক নোটকে ক্লায়েন্ট বা সহকর্মীদের কাছে পাঠানোর উপযোগী পেশাদার ইমেইলে পরিণত করুন
> - **মন্তব্য**: বাংলায় উত্তর চাইলে শেষে "respond in Bengali" যোগ করুন

## প্রম্পট সম্পাদনা করুন

আমার সংগ্রহ ভিউতে, আপনার তৈরি করা প্রম্পট কার্ডে ক্লিক করে "প্রম্পট সম্পাদনা করুন" উইন্ডো খুলুন:

- নাম, বিষয়বস্তু, কাজ, মন্তব্য সংশোধন করুন
- শেয়ারিং স্থিতি (সর্বজনীন/ব্যক্তিগত) পরিবর্তন করুন
- "সংরক্ষণ করুন" ক্লিক করে আপডেট করুন

![প্রম্পট সম্পাদনা ইন্টারফেস](/img/docs/user-prompts-edit.png)

## প্রম্পট মুছুন

সম্পাদনা উইন্ডোতে "মুছুন" এ ক্লিক করুন। একবার মুছে ফেললে আর ফেরানো যাবে না, তাই সাবধানে করুন।

## কমিউনিটিতে শেয়ার করুন

তৈরি বা সম্পাদনা করার সময়, নিচের "এই প্রম্পটটি পাবলিক পৃষ্ঠায় শেয়ার করতে ইচ্ছুক?" সুইচ দৃশ্যমানতা নিয়ন্ত্রণ করে:

- **চালু (ডিফল্ট)**: প্রম্পটটি [কমিউনিটি প্রম্পট](./community) পৃষ্ঠায় প্রদর্শিত হয়, অন্যান্য ব্যবহারকারীরা দেখতে এবং সংগ্রহ করতে পারে
- **বন্ধ**: ব্যক্তিগত প্রম্পট, শুধুমাত্র আপনার কাছে দৃশ্যমান

শেয়ারিং স্থিতি যেকোনো সময় পরিবর্তন করা যেতে পারে।

## প্রম্পট এক্সপোর্ট করুন

1. "আমার অ্যাকাউন্ট" → "ডেটা ব্যবস্থাপনা → প্রম্পট এক্সপোর্ট করুন" বিভাগটি খুঁজুন
2. "ডেটা এক্সপোর্ট করুন" বাটনে ক্লিক করুন
3. সিস্টেম JSON ফাইল তৈরি করে স্বয়ংক্রিয়ভাবে ডাউনলোড করে

এক্সপোর্টে অন্তর্ভুক্ত:

- প্রম্পটের নাম, বিষয়বস্তু, কাজ/উদ্দেশ্য, মন্তব্য
- তৈরি এবং আপডেটের সময়
- শেয়ারিং স্থিতি

ডেটা ক্ষতি রোধ করতে নিয়মিত ব্যাকআপ এক্সপোর্ট করার পরামর্শ দেওয়া হয়।

## প্রম্পট ইম্পোর্ট করুন

JSON ফাইল থেকে প্রম্পট এবং সংগ্রহ ইম্পোর্ট করুন:

1. "আমার অ্যাকাউন্ট" → "ডেটা ব্যবস্থাপনা → প্রম্পট ইম্পোর্ট করুন" বিভাগটি খুঁজুন
2. "ডেটা ইম্পোর্ট করুন" বাটনে ক্লিক করুন
3. JSON ফাইল নির্বাচন করুন
4. সিস্টেম স্বয়ংক্রিয়ভাবে ডেটা মার্জ করে (শিরোনাম দ্বারা ডিডুপ্লিকেট; অন্যদের ID-এর প্রম্পট স্বয়ংক্রিয়ভাবে ব্যক্তিগত হিসাবে সেট করা হয়)

### টিম সহযোগিতা

একটি টিমের মধ্যে প্রম্পট শেয়ার করার প্রস্তাবিত প্রক্রিয়া:

1. **ফিল্টার এবং শেয়ার**: এক্সপোর্ট করার পরে ম্যানুয়ালি সংগ্রহ তালিকা মুছুন বা শেয়ার করার জন্য প্রম্পট ফিল্টার করুন, ফাইলটি টিম সদস্যদের ইম্পোর্ট করার জন্য পাঠান
2. **গোপনীয়তা সুরক্ষা**: অন্যদের কাছ থেকে শেয়ার করা প্রম্পট ইম্পোর্ট (যাদের ID বর্তমান অ্যাকাউন্টের নয়) করলে স্বয়ংক্রিয়ভাবে **ব্যক্তিগত** হিসাবে সেট হয়, প্রতিটি সদস্যের ডেটা পরস্পর হস্তক্ষেপমুক্ত
