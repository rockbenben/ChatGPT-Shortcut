# ইন্টারফেস ব্যাখ্যা

এই বিভাগটি AiShort হোমপেজের তিনটি প্রধান এলাকা পরিচয় করিয়ে দেয় — **ট্যাগ এলাকা** (শীর্ষে পরিস্থিতি ট্যাগ), **অনুসন্ধান এলাকা** (কীওয়ার্ড ইনপুট বক্স), **প্রম্পট প্রদর্শন এলাকা** (নিচে কার্ড তালিকা) — এবং কীভাবে সেগুলি ব্যবহার করে দ্রুত পছন্দসই প্রম্পট খুঁজে পাবেন।

![](/img/docs/interface-home.png)

## 🏷︎ ট্যাগ ফিল্টারিং

প্রম্পটের ডোমেইন এবং ফাংশন দ্বারা ফিল্টার করুন। উপরের ডানদিকে **OR / AND** সুইচার একাধিক ট্যাগের ফিল্টারিং নিয়ম নির্ধারণ করে:

- **OR** (ডিফল্ট): যেকোনো একটি ট্যাগ মিললেই প্রদর্শিত হয়
- **AND**: সমস্ত ট্যাগ একসাথে মিলতে হবে

![](/img/docs/interface-filter.png)

## 🔍 কীওয়ার্ড অনুসন্ধান

প্রম্পটের শিরোনাম, বিবরণ ও বিষয়বস্তুতে অনুসন্ধান করা যায়।

- হোমপেজ অনুসন্ধানের পরিধি হল **নির্বাচিত প্রম্পট** (কমিউনিটি বাদে)। ট্যাগ নির্বাচিত থাকলে, অনুসন্ধানের পরিধি সেই ট্যাগগুলির মধ্যে সীমাবদ্ধ থাকে
- লগইন করার পরে, অনুসন্ধানের পরিধি আপনার **সংগ্রহ** এবং **কাস্টম প্রম্পট** এ প্রসারিত হয়
- ছোট কীওয়ার্ড ব্যবহার করে হোমপেজে কিছু না পেলে, [কমিউনিটি প্রম্পট](./community) পৃষ্ঠায় আবার অনুসন্ধান করুন

![](/img/docs/interface-search.png)

## 🔬 প্রম্পট কার্ড

### বিবরণ দেখুন

কার্ডে ক্লিক করে বিবরণ পপ-আপ উইন্ডো খুলুন — সম্পূর্ণ বিষয়বস্তু, মন্তব্য দেখুন এবং এক ক্লিকে কপি করুন।

![](/img/docs/prompt-card.gif)

### প্রম্পট কপি করুন

কার্ডের নিচে "কপি" বাটনে ক্লিক করুন, যেকোনো AI টুলে পেস্ট করে ব্যবহার করুন (ডায়ালগ পৃষ্ঠা, কোডিং টুল, API কল ইত্যাদি সব পরিস্থিতিতেই কাজ করে)।

### প্রম্পট সংগ্রহ করুন

লগইন করার পরে, কার্ডে সংগ্রহ বাটন প্রদর্শিত হবে, ক্লিক করে আমার সংগ্রহে যোগ করুন।

## 🔥 জনপ্রিয়তা বাছাই

প্রম্পটগুলোকে জনপ্রিয়তা অনুসারে সাজিয়ে আপনাকে ট্রেন্ডিং বিষয়বস্তু খুঁজে পেতে সাহায্য করে। ট্যাগের ক্রমও জনপ্রিয়তা অনুযায়ী বদলায়, যা কমিউনিটির সাম্প্রতিক পছন্দ তুলে ধরে।
