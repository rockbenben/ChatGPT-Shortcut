# Chrome CRX এক্সটেনশন স্থানীয় ইনস্টলেশন গাইড

## CRX ইনস্টলেশন প্যাকেজ ডাউনলোড করুন

ChatGPT Shortcut crx ইনস্টলেশন প্যাকেজ (ChatGPT_Shortcut-crx-3.x.x.zip) ডাউনলোড করুন, এবং এটি আনজিপ করুন (crx ফাইলটি আনজিপ করা ফোল্ডারে রয়েছে)।

- **GitHub**: [GitHub Release](https://github.com/rockbenben/ChatGPT-Shortcut/releases/latest)
- **ঘরোয়া ডাউনলোড**: [Alist Cloud](https://alist.newzone.top:9003/apps/ChatGPT%20Shortcut%20Extension); [Lanzou Cloud](https://wwva.lanzouq.com/b01lsc9vi), পাসওয়ার্ড: 1qow

![](https://img.newzone.top/2024-08-12-21-47-10.png?imageMogr2/format/webp)

## ডেভেলপার মোড সক্ষম করুন

ক্রোমের "এক্সটেনশন পরিচালনা করুন" পৃষ্ঠা খুলুন, এবং "ডেভেলপার মোড" চালু করুন।

নিচের ঠিকানাটি কপি করুন এবং ব্রাউজার অ্যাড্রেস বারে পেস্ট করুন, খুলতে এন্টার টিপুন। পৃষ্ঠার উপরের ডান কোণে "ডেভেলপার মোড" চালু করুন।

```txt
chrome://extensions
```

![](https://img.newzone.top/2024-08-12-22-05-52.png?imageMogr2/format/webp)

## প্লাগইন ইনস্টল করুন

ChatGPT Shortcut প্লাগইন ইনস্টল করুন (নোট ⚠️: আপনাকে .crx ফাইলটি টেনে আনতে হবে, [আনপ্যাক করা লোড করুন] এ ক্লিক করবেন না)

![](https://img.newzone.top/2024-08-12-22-16-38.png?imageMogr2/format/webp)

সফল ইনস্টলেশনের পরে, আপনি [এক্সটেনশন ব্যবহার টিউটোরিয়াল](./usage.md) দেখতে পারেন।

## ইনস্টলেশনে সমস্যা হচ্ছে?

১. উইন্ডোজ ব্যবহারকারীরা, দয়া করে পরীক্ষা করুন যে আপনি ডাউনলোড করা ইনস্টলেশন প্যাকেজটি আনজিপ করেছেন কি না (ডাবল-ক্লিক করার পরিবর্তে)?

২. "ডেভেলপার মোড" কি চালু আছে? যদি না হয়, তবে দয়া করে দ্বিতীয় ধাপটি দেখুন।

৩. আপনি কি crx ফাইলটি "এক্সটেনশন" পৃষ্ঠায় টেনে এনেছিলেন? নোট ⚠️: [আনপ্যাক করা লোড করুন] এ ক্লিক করবেন না, আপনাকে crx ফাইলটি টেনে আনতে হবে।

৪. ব্রাউজার crx ফাইল ইনস্টল করার অনুমতি দিচ্ছে না? জিপ ফাইল ইনস্টল করার চেষ্টা করুন! [জিপ ইনস্টলেশন টিউটোরিয়াল দেখতে এখানে ক্লিক করুন](./manual-chrome-extension-zip.md)।
