# Chrome ZIP এক্সটেনশন স্থানীয় ইনস্টলেশন গাইড

> এই গাইডটি ধরে নিচ্ছে যে আপনি ইতিমধ্যেই [ইনস্টলেশন পৃষ্ঠা](./README.md) থেকে ZIP প্যাকেজটি ডাউনলোড ও আনজিপ করেছেন।
>
> ⚠️ **দুটি সাধারণ ভুল**:
> ১. আনজিপ করা ফোল্ডারটি **দীর্ঘমেয়াদে রাখতে হবে** (এক্সটেনশনটি রানটাইমে এই ডিরেক্টরি থেকে পড়ে) — চালিয়ে যাওয়ার আগে এটি একটি স্থায়ী স্থানে রাখুন
> ২. টেনে আনতে হবে **পুরো আনজিপ করা ফোল্ডার** (zip ফাইল নিজে নয়, ফোল্ডারের ভেতরের একক ফাইলও নয়)

## ডেভেলপার মোড সক্ষম করুন

ক্রোমের "এক্সটেনশন পরিচালনা করুন" পৃষ্ঠা খুলুন, এবং "ডেভেলপার মোড" চালু করুন।

নিচের ঠিকানাটি কপি করুন এবং ব্রাউজার অ্যাড্রেস বারে পেস্ট করুন, খুলতে এন্টার টিপুন। পৃষ্ঠার উপরের ডান কোণে "ডেভেলপার মোড" চালু করুন।

```txt
chrome://extensions
```

## প্লাগইন ইনস্টল করুন

ইনস্টলেশন সম্পূর্ণ করতে আনজিপ করা ফোল্ডারটি এক্সটেনশন পৃষ্ঠায় টেনে আনুন।

![](https://img.newzone.top/2024-08-12-22-27-47.png?imageMogr2/format/webp)

সফল ইনস্টলেশনের পরে, আপনি [এক্সটেনশন ব্যবহার টিউটোরিয়াল](./usage) দেখতে পারেন।
