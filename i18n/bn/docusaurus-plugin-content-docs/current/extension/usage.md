# ব্রাউজার এক্সটেনশন কীভাবে ব্যবহার করবেন?

ChatGPT Shortcut ১৩টি মূলধারার ভাষা সমর্থন করে, এবং এক্সটেনশন ভাষা আপনার ব্রাউজার পরিবেশের উপর ভিত্তি করে স্বয়ংক্রিয়ভাবে সেট হবে। এক্সটেনশনে এমবেডেড ChatGPT পৃষ্ঠা এবং সাইডবার ভাষাও এই সেটিং অনুসরণ করবে। দয়া করে মনে রাখবেন যে তৃতীয় পক্ষের সাইট থেকে অনুমতি সতর্কতা ট্রিগার করা এড়াতে, **এমবেডেড পৃষ্ঠায় ভাষা পরিবর্তন করবেন না**।

![](/img/docs/extension-settings.png)

## AiShort সাইডবার

AiShort এক্সটেনশন বর্তমানে একটি সাইড প্যানেল (Side Panel) হিসাবে কাজ করে, যা একটি মসৃণ ইন্টারেক্টিভ অভিজ্ঞতা প্রদান করে।

যদি সেটিংসে "সাইড প্যানেল খোলা রাখুন (Keep side panel open)" বিকল্পটি সক্ষম করা থাকে, তবে ব্রাউজার ট্যাব স্যুইচ করার সময় সাইডবারটি খোলা থাকবে।

![](/img/docs/extension-sidebar.png)

যদি আপনি অন্য ওয়েবসাইটগুলিতে AiShort সাইডবার ব্যবহার করতে চান, তবে দয়া করে [ChatGPT Shortcut Anywhere Script](https://greasyfork.org/scripts/482907-chatgpt-shortcut-anywhere) ইনস্টল করুন। ট্যাম্পারমাঙ্কি একটি ব্রাউজার এক্সটেনশন যা ব্যবহারকারীদের ওয়েব পৃষ্ঠা সংশোধন করতে কাস্টম স্ক্রিপ্ট চালানোর অনুমতি দেয়। দয়া করে মনে রাখবেন যে ফিচার ওভারল্যাপ এবং সম্ভাব্য দ্বন্দ্ব এড়াতে, ChatGPT Shortcut Anywhere স্ক্রিপ্ট ডিফল্টরূপে এক্সটেনশন দ্বারা সমর্থিত ওয়েবসাইটগুলিতে কার্যকর হবে না।

## হটকি অ্যাক্টিভেশন

AiShort সাইডবারটি সরাসরি সক্রিয় করতে `Alt+Shift+S` শর্টকাট ব্যবহার করুন।
