# ব্রাউজার এক্সটেনশন ব্যবহার কিভাবে করবেন?

ChatGPT Shortcut 18টি প্রধান ভাষা সমর্থন করে। এক্সটেনশনের ভাষা আপনার ব্রাউজার অনুযায়ী স্বয়ংক্রিয়ভাবে সেট হয়। আপনি সেটিংসে ম্যানুয়ালি পরিবর্তন করতে পারেন।

![](/img/docs/extension-settings.png)

## ডিসপ্লে মোড

এক্সটেনশন তিনটি ডিসপ্লে মোড সমর্থন করে, আপনার ব্যবহারের অভ্যাস অনুসারে একটি বেছে নিন:

- **সাইডবার মোড**: ব্রাউজার সাইডবারে খোলে — ওয়েব পৃষ্ঠার পাশে প্রদর্শিত হয়, AI ডায়ালগ পৃষ্ঠা দেখার পাশাপাশি প্রম্পট ব্রাউজ করার জন্য উপযুক্ত
- **পপআপ মোড**: এক্সটেনশন আইকনে ক্লিক করলে ছোট উইন্ডো পপআপ হয় — দ্রুত দেখে বন্ধ করুন, স্ক্রিন দখল করে না
- **স্বতন্ত্র উইন্ডো মোড**: একটি স্বাধীন ফ্লোটিং উইন্ডোতে খোলে — সেকেন্ডারি স্ক্রিনে টেনে নিতে বা আলাদাভাবে রাখা যায়

## সাইড প্যানেল খোলা রাখুন

সক্রিয় থাকলে, ব্রাউজার ট্যাব পরিবর্তন করলেও সাইডবার খোলা থাকে।

## ডার্ক মোড

এক্সটেনশন ডার্ক মোড সমর্থন করে। ইন্টারফেস ডার্ক অ্যাপিয়ারেন্সে পরিবর্তন করতে সেটিংসে "ডার্ক মোড" সক্রিয় করুন।

![](/img/docs/extension-sidebar.webp)

## হটকি

এক্সটেনশন সরাসরি সক্রিয় করতে `Alt+Shift+S` শর্টকাট ব্যবহার করুন।
