# ব্রাউজার এক্সটেনশন কীভাবে ব্যবহার করবেন?

ChatGPT Shortcut 13টি প্রধান ভাষা সমর্থন করে, এবং এক্সটেনশন ভাষা আপনার ব্রাউজার পরিবেশ অনুযায়ী স্বয়ংক্রিয়ভাবে সেট হবে। আপনি সেটিংস পেজে ম্যানুয়ালি ভাষা পরিবর্তন করতে পারেন।

![](/img/docs/extension-settings.png)

## AiShort সাইডবার

AiShort এক্সটেনশন বর্তমানে সাইড প্যানেল হিসেবে কাজ করে, একটি নির্বিঘ্ন ইন্টারেক্টিভ অভিজ্ঞতা প্রদান করে।

সেটিংসে "সাইড প্যানেল খোলা রাখুন" সক্রিয় থাকলে, ব্রাউজার ট্যাব পরিবর্তন করলেও সাইডবার খোলা থাকবে।

![](/img/docs/extension-sidebar.png)

## ডার্ক মোড

এক্সটেনশন ডার্ক মোড সমর্থন করে। সাইডবার এবং সেটিংস পেজকে ডার্ক অ্যাপিয়ারেন্সে স্যুইচ করতে সেটিংস পেজে "ডার্ক মোড" সক্রিয় করুন।

## হটকি অ্যাক্টিভেশন

AiShort সাইডবার সরাসরি সক্রিয় করতে `Alt+Shift+S` শর্টকাট ব্যবহার করুন।
