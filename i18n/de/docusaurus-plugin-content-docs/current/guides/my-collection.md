# Meine Sammlung

Nach der Anmeldung können Sie in der Ansicht **Meine Sammlung** Ihre Lieblings-Prompts wie Lesezeichen organisieren — per Drag-and-Drop sortieren, benutzerdefinierte Tags hinzufügen und nach Tags filtern, um sie schnell wiederzufinden.

:::tip Online- und Offline-Version
In der Online-Version werden Sammlungsdaten auf dem Server gespeichert (Anmeldung erforderlich). In der [Offline-Version](./offline) werden sie ausschließlich lokal im Browser (localStorage) abgelegt — keine Anmeldung, keine Serverübertragung, ideal für DSGVO-sensible Unternehmens- und Behördenumgebungen. Die Funktionen sind in beiden Varianten identisch.
:::

## Ansicht wechseln

Oben auf der Startseite befindet sich ein Ansichts-Umschalter mit zwei Modi:

- **Meine Sammlung**: Zeigt Ihre gesammelten und benutzerdefinierten Prompts an
- **Prompt-Bibliothek**: Durchsuchen Sie die öffentliche Prompt-Bibliothek

Ihre Auswahl bleibt erhalten — beim nächsten Besuch öffnet sich automatisch die zuletzt verwendete Ansicht.

![Ansicht Meine Sammlung](/img/docs/my-collection-view.gif)

## Zur Sammlung hinzufügen

Klicken Sie auf einer Karte auf die Schaltfläche "Sammeln", um sie zu Meine Sammlung hinzuzufügen.

Beim Sammeln eines Community-Prompts wird automatisch eine positive Stimme abgegeben, was hochwertigen Inhalten zu mehr Sichtbarkeit verhilft.

## Aus Sammlung entfernen

Klicken Sie in der Ansicht Meine Sammlung auf das Herz-Symbol der Karte, um sie aus der Sammlung zu entfernen.

## Benutzerdefinierte Tags

Versehen Sie gesammelte Prompts mit Tags, um sie übersichtlich einzuordnen:

### Tag hinzufügen

1. Klicken Sie auf das Tag-Symbol in der Steuerleiste
2. Wählen Sie ein vorhandenes Tag oder klicken Sie auf "+", um ein neues zu erstellen
3. Neue Tags können einen eigenen Namen und eine eigene Farbe haben

### Tags verwalten

Klicken Sie auf "Tags verwalten", um die Verwaltungsoberfläche zu öffnen:

- Tag-Namen ändern
- Tag-Farbe ändern (11 voreingestellte Farben)
- Tags löschen

![Tag-Verwaltungs-Popup](/img/docs/my-collection-tag.png)

### Nach Tag filtern

In der Filterleiste der Ansicht Meine Sammlung können Sie nach benutzerdefinierten Tags filtern.

## Drag-and-Drop-Sortierung

In der Ansicht Meine Sammlung können Sie die Kartenreihenfolge per Drag-and-Drop anpassen:

1. Ziehen Sie den Griff links neben der Karte
2. Bewegen Sie sie zur Zielposition und lassen Sie los
3. Die neue Reihenfolge wird automatisch gespeichert (Online: auf dem Server, Offline: lokal im Browser)

Die Sortierung ist nur in der Ansicht Meine Sammlung verfügbar.

![Drag-and-Drop-Sortierung](/img/docs/my-collection-drag.gif)
