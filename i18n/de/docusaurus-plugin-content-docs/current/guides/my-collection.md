# Meine Sammlung

Nach der Anmeldung können Sie Ihre Lieblingsprompts sammeln und persönlich verwalten. Die Ansicht "Meine Sammlung" bietet umfangreiche Verwaltungsfunktionen, die Ihnen helfen, Prompts effizient zu organisieren und zu nutzen.

## Ansicht wechseln

Oben auf der Startseite befinden sich Schaltflächen zum Wechseln der Ansicht, die zwei Modi unterstützen:

- **Meine Sammlung**: Zeigt Ihre gesammelten und benutzerdefinierten Prompts an
- **Prompt-Bibliothek**: Durchsuchen Sie die öffentliche Prompt-Bibliothek, um neue Inhalte zu entdecken

Der Browser merkt sich automatisch Ihre Ansichtsauswahl und wechselt bei Ihrem nächsten Besuch zur letzten Ansicht.

![Ansicht Meine Sammlung](/img/docs/my-collection-view.gif)

## Zur Sammlung hinzufügen

Klicken Sie auf die Schaltfläche "Sammeln" auf einer Prompt-Karte, um sie zu "Meine Sammlung" hinzuzufügen.

Wenn Sie einen Community-Prompt sammeln, gibt das System automatisch eine positive Stimme ab, was hochwertigen Inhalten zu mehr Sichtbarkeit verhilft.

## Aus Sammlung entfernen

Klicken Sie in der Ansicht "Meine Sammlung" auf das Herz-Symbol auf der Karte, um sie aus der Sammlung zu entfernen.

## Benutzerdefinierte Tags

Fügen Sie gesammelten Prompts persönliche Tags für eine bequeme Klassifizierungsverwaltung hinzu:

### Tag hinzufügen

1. Klicken Sie auf das Tag-Symbol in der Steuerleiste
2. Wählen Sie ein vorhandenes Tag aus oder klicken Sie auf "+", um ein neues Tag zu erstellen
3. Neue Tags können benutzerdefinierte Namen und Farben haben

### Tags verwalten

Klicken Sie auf "Tags verwalten", um die Tag-Verwaltungsoberfläche aufzurufen:

- Tag-Namen ändern
- Tag-Farbe ändern (11 voreingestellte Farben verfügbar)
- Unerwünschte Tags löschen

![Tag-Verwaltungs-Popup](/img/docs/my-collection-tag.png)

### Nach Tag filtern

In der Filterleiste der Ansicht "Meine Sammlung" können Sie Inhalte nach benutzerdefinierten Tags filtern.

## Drag-and-Drop-Sortierung

Das Anpassen der Kartenreihenfolge durch Drag-and-Drop wird in der Ansicht "Meine Sammlung" unterstützt:

1. Ziehen Sie den Griff links neben der Karte
2. Bewegen Sie ihn zur Zielposition und lassen Sie los
3. Die neue Reihenfolge wird automatisch auf dem Server gespeichert

Die Sortierfunktion ist nur in der Ansicht "Meine Sammlung" verfügbar.

![Drag-and-Drop-Sortierung](/img/docs/my-collection-drag.gif)
