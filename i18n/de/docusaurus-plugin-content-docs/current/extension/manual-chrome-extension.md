# Chrome CRX Erweiterung Lokale Installationsanleitung

## CRX-Installationspaket herunterladen

Laden Sie das ChatGPT Shortcut crx-Installationspaket (ChatGPT_Shortcut-crx-3.x.x.zip) herunter und entpacken Sie es (die crx-Datei befindet sich im entpackten Ordner).

- **GitHub**: [GitHub Release](https://github.com/rockbenben/ChatGPT-Shortcut/releases/latest)
- **China Download**: [Alist Drive](https://alist.newzone.top:9003/apps/ChatGPT%20Shortcut%20Extension); [Lanzou Cloud](https://wwva.lanzouq.com/b01lsc9vi), Passwort: 1qow

![](https://img.newzone.top/2024-08-12-21-47-10.png?imageMogr2/format/webp)

## Entwicklermodus aktivieren

Öffnen Sie die Chrome-Seite "Erweiterungen verwalten" und schalten Sie den "Entwicklermodus" ein.

Kopieren Sie die folgende Adresse und fügen Sie sie in die Adressleiste des Browsers ein, drücken Sie dann die Eingabetaste, um sie zu öffnen. Schalten Sie den "Entwicklermodus" in der oberen rechten Ecke der Seite ein.

```txt
chrome://extensions
```

![](https://img.newzone.top/2024-08-12-22-05-52.png?imageMogr2/format/webp)

## Erweiterung installieren

Installieren Sie die ChatGPT Shortcut Erweiterung (Hinweis ⚠️: Sie müssen die .crx-Datei ziehen, klicken Sie NICHT auf "Entpackte Erweiterung laden")

![](https://img.newzone.top/2024-08-12-22-16-38.png?imageMogr2/format/webp)

Nach erfolgreicher Installation können Sie das [Plugin-Nutzungs-Tutorial](./usage.md) ansehen.

## Installationsprobleme?

1. Windows-Benutzer: Haben Sie das heruntergeladene Installationspaket entpackt (statt es per Doppelklick zu öffnen)?

2. Ist der "Entwicklermodus" eingeschaltet? Wenn nicht, siehe Operation in Schritt 2.

3. Haben Sie die crx-Datei auf die "Erweiterungen"-Seite gezogen? Hinweis ⚠️: Klicken Sie nicht auf "Entpackte Erweiterung laden", Sie müssen die crx-Datei ziehen.

4. Erlaubt der Browser die Installation von crx-Dateien nicht? Versuchen Sie, die zip-Datei zu installieren! [Klicken Sie hier, um das Zip-Installations-Tutorial anzusehen](./manual-chrome-extension-zip.md).
