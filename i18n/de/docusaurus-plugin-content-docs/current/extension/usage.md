# Wie verwende ich die Browsererweiterung?

ChatGPT Shortcut unterstützt 13 gängige Sprachen und die Sprache der Erweiterung wird automatisch entsprechend Ihrer Browserumgebung eingestellt. Die Sprache der in ChatGPT integrierten Seite und Seitenleiste in der Erweiterung folgt ebenfalls dieser Einstellung. Bitte beachten Sie: Um keine Berechtigungswarnungen von Websites Dritter auszulösen, sollten Sie die Sprache in eingebetteten Seiten nicht direkt ändern.

![](https://img.newzone.top/2023-12-23-12-04-29.png?imageMogr2/format/webp)

## Anzeigeeinstellungen

### AiShort-Seitenleiste

Nachdem Sie die AiShort-Seitenleiste aktiviert haben, sehen Sie unten rechts auf den unterstützten Webseiten einen grünen Symbolschalter. Durch Klicken auf dieses Symbol können Sie die Seitenleiste ein- oder ausschalten. Derzeit werden standardmäßig ChatGPT, Bard, Claude und Wenxin Yiyan unterstützt. Wenn Sie „Seitenleiste automatisch aktivieren“ wählen, wird die Seitenleiste automatisch geöffnet, wenn Sie diese unterstützten Websites besuchen.

![](https://img.newzone.top/2023-12-23-04-16-15.gif?imageMogr2/format/webp)

Wenn Sie die AiShort-Seitenleiste auf anderen Websites verwenden möchten, installieren Sie bitte das Skript [ChatGPT Shortcut Anywhere](https://greasyfork.org/scripts/482907-chatgpt-shortcut-anywhere). Bitte beachten Sie, dass das Skript ChatGPT Shortcut Anywhere auf Websites, die von der Erweiterung nativ unterstützt werden, nicht wirksam ist, um überlappende Funktionen und potenzielle Konflikte zu vermeiden.

### Integrierte Homepage

Nachdem Sie die integrierte Homepage-Funktion aktiviert haben, wird in der oberen linken Ecke der ChatGPT-Webversion eine Schaltfläche für die integrierte Seite angezeigt. Wenn Sie darauf klicken, wird die ChatGPT-App-Oberfläche durch die AiShort-Seite ersetzt.

![](https://img.newzone.top/ai/2023-12-22-19-40-15.png?imageMogr2/format/webp)

## Separater Fenstermodus

Wenn der separate Fenstermodus aktiviert ist, wird die Erweiterungsoberfläche dauerhaft in einem separaten Fenster angezeigt, was Multitasking erleichtert. Sie können in den Erweiterungseinstellungen Websites für die automatische Aktivierung festlegen. Wenn der Browser diese Websites besucht, wird das AiShort-Fenster automatisch aktiviert.

![](https://img.newzone.top/2023-12-23-12-07-09.png?imageMogr2/format/webp)

### Websites mit automatischer Aktivierung

In den Erweiterungseinstellungen können Sie angeben, welche Websites das separate AiShort-Fenster automatisch aktivieren.

![](https://img.newzone.top/2023-12-23-12-09-51.png?imageMogr2/format/webp)

### Hotkey-Aktivierung

Mit der Tastenkombination „Alt+Umschalt+S“ können Sie das AiShort-Fenster direkt aktivieren, egal ob im Popup-Modus oder im separaten Fenstermodus.
