# Prompt Tùy chỉnh

Sau khi đăng nhập, bạn có thể tạo prompt của riêng mình để truy cập nhanh sau này. Prompt tùy chỉnh sẽ được lưu vào tài khoản của bạn và hiển thị trong chế độ xem Bộ sưu tập của tôi.

## Tạo Prompt

1. Nhấp nút "Thêm Prompt" ở góc trên bên phải trang
2. Điền thông tin prompt:
   - **Tiêu đề**: Mô tả ngắn gọn mục đích của prompt
   - **Mô tả**: Nội dung đầy đủ của prompt
   - **Ghi chú**: Hướng dẫn sử dụng hoặc ghi chú (tùy chọn)
3. Chọn có chia sẻ lên cộng đồng hay không
4. Nhấp Gửi

![Popup Thêm Prompt](/img/docs/user-prompts-create.png)

## Chỉnh sửa Prompt

Trong chế độ xem Bộ sưu tập của tôi, nhấp vào thẻ prompt bạn đã tạo để:

- Sửa đổi tiêu đề, mô tả và ghi chú
- Chuyển đổi trạng thái chia sẻ (Công khai/Riêng tư)
- Lưu thay đổi

![Giao diện Chỉnh sửa Prompt](/img/docs/user-prompts-edit.png)

## Xóa Prompt

Nhấp nút "Xóa" trong giao diện chỉnh sửa để xóa prompt. Việc xóa không thể hoàn tác, vui lòng thao tác cẩn thận.

## Chia sẻ lên Cộng đồng

Khi tạo hoặc chỉnh sửa prompt, đánh dấu tùy chọn "Chia sẻ lên Cộng đồng":

- **Công khai**: Prompt sẽ được hiển thị trên trang cộng đồng, người dùng khác có thể xem và thu thập
- **Riêng tư**: Chỉ hiển thị với bạn, sẽ không xuất hiện trên trang cộng đồng

Bạn có thể sửa đổi trạng thái chia sẻ bất cứ lúc nào.

## Xuất Bản sao lưu

Xuất tất cả prompt do bạn tạo:

1. Vào Trung tâm Người dùng
2. Nhấp nút "Xuất Prompt"
3. Hệ thống tạo tệp JSON và tự động tải xuống

Nội dung xuất bao gồm:

- Tiêu đề, mô tả, ghi chú prompt
- Thời gian tạo và cập nhật
- Trạng thái chia sẻ

Khuyến nghị xuất bản sao lưu thường xuyên để tránh mất dữ liệu.

## Nhập Prompt

Nhập prompt và mục yêu thích từ tệp JSON:

1. Vào Trung tâm Người dùng
2. Nhấp nút "Nhập Prompt"
3. Chọn tệp JSON
4. Hệ thống tự động hợp nhất dữ liệu
