# Bắt đầu

Prompt tốt quyết định chất lượng đầu ra của AI. AiShort tuyển chọn một lượng lớn prompt thực dụng, sao chép-dán là dùng được.

> 📋 **Trước khi bắt đầu, bạn cần**: Một trình duyệt + bất kỳ công cụ AI nào để dán prompt vào — trang chat (ChatGPT, Gemini, Claude...), công cụ lập trình (Cursor, Copilot), agent, hoặc gọi API. Phiên bản web AiShort không cần đăng ký.

## Cách sử dụng cơ bản

1. Tìm kiếm hoặc duyệt prompt bạn quan tâm trên trang chủ
2. Khi tìm được prompt phù hợp, nhấp nút "Sao chép" (hoặc mở thẻ để xem nội dung đầy đủ)
3. Dán vào bất kỳ công cụ AI nào — trang chat như ChatGPT, công cụ lập trình như Cursor, gọi API, v.v.
4. Theo các **placeholder** trong prompt (như `[nội dung]`, `[chủ đề]`) bổ sung nhu cầu cụ thể của bạn, rồi gửi cho AI

![](/img/docs/how-to-use-aishort.gif)

> 💡 **Mẹo nhỏ**: Hầu hết prompt dùng dấu ngoặc vuông `[...]` để đánh dấu nội dung bạn cần bổ sung — chỉ cần thay phần văn bản trong ngoặc vuông bằng nội dung cụ thể của bạn.

## Các Mô hình AI Phổ biến

### Mô hình Quốc tế

- [ChatGPT](https://chatgpt.com)
- [Gemini](https://gemini.google.com/app)
- [Claude](https://claude.ai/chats)
- [Grok](https://grok.com/)

> 💡 Cả bốn mô hình trên đều hiểu và trả lời tiếng Việt khá tốt. Chỉ cần thêm `respond in Vietnamese` (hoặc `trả lời bằng tiếng Việt`) ở cuối prompt là có ngay câu trả lời tiếng Việt tự nhiên.

### Mô hình tiếng Việt

- [PhoGPT](https://github.com/VinAIResearch/PhoGPT) — LLM mã nguồn mở do VinAI Research huấn luyện chuyên cho tiếng Việt, phù hợp khi cần tinh chỉnh hoặc triển khai nội bộ.
