# Hướng dẫn Cài đặt Tiện ích Chrome ZIP Cục bộ

## Tải xuống Tiện ích

Tải xuống gói cài đặt ZIP tiện ích ChatGPT Shortcut (ChatGPT_Shortcut-zip-3.x.x.zip) và giải nén. (Thư mục đã giải nén cần được giữ trên máy tính, nên nhớ giải nén đến vị trí cố định.)

- **GitHub**: [GitHub Release](https://github.com/rockbenben/ChatGPT-Shortcut/releases/latest)

## Bật Chế độ Nhà phát triển

Mở trang "Quản lý Tiện ích" của Chrome và bật "Chế độ nhà phát triển".

Sao chép địa chỉ sau và dán vào thanh địa chỉ trình duyệt, sau đó nhấn Enter để mở. Bật "Chế độ nhà phát triển" ở góc trên bên phải trang.

```txt
chrome://extensions
```

![](https://img.newzone.top/2024-08-12-22-05-52.png?imageMogr2/format/webp)

## Cài đặt Tiện ích

Kéo thư mục đã giải nén vào trang tiện ích để hoàn tất cài đặt.

![](https://img.newzone.top/2024-08-12-22-27-47.png?imageMogr2/format/webp)

Sau khi cài đặt thành công, kiểm tra [Hướng dẫn Sử dụng Plugin](./usage.md).
