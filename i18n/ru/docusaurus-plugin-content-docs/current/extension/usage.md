# Как использовать расширение браузера?

ChatGPT Shortcut поддерживает 13 основных языков. Язык расширения устанавливается автоматически в соответствии с вашим браузером. Вы также можете изменить его вручную в настройках.

![](/img/docs/extension-settings.png)

## Режим отображения

Расширение поддерживает три режима отображения:

- **Режим боковой панели**: Открывается в боковой панели браузера рядом с веб-контентом
- **Режим всплывающего окна**: Отображается как всплывающее окно при нажатии на значок
- **Режим окна**: Открывается в отдельном плавающем окне

## Держать боковую панель открытой

При включении боковая панель остается открытой при переключении вкладок браузера.

## Темный режим

Расширение поддерживает темный режим. Включите "Темный режим" в настройках, чтобы переключить интерфейс на темный вид.

![](/img/docs/extension-sidebar.png)

## Горячая клавиша

Используйте сочетание клавиш `Alt+Shift+S` для прямой активации расширения.
