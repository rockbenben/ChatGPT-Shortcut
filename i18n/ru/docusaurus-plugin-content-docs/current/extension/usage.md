# Как использовать расширение браузера?

ChatGPT Shortcut поддерживает 18 основных языков. Язык расширения устанавливается автоматически в соответствии с языком браузера. Его также можно изменить вручную на странице настроек.

![](/img/docs/extension-settings.png)

## Режимы отображения

Расширение поддерживает три режима отображения — выберите тот, что вам удобнее:

- **Режим боковой панели**: открывается в боковой панели браузера рядом с веб-контентом — удобно, когда нужно одновременно видеть ИИ-чат и просматривать промпты
- **Режим всплывающего окна**: отображается как небольшое окно при нажатии на значок — быстро посмотрел и закрыл, экран не занят
- **Режим окна**: открывается в отдельном плавающем окне — можно перетащить на второй монитор или разместить где удобно

## Закрепить боковую панель

При включении этой опции боковая панель остаётся открытой при переключении вкладок браузера — удобно для долгой работы.

## Тёмная тема

Расширение поддерживает тёмную тему. Включите «Тёмная тема» в настройках — интерфейс переключится на тёмное оформление.

![](/img/docs/extension-sidebar.webp)

## Горячая клавиша

Используйте сочетание `Alt+Shift+S`, чтобы вызвать расширение напрямую.
