# Установка расширения ZIP в Chrome (локальная)

> Это руководство предполагает, что вы уже скачали и распаковали пакет ZIP со [страницы установки](./README.md).
>
> ⚠️ **Две распространённые ошибки**:
> 1. Распакованная папка должна **храниться постоянно** (расширение читает из этой директории во время работы) — поместите её в надёжное место перед тем, как продолжить
> 2. Перетаскивать нужно **всю распакованную папку** (не сам ZIP-файл и не отдельные файлы внутри папки)

## Включить режим разработчика

Откройте страницу «Управление расширениями» в Chrome и включите «Режим разработчика».

Скопируйте следующий адрес, вставьте его в адресную строку браузера и нажмите Enter. В правом верхнем углу страницы включите «Режим разработчика».

```txt
chrome://extensions
```

![](https://img.newzone.top/2024-08-12-22-05-52.png?imageMogr2/format/webp)

## Установить расширение

Перетащите распакованную папку на страницу расширений — установка завершена.

![](https://img.newzone.top/2024-08-12-22-27-47.png?imageMogr2/format/webp)

После успешной установки см. [Руководство по использованию расширения](./usage.md).
