# Настройки расширения Firefox

В Firefox модель разрешений строже, чем в Chrome — после установки расширения нужны ещё два шага настройки, чтобы боковая панель корректно отображалась на ChatGPT, Gemini и других сайтах.

## 1. Закрепить расширение и настроить доступ

Сначала на панели инструментов Firefox выберите «Закрепить расширение ChatGPT Shortcut» (Pin to Toolbar). Затем зайдите в центр расширений, найдите запись ChatGPT Shortcut и выберите «Параметры» (Options), чтобы перейти к настройкам расширения. Подробные шаги показаны ниже:

![Firefox settings](https://img.newzone.top/2023-12-25-05-51-47.png?imageMogr2/format/webp)

## 2. Разрешить запуск расширения

Затем, чтобы расширение работало на сайтах ChatGPT, Gemini и других, на этих сайтах щёлкните правой кнопкой мыши по значку расширения и выберите «Всегда разрешать на этом сайте» (Always allow on \*\*\*). Это даст расширению право добавлять боковую панель на указанном домене.

![Firefox Extension Permission](https://img.newzone.top/2023-12-25-05-59-48.png?imageMogr2/format/webp)

После успешной установки см. [Руководство по использованию расширения](./usage).
