# Настройки Расширения Firefox

Перед использованием расширения Firefox рекомендуется выполнить эти два шага для настройки.

## 1. Закрепить Расширение и Настройки Доступа

Сначала выберите "Закрепить расширение ChatGPT Shortcut" (Pin to Toolbar) на панели инструментов Firefox. Затем зайдите в Центр Расширений, найдите пункт расширения ChatGPT Shortcut и выберите "Опции" (Options). Войдите в интерфейс настроек расширения. Конкретные шаги показаны ниже:

![Firefox settings](https://img.newzone.top/2023-12-25-05-51-47.png?imageMogr2/format/webp)

## 2. Авторизовать Запуск Расширения

Во-вторых, чтобы гарантировать нормальную работу расширения на веб-сайтах, таких как ChatGPT, Gemini, вам нужно нажать правой кнопкой мыши на иконку расширения на этих сайтах и выбрать "Всегда разрешать на этом веб-сайте" (Always allow on \*\*\*). Эта операция предоставляет расширению разрешение на добавление боковой панели на указанном домене.

![Firefox Extension Permission](https://img.newzone.top/2023-12-25-05-59-48.png?imageMogr2/format/webp)

После успешной установки вы можете ознакомиться с [Уроком по Использованию Плагина](./usage.md).
