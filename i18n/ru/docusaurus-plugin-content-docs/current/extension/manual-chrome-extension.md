# Руководство по Локальной Установке Расширения CRX для Chrome

## Скачать Пакет CRX

Скачайте установочный пакет crx ChatGPT Shortcut (ChatGPT_Shortcut-crx-3.x.x.zip) и распакуйте его (файл crx находится внутри распакованной папки).

- **GitHub**: [GitHub Release](https://github.com/rockbenben/ChatGPT-Shortcut/releases/latest)
- **Скачивание в Китае**: [Alist Drive](https://alist.newzone.top:9003/apps/ChatGPT%20Shortcut%20Extension); [Lanzou Cloud](https://wwva.lanzouq.com/b01lsc9vi), пароль: 1qow

![](https://img.newzone.top/2024-08-12-21-47-10.png?imageMogr2/format/webp)

## Включить Режим Разработчика

Откройте страницу "Управление расширениями" в Chrome и включите "Режим разработчика".

Скопируйте следующий адрес и вставьте его в адресную строку браузера, затем нажмите Enter, чтобы открыть. Включите "Режим разработчика" в правом верхнем углу страницы.

```txt
chrome://extensions
```

![](https://img.newzone.top/2024-08-12-22-05-52.png?imageMogr2/format/webp)

## Установить Расширение

Установите расширение ChatGPT Shortcut (Примечание ⚠️: Вам нужно перетащить файл .crx, НЕ нажимайте "Загрузить распакованное расширение")

![](https://img.newzone.top/2024-08-12-22-16-38.png?imageMogr2/format/webp)

После успешной установки вы можете ознакомиться с [Уроком по Использованию Плагина](./usage.md).

## Проблемы с Установкой?

1. Пользователи Windows: Вы распаковали скачанный установочный пакет (вместо того чтобы дважды кликнуть, чтобы открыть)?

2. Включен ли "Режим разработчика"? Если нет, см. операцию на шаге 2.

3. Вы перетащили файл crx на страницу "Расширения"? Примечание ⚠️: Не нажимайте "Загрузить распакованное расширение", вы должны перетащить файл crx.

4. Браузер не разрешает установку файлов crx? Попробуйте установить файл zip! [Нажмите здесь, чтобы посмотреть урок по установке zip](./manual-chrome-extension-zip.md).
