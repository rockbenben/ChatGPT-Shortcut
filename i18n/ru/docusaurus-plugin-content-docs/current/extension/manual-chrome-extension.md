# Установка расширения CRX в Chrome (локальная)

> Это руководство предполагает, что вы уже скачали и распаковали пакет CRX со [страницы установки](./README.md) (файл .crx находится внутри распакованной папки).
>
> ⚠️ **Две распространённые ошибки**:
> 1. Перетаскивать в страницу расширений нужно именно **файл `.crx`** (распакованный), а не сам ZIP-архив
> 2. Файл нужно именно **перетащить** — кнопкой «Загрузить распакованное расширение» пользоваться не нужно (она для ZIP-версии)

## Включить режим разработчика

Откройте страницу «Управление расширениями» в Chrome и включите «Режим разработчика».

Скопируйте следующий адрес, вставьте его в адресную строку браузера и нажмите Enter. В правом верхнем углу страницы включите «Режим разработчика».

```txt
chrome://extensions
```

![](https://img.newzone.top/2024-08-12-22-05-52.png?imageMogr2/format/webp)

## Установить расширение

Установите расширение ChatGPT Shortcut (внимание ⚠️: нужно именно перетащить .crx-файл — НЕ нажимайте «Загрузить распакованное расширение»).

![](https://img.newzone.top/2024-08-12-22-16-38.png?imageMogr2/format/webp)

После успешной установки см. [Руководство по использованию расширения](./usage).

## Проблемы с установкой?

1. Пользователи Windows: вы действительно распаковали скачанный архив (а не открыли его двойным кликом)?

2. Включён ли «Режим разработчика»? Если нет — см. шаг выше.

3. Вы именно перетащили CRX-файл на страницу «Расширения»? Внимание ⚠️: не нажимайте «Загрузить распакованное расширение» — файл нужно перетащить.

4. Браузер не разрешает устанавливать CRX-файлы? Попробуйте ZIP-версию! [Перейти к инструкции по установке ZIP](./manual-chrome-extension-zip).
