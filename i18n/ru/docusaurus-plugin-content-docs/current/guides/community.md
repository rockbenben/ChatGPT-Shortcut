# Промпты сообщества

Промпты сообщества — это публикации пользователей. Здесь можно найти вдохновение и поделиться собственными идеями. **Механизм голосования** выводит качественный контент в топ: то, что нравится сообществу, видят больше людей, а слабые промпты постепенно опускаются вниз.

![](/img/docs/community-page.png)

## Сортировка и фильтрация

- **Время публикации**: смотреть последние публикации
- **Сортировка по голосам**: смотреть промпты с наибольшим числом голосов «за»
- **Поиск по ключевым словам**: искать промпты по конкретной теме

## Механизм голосования

Голоса «за»/«против» определяют видимость промпта:

- **За**: привлекает внимание к качественному контенту
- **Против**: при накоплении определённого числа голосов «против» промпт скрывается

**Связь с коллекцией**: при добавлении промпта из сообщества в коллекцию автоматически засчитывается голос «за».

![Состояние кнопки голосования](/img/docs/community-vote.png)

> Голосование доступно только авторизованным пользователям; для каждого промпта засчитывается последний выбор пользователя.

## Приватные промпты

Идеи, которыми вы не хотите делиться публично, можно оставить приватными:

- При создании или редактировании промпта отключите переключатель «Поделиться в сообществе»
- Приватные промпты видны только вам
- Статус «публичный/приватный» можно менять в любое время
