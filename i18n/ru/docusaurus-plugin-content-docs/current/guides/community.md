# Промпты Сообщества

Раздел Промпты Сообщества отображает пользовательские промпты, которыми поделились пользователи. Вы можете найти здесь вдохновение или поделиться своим творчеством.

![](/img/docs/community-page.png)

## Сортировка и Фильтрация

Страница сообщества предлагает несколько методов фильтрации:

- **Время публикации**: Просмотр последних опубликованных промптов
- **Порядок голосования**: Просмотр промптов с наибольшим количеством голосов "за"
- **Поиск по ключевым словам**: Поиск промптов по конкретным темам

## Механизм Голосования

Решайте судьбу промптов, голосуя за/против:

- **За**: Помогает качественному контенту получить больше внимания
- **Против**: После достижения определенного количества голосов "против", промпт будет скрыт

**Связь с Коллекцией**: При добавлении промпта из сообщества в коллекцию система автоматически ставит голос "за".

![Статус Кнопки Голосования](/img/docs/community-vote.png)

> Голосование доступно только для вошедших пользователей, и голос каждого человека за один и тот же промпт основывается на самом последнем.

## Личные Промпты

Творчество, которое вы не хотите делать публичным, можно настроить как личное:

- При создании или редактировании промпта снимите галочку с опции "Поделиться с сообществом"
- Личные промпты видны только вам
- Вы можете переключаться между публичным/личным статусом в любое время
