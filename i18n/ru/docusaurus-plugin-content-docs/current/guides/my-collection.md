# Моя Коллекция

После входа вы можете собирать свои любимые промпты и управлять ими лично. Вид Моя Коллекция предоставляет богатые функции управления, чтобы помочь вам эффективно организовывать и использовать промпты.

## Переключение Вида

Верхняя часть главной страницы предоставляет кнопки переключения вида, поддерживая два режима:

- **Моя Коллекция**: Показывает ваши собранные промпты и пользовательские промпты
- **Библиотека Промптов**: Просматривайте публичную библиотеку промптов, чтобы находить новый контент

Браузер автоматически запомнит ваш выбор вида и переключится на последний вид при следующем посещении.

![Вид Моя Коллекция](/img/docs/my-collection-view.gif)

## Добавить в Коллекцию

Нажмите кнопку "Коллекция" на карточке промпта, чтобы добавить его в Мою Коллекцию.

При добавлении промпта из сообщества в коллекцию система автоматически ставит голос "за", помогая качественному контенту получить больше внимания.

## Удалить из Коллекции

В виде Моя Коллекция нажмите иконку сердца на карточке, чтобы удалить из коллекции.

## Пользовательские Теги

Добавляйте личные теги к собранным промптам для удобного управления классификацией:

### Добавить Тег

1. Нажмите иконку тега в панели управления
2. Выберите существующий тег или нажмите "+", чтобы создать новый тег
3. Новые теги могут иметь пользовательские названия и цвета

### Управление Тегами

Нажмите "Управление Тегами", чтобы войти в интерфейс управления тегами:

- Изменить название тега
- Изменить цвет тега (доступно 11 предустановленных цветов)
- Удалить ненужные теги

![Всплывающее Окно Управления Тегами](/img/docs/my-collection-tag.png)

### Фильтр по Тегу

В панели фильтрации вида Моя Коллекция вы можете фильтровать контент по пользовательским тегам.

## Сортировка Перетаскиванием

Настройка порядка карточек перетаскиванием поддерживается в виде Моя Коллекция:

1. Перетащите ручку слева от карточки
2. Переместите в целевую позицию и отпустите
3. Новый порядок автоматически сохраняется на сервере

Функция сортировки доступна только в виде Моя Коллекция.

![Сортировка Перетаскиванием](/img/docs/my-collection-drag.gif)
