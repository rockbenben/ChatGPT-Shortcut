# Моя коллекция

После входа представление **«Моя коллекция»** позволяет упорядочить любимые промпты так же, как закладки в браузере: сортировка перетаскиванием, пользовательские теги и фильтрация по тегам — всё под рукой.

:::tip Онлайн- и офлайн-версии
В онлайн-версии коллекции хранятся на сервере (требуется вход); в [офлайн-версии](./offline) — в локальном хранилище браузера (localStorage, вход не нужен). Функциональность одинаковая.
:::

## Переключение представления

Кнопки переключения в верхней части главной страницы дают два режима:

- **Моя коллекция**: ваши собранные промпты + свои промпты
- **Библиотека промптов**: публичная библиотека

Браузер запомнит ваш выбор и при следующем посещении автоматически откроет последнее представление.

![Представление «Моя коллекция»](/img/docs/my-collection-view.gif)

## Добавление в коллекцию

Нажмите кнопку «Коллекция» на карточке, чтобы добавить промпт в «Мою коллекцию».

При добавлении промпта из сообщества автоматически засчитывается голос «за» — это привлекает внимание к качественному контенту.

## Удаление из коллекции

В представлении «Моя коллекция» нажмите иконку сердца на карточке, чтобы убрать промпт из коллекции.

## Пользовательские теги

Назначайте собственные теги собранным промптам для удобной классификации:

### Добавление тега

1. Нажмите иконку тега в панели управления
2. Выберите существующий тег или нажмите «+», чтобы создать новый
3. У нового тега можно настроить название и цвет

### Управление тегами

Нажмите «Управление тегами», чтобы открыть интерфейс управления:

- Переименовать тег
- Изменить цвет (11 заранее заданных цветов)
- Удалить ненужный тег

![Окно управления тегами](/img/docs/my-collection-tag.png)

### Фильтр по тегу

В панели фильтрации представления «Моя коллекция» можно отфильтровать содержимое по пользовательским тегам.

## Сортировка перетаскиванием

В представлении «Моя коллекция» доступна сортировка перетаскиванием:

1. Возьмитесь за ручку слева на карточке
2. Перетащите её в нужное место и отпустите
3. Новый порядок сохранится автоматически (в онлайн-версии — на сервере, в офлайн-версии — локально в браузере)

Сортировка доступна только в представлении «Моя коллекция».

![Сортировка перетаскиванием](/img/docs/my-collection-drag.gif)
