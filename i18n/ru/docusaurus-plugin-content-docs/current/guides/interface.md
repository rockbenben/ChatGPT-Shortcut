# Руководство по Интерфейсу

Страница AiShort предоставляет пользователям интуитивно понятный интерфейс для просмотра и фильтрации промптов. Страница в основном состоит из области тегов, области поиска и области отображения промптов.

![](/img/docs/interface-home.png)

## 🏷︎ Фильтрация по Тегам

Область тегов помогает пользователям фильтровать промпты по домену и функции.

С помощью кнопки "Переключить Правило Фильтрации Тегов" в правом верхнем углу вы можете добиться более точной фильтрации по нескольким тегам:

- **Правило ИЛИ** (По умолчанию): Показывает промпты, имеющие любой из выбранных тегов
- **Правило И**: Показывает только промпты, имеющие все выбранные теги одновременно

![](/img/docs/interface-filter.png)

## 🔍 Поиск по Ключевым Словам

Функция поиска по ключевым словам поддерживает поиск по названиям промптов, описаниям и содержанию.

- Поиск на главной странице в основном включает **отборные промпты** (исключая промпты сообщества). Если теги уже выбраны, область поиска ограничивается ими
- После входа в систему область поиска расширяется и включает ваши **коллекции** и **пользовательские промпты**
- Если вы не нашли ожидаемых результатов на главной странице, используя короткие ключевые слова, перейдите на страницу [Промпты сообщества](./community) для поиска

![](/img/docs/interface-search.png)

## 🔬 Карточки Промптов

### Просмотр Деталей

Нажмите на карточку промпта, чтобы открыть всплывающее окно с деталями, где вы можете увидеть название, описание, полное содержание промпта, примечания и кнопку копирования.

![](/img/docs/prompt-card.gif)

### Копирование Промпта

Нажмите кнопку "Копировать" в нижней части карточки, чтобы получить промпт. Вставьте его в модель ИИ для использования.

### Добавить в Коллекцию

После входа на карточке появится кнопка коллекции. Нажмите, чтобы добавить промпт в Мою Коллекцию.

## 🔥 Сортировка по Популярности

Страница сортирует промпты на основе их популярности, чтобы помочь вам быстро находить трендовый контент.

Сортировка тегов также корректируется на основе значений популярности, чтобы гарантировать, что она отражает последние предпочтения сообщества.
