# Руководство по интерфейсу

В этом разделе описаны три основные области главной страницы AiShort — **область тегов** (теги сценариев сверху), **область поиска** (поле ввода ключевых слов) и **область промптов** (список карточек ниже), — и то, как с их помощью быстро найти нужный промпт.

![](/img/docs/interface-home.png)

## 🏷︎ Фильтрация по тегам

Фильтрация промптов по предметной области и функции. Переключатель **OR / AND** в правом верхнем углу определяет правило для нескольких выбранных тегов:

- **OR** (по умолчанию): показываются промпты с любым из выбранных тегов
- **AND**: показываются только промпты, у которых есть все выбранные теги одновременно

![](/img/docs/interface-filter.png)

## 🔍 Поиск по ключевым словам

Ищет по названию, описанию и содержанию промптов.

- Поиск на главной странице охватывает **отборные промпты** (без сообщества). Если выбраны теги, поиск ограничивается ими
- После входа в область поиска добавляются ваши **коллекции** и **свои промпты**
- Если по короткому ключевому слову на главной ничего не найдено, повторите поиск на странице [Промпты сообщества](./community)

![](/img/docs/interface-search.png)

## 🔬 Карточки промптов

### Просмотр деталей

Нажмите на карточку, чтобы открыть окно с деталями — в нём доступны полное содержание, примечания и кнопка копирования.

![](/img/docs/prompt-card.gif)

### Копирование промпта

Нажмите кнопку «Копировать» под карточкой и вставьте промпт в любой ИИ-инструмент — чат, редакторы кода (Cursor, GitHub Copilot), вызов через API и т. п.

### Добавление в коллекцию

После входа на карточке появится кнопка коллекции — нажмите её, чтобы добавить промпт в «Мою коллекцию».

## 🔥 Сортировка по популярности

Промпты сортируются по популярности — так проще увидеть, что сейчас востребовано. Порядок тегов также подстраивается под популярность и отражает актуальные интересы сообщества.
