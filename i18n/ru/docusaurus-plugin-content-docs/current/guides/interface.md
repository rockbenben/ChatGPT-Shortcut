# Руководство по интерфейсу

Главная страница AiShort состоит из четырёх частей: **заголовок и кнопки входа** вверху, **фильтры по тегам** под ними, **строка поиска** и **карточки промптов**, занимающие остальное пространство страницы. В этом разделе описано, как использовать их вместе, чтобы быстро найти нужный промпт.

![](/img/docs/interface-home.png)

## Кнопки входа вверху

Кнопки под заголовком страницы немного различаются до и после входа в систему:

- **Не авторизованы**: «**Войти бесплатно**» + «**Обзор сообщества**»
- **Авторизованы**: переключатель режима («**Обзор**» ⇄ «**Моя коллекция**»), «**Создать промпт**» и меню настроек справа («Мой Аккаунт», «Выйти»)

> 💡 Просматривать, искать и копировать все отборные промпты можно без входа в систему. Авторизация открывает только возможности коллекции, создания промптов и синхронизации между устройствами.

## Фильтрация по тегам

Фильтрация промптов по предметной области и функции. Переключатель **OR / AND** в правом верхнем углу определяет правило объединения нескольких тегов:

- **OR** (по умолчанию): показываются промпты с любым из выбранных тегов
- **AND**: показываются только промпты, у которых есть все выбранные теги одновременно

После выбора тегов появляется кнопка «**Очистить фильтры**» — один клик сбрасывает все выборы. На мобильных устройствах теги свёрнуты по умолчанию — нажмите «**Показать теги**», чтобы развернуть.

![](/img/docs/interface-filter.png)

## Поиск по ключевым словам

Ищет по названию, описанию, содержанию и примечаниям промптов.

- На компьютере нажмите **`Ctrl + K`** (`⌘ + K` на macOS), чтобы перейти в строку поиска; `Esc` — для выхода
- Поиск на главной странице охватывает **отборные промпты**. Если выбраны теги, поиск ограничивается ими
- После входа в систему поиск расширяется — добавляются ваши **коллекции** и **свои промпты**
- Если по короткому ключевому слову на главной ничего не найдено, AiShort автоматически ищет также в [Промптах сообщества](./community)

![](/img/docs/interface-search.png)

## Карточки промптов

Отборные промпты сортируются по суммарному количеству копирований, поэтому самый востребованный контент показывается первым.

### Информация карточки

На каждой карточке отображаются название, описание, превью содержания и теги промпта. Значок пламени и число в правом верхнем углу — суммарное количество копирований.

### Просмотр деталей

Нажмите на карточку, чтобы открыть окно с деталями — прочитайте полное содержание и примечания, затем скопируйте одним кликом.

![](/img/docs/prompt-card.webp)

### Копирование промпта

Нажмите кнопку «**Копировать**» на карточке или в окне деталей, затем вставьте в любой ИИ-инструмент (чаты, редакторы кода, вызовы API и т. д.).

### Добавление в коллекцию

После входа на каждой карточке появляется кнопка-сердце — нажмите её, чтобы добавить промпт в [Мою коллекцию](./my-collection).
