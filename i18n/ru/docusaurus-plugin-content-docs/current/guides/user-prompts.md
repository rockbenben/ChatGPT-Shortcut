# Свои промпты

После входа вы можете создавать собственные промпты и сохранять их в аккаунте для последующего вызова. Свои промпты отображаются в представлении «Моя коллекция».

## Создание промпта

1. Нажмите кнопку «Добавить» в правом верхнем углу
2. В открывшемся окне «Создать промпт» заполните форму:
   - **Название промпта** (обязательно): краткое имя промпта
   - **Содержание промпта** (обязательно): сам текст промпта
   - **Цель / использование** (по желанию): для чего нужен промпт
   - **Примечания** (по желанию): источник, переводы на другие языки или дополнительные пояснения
3. Внизу переключатель «Хотите поделиться этим промптом на публичной странице?» включён по умолчанию — выключите, если хотите оставить промпт приватным
4. Нажмите «Создать промпт» для подтверждения

![Окно «Создать промпт»](/img/docs/user-prompts-create.png)

> 💡 **Пример заполнения** — промпт «Помощник для протоколов совещаний»:
> - **Название промпта**: Помощник для протоколов совещаний
> - **Содержание промпта**: Вы — опытный секретарь-референт. На основе расшифровки ниже составьте протокол совещания: укажите участников, ключевые решения, поручения с ответственными и сроками, открытые вопросы. Сохраняйте деловой стиль: [расшифровка или заметки со встречи]
> - **Цель / использование**: превратить «сырую» расшифровку или заметки в готовый к рассылке протокол
> - **Примечания**: подходит для планёрок, проектных встреч и созвонов в Zoom/Telemost; перед отправкой проверьте ФИО и сроки

## Редактирование промпта

В представлении «Моя коллекция» нажмите на карточку созданного вами промпта, чтобы открыть окно «Редактировать промпт». Здесь можно:

- Изменить название, содержание, цель, примечания
- Переключить статус обмена (публичный/приватный)
- Нажать «Сохранить» для обновления

![Интерфейс редактирования промпта](/img/docs/user-prompts-edit.png)

## Удаление промпта

В окне редактирования нажмите «Удалить». Действие необратимо — будьте внимательны.

## Обмен с сообществом

При создании или редактировании переключатель «Хотите поделиться этим промптом на публичной странице?» внизу определяет видимость:

- **Включён (по умолчанию)**: промпт отображается на странице [Промпты сообщества](./community), другие пользователи могут его видеть и сохранять в коллекцию
- **Выключен**: приватный промпт, виден только вам

Статус обмена можно менять в любое время.

## Экспорт резервной копии

1. Зайдите в «Мой аккаунт» → найдите блок «Управление данными → Экспорт промптов»
2. Нажмите кнопку «Экспорт данных»
3. Система сформирует JSON-файл и автоматически скачает его

В экспорт входят:

- Название, содержание, цель/использование и примечания промптов
- Время создания и обновления
- Статус обмена

Регулярные резервные копии помогут избежать потери данных.

## Импорт промптов

Импорт промптов и коллекций из JSON-файла:

1. Зайдите в «Мой аккаунт» → найдите блок «Управление данными → Импорт промптов»
2. Нажмите кнопку «Импорт данных»
3. Выберите JSON-файл
4. Система автоматически объединит данные (дубликаты определяются по заголовку; промпты с чужими ID автоматически становятся приватными)

### Совместная работа в команде

Рекомендуемый сценарий обмена промптами внутри команды:

1. **Отбор и обмен**: после экспорта вручную удалите коллекцию или отберите конкретные промпты для обмена, затем передайте файл коллегам для импорта
2. **Защита приватности**: импортированные чужие промпты (с ID, не принадлежащими текущему аккаунту) автоматически становятся **приватными** — данные разных участников не пересекаются
