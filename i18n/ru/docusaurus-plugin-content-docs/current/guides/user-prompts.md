# Пользовательские Промпты

После входа вы можете создавать свои собственные промпты для быстрого доступа в будущем. Пользовательские промпты будут сохранены в вашем аккаунте и показаны в виде Моя Коллекция.

## Создать Промпт

1. Нажмите кнопку "Добавить Промпт" в правом верхнем углу страницы
2. Заполните информацию о промпте:
   - **Название**: Краткое описание цели промпта
   - **Описание**: Полное содержание промпта
   - **Примечание**: Инструкции по использованию или заметки (необязательно)
3. Выберите, делиться ли с сообществом
4. Нажмите Отправить

![Всплывающее Окно Добавления Промпта](/img/docs/user-prompts-create.png)

## Редактировать Промпт

В виде Моя Коллекция нажмите на карточку промпта, который вы создали, чтобы:

- Изменить название, описание и примечание
- Переключить статус обмена (Публичный/Личный)
- Сохранить изменения

![Интерфейс Редактирования Промпта](/img/docs/user-prompts-edit.png)

## Удалить Промпт

Нажмите кнопку "Удалить" в интерфейсе редактирования, чтобы удалить промпт. Удаление невозможно отменить, пожалуйста, действуйте с осторожностью.

## Поделиться с Сообществом

При создании или редактировании промпта отметьте опцию "Поделиться с Сообществом":

- **Публичный**: Промпт будет показан на странице сообщества, и другие пользователи могут видеть его и добавлять в коллекцию
- **Личный**: Виден только вам, не появится на странице сообщества

Вы можете изменить статус обмена в любое время.

## Экспорт Резервной Копии

Экспортируйте все созданные вами промпты:

1. Зайдите в Центр Пользователя
2. Нажмите кнопку "Экспорт Промптов"
3. Система генерирует JSON файл и автоматически скачивает его

Содержание экспорта включает:

- Название промпта, описание, примечание
- Время создания и обновления
- Статус обмена

Рекомендуется регулярно экспортировать резервные копии, чтобы избежать потери данных.
