# Свои промпты

После входа вы можете создавать собственные промпты и сохранять их в аккаунте для последующего быстрого доступа. Свои промпты отображаются в представлении «Моя коллекция».

## Создание промпта

1. После входа нажмите кнопку «**Создать промпт**» в верхней части страницы
2. Заполните диалог «Создать промпт»:
   - **Название промпта** (обязательно): имя промпта
   - **Содержание промпта** (обязательно): тело промпта; текст в квадратных скобках `[...]` подсвечивается как место для подстановки при отображении
   - **Использование** (по желанию): краткое описание того, что делает промпт
   - **Примечания** (по желанию): источник, версии на других языках или дополнительные пояснения
3. Переключатель «Хотите поделиться этим промптом на публичной странице?» внизу включён по умолчанию — выключите, если хотите оставить промпт приватным
4. Нажмите «Создать промпт» для подтверждения

![Окно «Создать промпт»](/img/docs/user-prompts-create.png)

> 💡 **Пример** — промпт «Генератор обновлений для стендапа»:
> - **Название промпта**: Генератор обновлений для стендапа
> - **Содержание промпта**: Вы — лаконичный ассистент по коммуникациям. Превратите следующие заметки в обновление для ежедневного стендапа с тремя разделами — Вчера, Сегодня, Блокеры. Пишите каждый пункт кратко, с акцентом на результат: [мои необработанные заметки]
> - **Использование**: Превращает разрозненные заметки в чистое обновление для стендапа
> - **Примечания**: Запускайте перед утренним стендапом

## Редактирование промпта

В представлении «Моя коллекция» нажмите кнопку редактирования (карандаш) на карточке созданного вами промпта, чтобы открыть диалог «Редактировать промпт». Здесь можно:

- Изменить название, содержание, использование и примечания
- Переключить статус обмена (публичный / приватный)
- Нажать «Сохранить» для обновления

![Интерфейс редактирования промпта](/img/docs/user-prompts-edit.png)

## Удаление промпта

В диалоге редактирования нажмите «Удалить». Действие необратимо — будьте внимательны.

## Обмен с сообществом

При создании или редактировании переключатель «Хотите поделиться этим промптом на публичной странице?» внизу определяет видимость:

- **Включён (по умолчанию)**: промпт отображается на странице [Промпты сообщества](./community), другие пользователи могут его видеть и добавлять в коллекцию
- **Выключен**: приватный — виден только вам

Статус обмена можно менять в любое время.

## Экспорт промптов

1. Перейдите в «Мой Аккаунт» и найдите блок «Управление данными → Экспортировать промпты»
2. Нажмите кнопку «Экспорт данных»
3. Система сформирует JSON-файл и автоматически скачает его

В экспорт входят:

- Название, содержание, использование и примечания промптов
- Статус обмена
- Ваши коллекции, порядок коллекций и пользовательские теги

Регулярные экспорты рекомендуются для предотвращения потери данных.

## Импорт промптов

Импорт промптов и коллекций из JSON-файла:

1. Перейдите в «Мой Аккаунт» и найдите блок «Управление данными → Импортировать промпты»
2. Нажмите кнопку «Импорт данных»
3. Выберите JSON-файл
4. Система автоматически объединит данные (дубликаты определяются по заголовку; промпты с чужими ID автоматически становятся приватными)

### Совместная работа в команде

Рекомендуемый сценарий обмена промптами внутри команды:

1. **Отбор и обмен**: после экспорта вручную удалите список коллекции или отберите конкретные промпты для обмена, затем передайте файл коллегам для импорта
2. **Защита приватности**: импортированные чужие промпты (с ID, не принадлежащими текущему аккаунту) автоматически становятся **приватными** — данные каждого участника остаются отдельными
