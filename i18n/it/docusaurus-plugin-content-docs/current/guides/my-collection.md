# La Mia Collezione

Dopo l'accesso, puoi collezionare i tuoi prompt preferiti e gestirli personalmente. La vista La Mia Collezione offre ricche funzioni di gestione per aiutarti a organizzare e utilizzare i prompt in modo efficiente.

## Cambio Vista

La parte superiore della home page fornisce pulsanti per cambiare vista, supportando due modalità:

- **La Mia Collezione**: Mostra i tuoi prompt collezionati e i prompt personalizzati
- **Libreria Prompt**: Sfoglia la libreria di prompt pubblica per scoprire nuovi contenuti

Il browser ricorderà automaticamente la tua selezione di vista e passerà all'ultima vista alla tua prossima visita.

![Vista La Mia Collezione](/img/docs/my-collection-view.gif)

## Aggiungi alla Collezione

Clicca sul pulsante "Colleziona" su una scheda prompt per aggiungerla a La Mia Collezione.

Quando collezioni un prompt della community, il sistema emetterà automaticamente un voto positivo, aiutando i contenuti di qualità a ottenere maggiore esposizione.

## Rimuovi dalla Collezione

Nella vista La Mia Collezione, clicca sull'icona del cuore sulla scheda per rimuovere dalla collezione.

## Tag Personalizzati

Aggiungi tag personali ai prompt collezionati per una comoda gestione della classificazione:

### Aggiungi Tag

1. Clicca sull'icona del tag nella barra di controllo
2. Seleziona un tag esistente o clicca su "+" per creare un nuovo tag
3. I nuovi tag possono avere nomi e colori personalizzati

### Gestisci Tag

Clicca su "Gestisci Tag" per accedere all'interfaccia di gestione dei tag:

- Modifica nome tag
- Cambia colore tag (11 colori preimpostati disponibili)
- Elimina tag indesiderati

![Popup Gestione Tag](/img/docs/my-collection-tag.png)

### Filtra per Tag

Nella barra di filtro della vista La Mia Collezione, puoi filtrare i contenuti per tag personalizzati.

## Ordinamento Drag-and-Drop

La regolazione dell'ordine delle schede tramite drag-and-drop è supportata nella vista La Mia Collezione:

1. Trascina la maniglia a sinistra della scheda
2. Sposta nella posizione di destinazione e rilascia
3. Il nuovo ordine viene salvato automaticamente sul server

La funzione di ordinamento è disponibile solo nella vista La Mia Collezione.

![Ordinamento Drag-and-Drop](/img/docs/my-collection-drag.gif)
