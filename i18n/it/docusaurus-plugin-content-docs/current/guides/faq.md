# FAQ

## Perché usare prompt in inglese?

I modelli IA hanno generalmente una migliore comprensione dell'inglese, e l'uso di prompt in inglese può produrre risultati più stabili.

Sebbene l'uso di prompt in cinese possa ottenere risultati decenti, l'input ripetuto dello stesso prompt in cinese può portare a variazioni significative nei risultati. Pertanto, si raccomanda di usare prompt in inglese in scenari importanti.

**Specificare risposta in cinese**: Aggiungi `respond in Chinese` alla fine del prompt per ottenere una risposta in cinese.

## Devo inserire il prompt ogni volta?

**Uso API**: Puoi impostare il prompt come "system prompt", che verrà applicato automaticamente alle conversazioni successive.

**Versione Web**: Se non hai cambiato argomento, ChatGPT ricorderà le impostazioni del prompt della conversazione attuale. Quando la risposta inizia a deviare dalle aspettative, significa che l'IA ha "dimenticato" il prompt e devi inserirlo di nuovo.

**Suggerimento**: Salva le conversazioni utilizzate frequentemente come segnalibri per un accesso rapido successivo.

## Cosa fare se l'IA genera false informazioni?

L'IA a volte ha "allucinazioni", generando informazioni che sembrano plausibili ma sono in realtà errate. Suggerimenti d'uso:

1. **Verifica informazioni chiave**: Soprattutto dati, citazioni, codice, ecc.
2. **Ottimizzazione a più turni**: Chiedi all'IA di controllare e ottimizzare nuovamente la risposta.
3. **Controllo incrociato**: Usa prompt o modelli diversi per verificare conclusioni importanti.

Prompt appropriati possono aiutarti a ridurre il verificarsi di allucinazioni IA.

## Come faccio il backup dei miei prompt?

1. Entra nel Centro Utente
2. Clicca sul pulsante "Esporta Prompt"
3. Il sistema genera automaticamente un file JSON da scaricare

Si raccomanda di fare backup regolarmente per evitare la perdita di dati.

## I risultati dei prompt non sono ideali?

1. **Prova a Ottimizzare**: Lascia che l'IA ottimizzi nuovamente la risposta.
2. **Cambia Prospettiva**: Usa prompt diversi per esprimere lo stesso requisito.
3. **Feedback e Condivisione**: Se trovi problemi o hai buoni prompt, sei benvenuto a informarci tramite [Feedback](/feedback) o condividere nella community.
