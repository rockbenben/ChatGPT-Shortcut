# Guida Installazione Locale Estensione ZIP Chrome

> Questa guida presuppone che tu abbia già scaricato e decompresso il pacchetto ZIP dalla [pagina di installazione](./README.md).
>
> ⚠️ **Due errori comuni**:
> 1. La cartella decompressa deve essere **mantenuta a lungo termine** (l'estensione legge da questa directory durante l'esecuzione) — spostala in una posizione definitiva prima di continuare
> 2. Devi trascinare l'**intera cartella decompressa** (non il file zip, né un singolo file all'interno della cartella)

## Abilita Modalità Sviluppatore

Apri la pagina "Gestisci estensioni" di Chrome e attiva la "Modalità sviluppatore".

Copia il seguente indirizzo e incollalo nella barra degli indirizzi del browser, poi premi Invio per aprire. Attiva la "Modalità sviluppatore" nell'angolo in alto a destra della pagina.

```txt
chrome://extensions
```

![](https://img.newzone.top/2024-08-12-22-05-52.png?imageMogr2/format/webp)

## Installa Estensione

Trascina la cartella decompressa nella pagina delle estensioni per completare l'installazione.

![](https://img.newzone.top/2024-08-12-22-27-47.png?imageMogr2/format/webp)

Dopo un'installazione riuscita, puoi consultare il [Tutorial Uso Plugin](./usage).
