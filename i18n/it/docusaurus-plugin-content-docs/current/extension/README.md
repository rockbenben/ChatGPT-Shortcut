# Installa Estensione Browser

L'estensione AiShort (ChatGPT Shortcut) incorpora la libreria di prompt direttamente nella barra laterale delle pagine di chat AI come ChatGPT, Gemini, Claude e Doubao, così non devi più tornare su aishort.top per copiare e incollare. Supporta Chrome, Edge e Firefox e si richiama rapidamente con `Alt + Shift + S`.

## 📥 Installazione

### 1. Store ufficiali (consigliato, installazione con un clic)

- **Chrome**: [Chrome Web Store](https://chrome.google.com/webstore/detail/chatgpt-shortcut/blcgeoojgdpodnmnhfpohphdhfncblnj)
- **Edge**: [Microsoft Edge Addons](https://microsoftedge.microsoft.com/addons/detail/chatgpt-shortcut/hnggpalhfjmdhhmgfjpmhlfilnbmjoin)
- **Firefox**: [Firefox Browser ADD-ONS](https://addons.mozilla.org/addon/chatgpt-shortcut/) (dopo l'installazione è necessaria una [configurazione delle autorizzazioni](./firefox-extension-setting.md) una tantum, altrimenti la barra laterale non viene mostrata su ChatGPT)

### 2. Store non accessibile? Pacchetto di installazione locale

Scarica da uno dei seguenti canali, poi segui la relativa guida di installazione:

- **GitHub**: [GitHub Release](https://github.com/rockbenben/ChatGPT-Shortcut/releases/latest)

> 📖 Guide di installazione: [File CRX](./manual-chrome-extension.md) · [File ZIP](./manual-chrome-extension-zip.md)

## ✅ Dopo l'installazione

Premi `Alt + Shift + S` per richiamarla, oppure clicca sull'icona nella barra degli strumenti. Per i dettagli consulta la [guida all'uso](./usage.md).

## 🧩 Script Tampermonkey

Oltre al formato estensione, forniamo anche lo [**Script Tampermonkey ChatGPT Shortcut Anywhere**](https://greasyfork.org/scripts/482907-chatgpt-shortcut-anywhere).
Tampermonkey è un'estensione del browser che permette agli utenti di eseguire script personalizzati per migliorare le funzioni delle pagine web.

Con questo script, puoi invocare la barra laterale AiShort su qualsiasi sito web. Tuttavia, a causa delle restrizioni della pagina ufficiale di ChatGPT sull'iniezione di script, su quella pagina lo script verrà eseguito come un **popup** invece di una barra laterale.

Una volta attivata la barra laterale di AiShort, vedrai un interruttore con un'icona verde nell'angolo in basso a destra delle pagine supportate. Clicca su questa icona per aprire o chiudere la barra laterale. Al momento sono supportati di default ChatGPT, Gemini, Claude, Doubao e altri.
