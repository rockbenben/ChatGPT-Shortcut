# Firefox Uzantı Ayarları

Firefox uzantısını kullanmadan önce, aşağıdaki iki adımı izleyerek kurulum yapmanız önerilir.

## 1. Uzantıyı Sabitleyin ve Ayarlara Erişin

İlk olarak, Firefox araç çubuğunda "ChatGPT Shortcut Uzantısını Sabitle" (Araç Çubuğuna Sabitle) seçeneğini seçin. Ardından, Uzantı Merkezi'ne girin, ChatGPT Shortcut uzantı girişini bulun ve "Seçenekler"i seçin. Uzantının ayarlar arayüzüne girin. Belirli adımlar aşağıda gösterilmiştir:

![Firefox ayarları](https://img.newzone.top/2023-12-25-05-51-47.png?imageMogr2/format/webp)

## 2. Uzantının Çalışmasını Yetkilendirin

İkinci olarak, uzantının ChatGPT, Gemini gibi web sitelerinde normal çalışmasını sağlamak için, bu web sitelerinde uzantı simgesine sağ tıklayıp "\*\*\* üzerinde her zaman izin ver" seçeneğini seçmeniz gerekir. Bu işlem, uzantıya belirtilen etki alanına kenar çubuğu ekleme izni verir.

![Firefox Uzantı İzni](https://img.newzone.top/2023-12-25-05-59-48.png?imageMogr2/format/webp)

Başarılı kurulumdan sonra, [Plugin Kullanım Öğreticisi](./usage.md)'ni kontrol edin.
