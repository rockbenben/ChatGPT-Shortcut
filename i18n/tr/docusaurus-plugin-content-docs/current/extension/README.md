# Tarayıcı Uzantısını Yükle

AiShort (ChatGPT Shortcut) uzantısı, prompt kütüphanesini doğrudan ChatGPT, Gemini, Claude, Doubao gibi AI sohbet sayfalarının kenar çubuğuna yerleştirir; böylece aishort.top adresine geri dönüp kopyala-yapıştır yapmanıza gerek kalmaz. Chrome, Edge ve Firefox'u destekler; `Alt + Shift + S` ile hızlıca çağırabilirsiniz.

## 📥 Kurulum

### 1. Uygulama Mağazaları (Önerilen, tek tıkla kurulum)

- **Chrome**: [Chrome Web Store](https://chrome.google.com/webstore/detail/chatgpt-shortcut/blcgeoojgdpodnmnhfpohphdhfncblnj)
- **Edge**: [Microsoft Edge Addons](https://microsoftedge.microsoft.com/addons/detail/chatgpt-shortcut/hnggpalhfjmdhhmgfjpmhlfilnbmjoin)
- **Firefox**: [Firefox Browser ADD-ONS](https://addons.mozilla.org/addon/chatgpt-shortcut/) (Kurulumdan sonra bir kez [izin ayarı](./firefox-extension-setting.md) yapmanız gerekir; aksi takdirde ChatGPT üzerinde kenar çubuğu görüntülenmez)

## ✅ Kurulumdan Sonra

`Alt + Shift + S` tuşlarına basarak çağırın veya araç çubuğundaki simgeye tıklayın. Ayrıntılı kullanım için [kullanım kılavuzuna](./usage.md) bakın.

## 🧩 Tampermonkey Script

Uzantıya ek olarak, [**ChatGPT Shortcut Anywhere Tampermonkey Script**](https://greasyfork.org/scripts/482907-chatgpt-shortcut-anywhere) de sunuyoruz.
Tampermonkey, kullanıcıların web sayfası işlevlerini geliştirmek için özel scriptler çalıştırmasına izin veren bir tarayıcı uzantısıdır.

Bu script ile herhangi bir web sitesinde AiShort kenar çubuğunu çağırabilirsiniz. Ancak ChatGPT resmi sayfası script enjeksiyonunu kısıtladığından, bu sayfada script kenar çubuğu yerine **açılır pencere** biçiminde çalışır.

AiShort kenar çubuğunu etkinleştirdikten sonra, desteklenen web sayfalarının sağ alt köşesinde yeşil bir simge düğmesi göreceksiniz. Bu simgeye tıklayarak kenar çubuğunu açıp kapatabilirsiniz. Şu anda varsayılan olarak ChatGPT, Gemini, Claude, Doubao gibi sayfalar desteklenmektedir.
